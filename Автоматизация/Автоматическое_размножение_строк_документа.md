Некоторые переменные обладают специальным свойством: при формировании документа по [ шаблону](Документы/Шаблоны_документов.md), в документ будет добавлено по одной строке для каждого значения, которое хранится в переменной. 


Например, у вас есть справочник автомобилей: 


![SbPnnp.png](images/SbPnnp.png)


  

В таблице документа необходимо получить марку и модель авто. Настраиваем шаблон с переменными: 


![BTGE1f.png](images/BTGE1f.png)


  

В итоге переменные документа автоматически размножились на нужное количество записей из списка: 


![K7xztH.png](images/K7xztH.png)


  

Размножение строк происходит для переменных, которые могут хранить список значений: 


  * Переменные для полей [ аналитики](Аналитика/Аналитика.md) ([пример использования](Документы/Таблицы_в_шаблонах_документов.md)).

  * Переменные для полей типа [Набор значений справочника](Аналитика/Набор_значений_справочника.md).

  * Переменная **Файлы задачи** ([пример использования](Задачи/Вывод_в_документ_всех_файлов_задачи.md)).

  * Переменные для пользовательского поля типа **Файл** ([пример использования](Документы/Вывод_в_документ_всех_файлов_пользовательского_поля_типа_Файл.md)).

  * Переменные полей Подзадач ([пример использования](Задачи/Вывод_в_документ_данных_полей_подзадач.md)).

  * Переменная **Исполнители**.


## Важно


Существует различие в формировании документа по шаблонам, созданным в MS Word и MS Excel: 


  * при формировании документа по шаблону MS Word, строки будут размножены только в том случае, если переменная вставлена в таблицу;

  * при формировании документа по шаблону MS Excel, строки будут размножены в любом случае, вне зависимости от места, куда была вставлена переменная.


## Дополнительная информация


  * Так как при использовании переменных, автоматически размножающих строки, размер итогового документа может изменяться, существует специальный механизм вставки [ печати, подписи и других изображений](Документы/Печать_и_подпись_в_шаблонах_документов.md), который обеспечивает их размещение в нужном месте документа.
