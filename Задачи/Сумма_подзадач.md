[ Поля](Пользовательские_поля_задачи.md) данного типа служат для подсчета и хранения суммарных данных по выбранному полю [подзадачи](Подзадачи.md). Изменение выбранного для подсчета поля в любой из подзадач приводит к перерасчету оперирующих этим полем полей типа **Сумма подзадач** во всех надзадачах данной подзадачи. 


## Настройка


  * В простом варианте для настройки поля типа **Сумма подзадач** достаточно выбрать поле подзадачи, по которому будет производиться суммирование:


![UtCSTo.png](images/UtCSTo.png)


  


  * Также есть возможность ограничить набором условий подзадачи, по которым будет производиться суммирование, выбрав вариант **Подзадачи соответствующие условию** :


![TQ6dB8.png](images/TQ6dB8.png)


  

Например, если указать такое условие, в поле будут считаться только данные из активных подзадач, а завершенные/отмененные и т.п. считаться не будут: 


![oi0DO0.png](images/oi0DO0.png)


## Параметры поля


  * **Только непосредственные подзадачи** — суммироваться будут данные только непосредственных подзадач надзадачи.
