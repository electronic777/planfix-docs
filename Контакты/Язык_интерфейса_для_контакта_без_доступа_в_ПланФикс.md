Установка языка для внешнего контакта без доступа в ПланФикс влияет на: 


  * Язык [ уведомлений из системы](../Коммуникации/Подписки_и_уведомления.md), которые он получает

  * Язык в [ Личном кабинете](../Общие/Личный_кабинет.md), который автоматически доступен всем контактам


## Установка языка интерфейса для контакта без доступа в ПланФикс


При создании контакта выберите нужный язык интерфейса в его карточке. 


Изменить язык интерфейса можно и в процессе редактирования карточки контакта. С момента сохранения нового значения, контакт начнет получать уведомления и работать в личном кабинете на выбранном языке.
