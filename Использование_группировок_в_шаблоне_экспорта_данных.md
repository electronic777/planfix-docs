Для того, чтобы данные в итоговом файле были сгруппированы, необходимо использовать специальные переменные. Эти переменные доступны в отчете, в котором есть группировки. Для каждой группировки в [ списке переменных](Вкладка__Экспорт__в_отчете.md "Вкладка "Экспорт" в отчете") есть отдельная переменная вида 


**Отчет.Столбец::Группировка N**


где N это порядковый номер группировки. 


  

Порядковый номер определяется местом группировки в перечне группировок на вкладке отчета **Параметры отбора**. Например, для таких настроек группировки: 


![4DJMQs.png](images/4DJMQs.png)


  


  * Группировка по проектам будет иметь номер 1

  * Группировка по задачам будет иметь номер 2


  

В шаблон экспорта отчета каждая переменная для группировки занимает отдельный столбец таблицы MS Excel. Например, группировки по проектам и задачам в файле шаблона могут выглядеть так: 


![ROVkV0.png](images/ROVkV0.png)


  

Результат экспорта данных из отчета по такому шаблону будет выглядеть следующим образом: 


![KD2jvN.png](images/KD2jvN.png)
