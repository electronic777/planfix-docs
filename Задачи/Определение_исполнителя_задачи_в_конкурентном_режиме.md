ПланФикс позволяет определять [ исполнителя задачи](Исполнители_задачи.md) из списка по принципу «Кто первый принял». Настраивается эта возможность в дополнительных параметрах шаблона задачи — поле **Подключение исполнителей**. 


Выберите для этого один из пунктов: 


  * Первый принявший — исполнитель, остальные — участники.

  * Первый принявший — исполнитель, остальные — отключаются.


## Как это работает


  * При создании задачи в список исполнителей вносится произвольный набор пользователей и/или групп пользователей.

  * Первый пользователь, принявший задачу, становится ее реальным исполнителем.

  * Остальные пользователи и группы, первоначально добавленные в список исполнителей, в зависимости от выбранного варианта либо отключаются, либо переносятся в участники задачи (если нужно сохранить им доступ к ней).
