При помощи [ автоматических сценариев](Автоматизация/Автоматические_сценарии.md) вы можете добавлять собственные команды управления ПланФиксом из внешних источников (чатов, мессенджеров, электронной почты и т.п.) подключенных при помощи [ интеграций](Интеграции/Интеграции.md). 


Для этого используется автоматический сценарий, срабатывающий на событие добавления нового комментария и анализирующий его текст. Из текста выделяется слово-команда и выполняются заложенные вами действия.
