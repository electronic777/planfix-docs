**СУММА()** — это функция, суммирующая все значения при вычислении [формулы](../Поля и формы/Формулы.md). Она применяется для каждого элемента множащейся переменной. 


  


## Формат


  * СУММА(выражение)

  * SUM(expression)


  

Аргументы: 


  * **выражение** — обязательный параметр, переменная или формула с итоговым значением суммирования нескольких строк.


  


## Пример


Например, у вас есть аналитика «Услуга», и вы хотите посчитать полную стоимость всех услуг. Для этого укажите функцию: 

    

    

    СУММА({{Аналитика.Услуга.Стоимость}})

    


  

Для того, чтобы получить полную стоимость оказанных услуг, которые могут предоставляться несколько раз, необходимо стоимость умножить на количество, затем всё суммировать. В итоге укажите такую формулу: 


СУММА({{Аналитика.Услуга.Стоимость}}*{{Аналитика.Услуга.Количество}})


  


## Важно


  * Функция СУММА() может содержать внутри себя любую формулу, состоящую из функций.
