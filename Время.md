Поля данного типа служат для хранения значений времени. Создать поле типа **Время** можно в качестве [ пользовательского поля](Пользовательские_поля.md "Пользовательские поля"), поля [ справочника](Справочники.md "Справочники") и поля [ аналитики](Аналитика.md "Аналитика"). 


## Параметры поля:


  * **Автоматически устанавливать текущее время** — позволяет при создании задачи автоматически устанавливать в поле текущее время.


## Дополнительные параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения;

  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.


## Дополнительные параметры для поля этого типа в аналитиках:


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.
