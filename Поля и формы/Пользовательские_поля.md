ПланФикс позволяет добавлять в различные объекты системы дополнительные поля (реквизиты) [ различных типов](Поля и формы/Типы_пользовательских_полей.md). Это один из важнейших инструментов настройки системы в соответствии с [Идеологией ПланФикса](https://planfix.ru/ideologiya/). 


## Использование


Пользовательские поля позволяют хранить и обрабатывать в системе данные разных типов. В ПланФиксе есть несколько сущностей, в которые можно добавить пользовательские поля: 


  * [Пользовательские поля задачи](Задачи/Пользовательские_поля_задачи.md) (используются в [Объектах](Прочее/Объекты.md))

  * [Пользовательские поля проекта](Проекты/Пользовательские_поля_проекта.md)

  * [Пользовательские поля контактов](Контакты/Пользовательские_поля_контактов.md)

  * [Пользовательские поля сотрудника](Контакты/Пользовательские_поля_сотрудника.md)


## Типы и возможности пользовательских полей


  * Вы можете добавлять пользовательские поля [ различных типов](Поля и формы/Типы_пользовательских_полей.md).

  * В том числе есть возможность использовать [вычисляемые поля](Прочее/Вычисляемые_поля.md), которые предназначены для вычисления значений по [ формулам](Прочее/Стандартные_функции.md) и могут оперировать данными из других полей, [ справочников](Аналитика/Справочники.md), системных переменных и т.п.

  * Вы можете использовать [Фильтры в пользовательских полях](Интерфейс/Фильтры_в_пользовательских_полях.md) некоторых типов для вывода в них ограниченного набора значений, соответствующих настроенным вами условиям.
