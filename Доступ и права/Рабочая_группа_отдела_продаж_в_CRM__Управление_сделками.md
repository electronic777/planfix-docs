Для удобства работы в ПланФиксе сотрудников объединяют в различные [рабочие группы](Прочее/Рабочие_группы.md). В конфигурацию уже добавлена рабочая группа **Менеджеры по продажам**. При создании сделки вручную группу можно указать в качестве исполнителя, и первый из группы, кто примет сделку в работу, станет её единственным исполнителем. Если сделка попадает в ПланФикс по любому каналу связи, то группа **Менеджеры по продажам** автоматически добавляется в исполнители. 


  * Добавить в группу своих сотрудников можно в разделе **Сотрудники** — **Рабочие группы** :


![Z1bNhr.png](images/Z1bNhr.png)


  

Здесь же можно создать новые группы, а также задать [правила для задач по почте для рабочих групп](Доступ и права/Правила_для_групп.md).
