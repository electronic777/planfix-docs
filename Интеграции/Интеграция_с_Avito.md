[ПланФикс](https://planfix.ru/) позволяет вести переписку с клиентами, которые пишут сообщения по вашим объявлениям на [Авито](https://www.avito.ru/). Это дает возможность обрабатывать заказы из Авито в едином интерфейсе, наряду с заказами из других чатов, социальных сетей и мессенджеров, а также пользоваться [дополнительными преимуществами интеграций](https://planfix.ru/integrations/). Интеграция использует официальный API Авито. 


  

Вопрос клиента, заданный в чате Авито, попадает в ПланФикс. Ваши сотрудники видят все запросы клиентов в общем списке и общаются с ними. Каждый диалог в Авито может стать заказом или сделкой - ПланФикс позволяет настроить нужный вам процесс. 


  


## Важно


  * Для доступа к API необходим [действующий тариф или подписка Авито](https://www.avito.ru/subscription?avcm=for_business).

  * Отправку файлов [API Авито](https://developers.avito.ru/api-catalog/messenger/documentation#operation/postSendMessage) пока не поддерживает.


  


## Получение API ключей в Авито


  * Перейдите на [портал разработчика Авито](https://developers.avito.ru/).

  * Найдите блок «Сервис собственной разработки».

  * Получите ключи в разделе «Для профессионалов» — «API»:


![Sejj1t.png](images/Sejj1t.png)


  


## Настройка интеграции в ПланФиксе


1\. Если вас еще нет своего ПланФикса, [зарегистрируйте аккаунт](https://planfix.ru/signup/?utm_source=docs_yandexchat&utm_medium=catalog&utm_campaign=yandexchat). 


2\. В ПланФиксе зайдите в раздел **Управление аккаунтом / Интеграции / Чаты / Avito** и нажмите **Добавить** : 


  

![jnICLi.png](images/jnICLi.png)


  

![8d50xW.png](images/8d50xW.png)


  

3\. Введите название, которое будете использовать для идентификации в ПланФиксе, а также полученные от Авито **Client ID** и **Client Secret** : 


  

![7AncDX.png](images/7AncDX.png)


  

4\. Выберите шаблоны, по которым будут создаваться задачи и контакты по интеграции. Помимо указанных в шаблоне задачи исполнителей, можно дополнительно подключить сотрудников-исполнителей, которые будут вести общение: 


  

![DEqssQ.png](images/DEqssQ.png)


  


## Работа интеграции


Пользователь пишет сообщение в Авито:
