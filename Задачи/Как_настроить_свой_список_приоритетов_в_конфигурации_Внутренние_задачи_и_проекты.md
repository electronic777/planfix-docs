В конфигурации [ Внутренние задачи и проекты](Конфигурация_Внутренние_задачи_и_проекты.md) установлен стандартный список приоритетов. Добавить свои приоритеты можно следующим образом: 


  * Открываем **Базовое рабочее пространство** (см. страницу [ Установка конфигурации](Установка_конфигурации_Внутренние_задачи_и_проекты.md)) и переходим в раздел **Управление аккаунтом** — **Настраиваемые поля** — **Поля задач** :


![X3K1th.png](images/X3K1th.png)


  


  * открываем на редактирование поле **Приоритет** :


![KIwszO.png](images/KIwszO.png)


  


  * вносим в поле значения, подходящие вам:


![XBPJJf.png](images/XBPJJf.png)


  


  * нажимаем кнопку "Сохранить". Теперь при создании задачи можно выбирать свой приоритет.
