К любому [ отчету](Отчеты.md) в ПланФиксе можно добавить один или несколько графиков следующих типов: 


  * [Гистограмма](../Общие/Гистограмма.md)

  * [График](График.md)

  * [Круговая диаграмма](Круговая_диаграмма.md)

  * [Комбинированная диаграмма](Комбинированная_диаграмма.md)

  * [Воронка](Воронка.md)

  * [Пирамида](../Общие/Пирамида.md)

  * [Накопительная диаграмма](Накопительная_диаграмма.md)

  * [График Ганта](График_Ганта.md)


## Добавление нового графика


Для этого в режиме создания или редактирования отчета нужно: 


  * Зайти во вкладку **Графики и диаграммы**.

  * Нажать **Добавить новый график** :


![ArM7rP.png](images/ArM7rP.png)


  


  * Дать графику **название**.

  * Выбрать **тип диаграммы** :


![uWo4sF.png](images/uWo4sF.png)


  


  * На вкладке **Данные для отображения** необходимо выбрать один или несколько **столбцов отчета** , по которым будет строиться график.

  * И **группировку** , которая будет использоваться:


![Cf5OK3.png](images/Cf5OK3.png)


    **Важно** : 


  * От выбранной группировки будет зависеть то, какие данные отобразятся на графике. Мы рекомендуем вначале указать эту группировку в общих параметрах отчета, запустить его и убедиться, что с этой группировкой он в основной таблице с данными в нужном вам столбце выводит именно те данные, которые вы ожидаете. После этого укажите в настройках графика этот же столбец и эту же группировку, чтобы получить эти же данные в графическом виде.

  * Подписи по оси Х всегда берутся из группировки установленной в настройках графика (или отчета, если на уровне графика группировка не задана). Если ни там, ни там данные не сгруппированы, то подписи будут взяты из исходной информации, по которой строится отчет.

  * Для графиков, содержащих несколько линий данных, изменяющихся во времени, например такого:


    ![63Tlcq.png](images/63Tlcq.png)


  


    необходимо вначале указать группировку по дате, а затем - по объекту, данные которого будут выводиться в линии графика, например по сотруднику:


  


    ![LkSufr.png](images/LkSufr.png)


  


  * На следующем этапе можно отрегулировать **внешний вид** графика:


![ORRSzF.jpg](images/ORRSzF.jpg)


    **Важно**


  * В зависимости от выбранного типа диаграммы, настройки могут выглядеть по-разному.

  * Высота графика влияет на его отображение: если сам график не отображается, а видны только подписи к нему, увеличьте высоту графика.


  * Вы также можете выбрать **расположение графика** в отчете. По умолчанию график располагается выше тела отчета, но его можно переместить:


![XKws5Q.gif](images/XKws5Q.gif)


## Дополнительно


  * В один отчет можно добавить несколько графиков, группируя данные отчета в разных разрезах. Подробнее в [блоге.](https://planfix.com/ru/blog/grafiki-v-raznyh-razrezah-v-odnom-otchete/)

  * У линейных графиков есть ползунки, перемещая которые можно “крупнее” увидеть нужный период на временной шкале.
