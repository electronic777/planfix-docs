**Аналитика** — это инструмент, который позволяет добавлять в ПланФикс различные реквизиты учета, которые изначально не вложены в систему, но нужны Вам для удобства управления бизнесом в Вашей компании. 


Использование аналитики значительно расширяет сферу применения ПланФикса и гибкость настройки его под нужды разных компаний. 


Аналитика используется в паре с функционалом [Отчеты](Отчеты.md "Отчеты"): с помощью аналитики пользователи вводят информацию в задачах, с помощью отчетов информация собирается и выводится в удобной для анализа форме.

## Полезно


  * Данные, введенные при помощи аналитик, могут автоматически суммироваться в задачах при помощи пользовательского поля [Итоги аналитик](Итоги_аналитик.md "Итоги аналитик").

  * С помощью [таймера](Таймер.md "Таймер") удобно добавлять аналитику для учёта рабочего времени.
