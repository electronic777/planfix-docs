Результаты выполнения [ отчетов](Отчеты.md "Отчеты") ПланФикса можно экспортировать. Доступны следующие форматы: 


  * XLSX

  * CSV

  * Google Таблица


Предварительно запустите отчет, чтобы увидеть кнопку **Экспорт**. 


## Дополнительно


  * При экспорте в MS Excel можно использовать [Экспорт данных из отчетов](Экспорт_данных_из_отчетов.md "Экспорт данных из отчетов").
