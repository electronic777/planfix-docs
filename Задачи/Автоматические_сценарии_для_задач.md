## Информация для пользователей новых аккаунтов


Для настройки бизнес-процессов в вашем аккаунте мы рекомендуем использовать [ автоматические сценарии Объекта](Автоматизация/Автоматические_сценарии_Объекта.md). Информация, представленная ниже сохранена в справке исключительно в целях поддержания обратной совместимости процессов, настроенных ранее в старых аккаунтах ПланФикса. 


## Обратите внимание


  * Вы можете отлаживать автоматические сценарии в [ режиме тестирования](Задачи/Режим_тестирования_сценариев_задач_и_контактов.md).

  * Сценарии доступны для всех тарифов. Количество сценариев, которые могут быть добавлены в рамках одного процесса, зависит от [ тарифа](Прочее/Какой_пакет_выбрать_.md), на котором находится ваш аккаунт. В бесплатной версии ПланФикса есть возможность добавить **только один** автоматический сценарий.

  * Если сценарий содержит операцию массового изменения задач (пример: **Изменить другие задачи - Задачи в одном проекте с текущей, по шаблону**), то такая операция будет применена максимум к 150 задачам, попадающим под условие срабатывания этой операции.
