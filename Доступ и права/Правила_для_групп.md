Администратор может задавать правила создания задач по e-mail для рабочей группы. Если правило создано для группы, оно будет распространяться на всех пользователей, которые будут добавлены в группу в последующем. 


Создание правил для группы происходит в карточке группы: 


раздел **Сотрудники** — **Рабочие группы** — **Нужная группа** — кнопка **Новое правило**. 


Это позволяет новым сотрудникам легко включаться в общую работу. Администратору достаточно добавить их в нужную группу. Правила группы будут применены к ним автоматически.
