**Карточка проекта** открывается по клику на название [ проекта](Проекты.md). Она состоит из двух панелей: основной (слева) и дополнительных панелей с информацией (справа). 


На основной панели отображается список задач проекта, а в панели с дополнительной информацией могут отображаться разнообразные данные: 


  * [Детали проекта](Детали_проекта.md)

  * [Документы проекта](Документы_проекта.md)

  * [Лог проекта](Лог_проекта.md)

  * [Контакты проекта](../Контакты/Контакты_проекта.md)

  * [Отчеты по проекту](Отчеты_по_проекту.md)

  * [Уведомления по проекту](Уведомления_по_проекту.md)
