ПланФикс позволяет добавлять в различные объекты системы дополнительные поля (реквизиты) [ различных типов](Типы_пользовательских_полей.md "Типы пользовательских полей"). Это один из важнейших инструментов настройки системы в соответствии с [Идеологией ПланФикса](https://planfix.ru/ideologiya/). 


## Использование


Пользовательские поля позволяют хранить и обрабатывать в системе данные разных типов. В ПланФиксе есть несколько сущностей, в которые можно добавить пользовательские поля: 


  * [Пользовательские поля задачи](Пользовательские_поля_задачи.md "Пользовательские поля задачи") (используются в [Объектах](Объекты.md "Объекты"))

  * [Пользовательские поля проекта](Пользовательские_поля_проекта.md "Пользовательские поля проекта")

  * [Пользовательские поля контактов](Пользовательские_поля_контактов.md "Пользовательские поля контактов")

  * [Пользовательские поля сотрудника](Пользовательские_поля_сотрудника.md "Пользовательские поля сотрудника")


## Типы и возможности пользовательских полей


  * Вы можете добавлять пользовательские поля [ различных типов](Типы_пользовательских_полей.md "Типы пользовательских полей").

  * В том числе есть возможность использовать [вычисляемые поля](Вычисляемые_поля.md "Вычисляемые поля"), которые предназначены для вычисления значений по [ формулам](Стандартные_функции.md "Стандартные функции") и могут оперировать данными из других полей, [ справочников](Справочники.md "Справочники"), системных переменных и т.п.

  * Вы можете использовать [Фильтры в пользовательских полях](Фильтры_в_пользовательских_полях.md "Фильтры в пользовательских полях") некоторых типов для вывода в них ограниченного набора значений, соответствующих настроенным вами условиям.
