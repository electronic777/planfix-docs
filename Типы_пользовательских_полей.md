В ПланФиксе существуют следующие типы [ пользовательских полей](Пользовательские_поля.md "Пользовательские поля"): 


  * [Строка](Строка.md "Строка") — предназначен для хранения отдельных коротких текстовых значений (обычно отдельных слов или коротких словосочетаний)

  * [Число](Число.md "Число") — предназначен для хранения числовых значений

  * [Текст](Текст.md "Текст") — предназначен для хранения объемных текстовых значений

  * [Дата](Дата.md "Дата") — предназначен для хранения указаний на конкретную календарную дату

  * [Время](Время.md "Время") — предназначен для хранения указаний на конкретное время в течение дня (но не объем времени — для него лучше использовать тип [Число](Число.md "Число"))

  * [Дата и время](Дата_и_время.md "Дата и время") — сочетает в одном поле указание на конкретную дату и время

  * [Период времени](Период_времени.md "Период времени") — предназначен для хранения временного отрезка — начала и окончания какого-то конкретного временного периода в течение дня

  * [Чекбокс](Чекбокс.md "Чекбокс") — предназначен для хранения логических данных типа Да/Нет (True/False)

  * [Список](Список.md "Список") — предназначен для хранения списка значений, которые потом будут выбираться из выпадающего списка

  * [Набор значений](Набор_значений.md "Набор значений") — предназначен для ввода и хранения наборов данных, подразумевающих возможность выбора нескольких значений одновременно

  * [Файлы](Файлы_-_Тип_поля.md "Файлы - Тип поля") — предназначен для хранения ссылок на файлы, загруженные в ПланФикс.

  * [Местоположение](Местоположение.md "Местоположение") — предназначены для преобразования адресов в геоданные и хранения этих данных.

  * [Проект ](Проект_-_Тип_поля.md "Проект - Тип поля") — предназначен для хранения ссылок на проекты.

  * [Задача](Задача_-_Тип_поля.md "Задача - Тип поля") — предназначен для хранения ссылок на задачу.

  * [Набор задач](Набор_задач.md "Набор задач") — предназначен для хранения ссылок на несколько задач.

  * [Запись справочника](Запись_справочника_-_Тип_поля.md "Запись справочника - Тип поля") — предназначен для хранения ссылки на конкретную запись выбранного справочника

  * [Набор значений справочника](Набор_значений_справочника.md "Набор значений справочника") — предназначен для хранения нескольких ссылок на записи выбранного справочника

  * [Контакт](Контакт.md "Контакт") — предназначен для хранения указания на конкретный [ контакт](Контакты.md "Контакты")

  * [Сотрудник](Сотрудник.md "Сотрудник") — предназначен для хранения указания на конкретного сотрудника

  * [Контрагент](Контрагент_-_Тип_поля.md "Контрагент - Тип поля") — предназначен для хранения указания на конкретного [ контрагента](Контрагент.md "Контрагент")

  * [Группа, сотрудник или контакт](Группа_сотрудник_или_контакт.md "Группа сотрудник или контакт") — предназначен для хранения конкретной группы, сотрудника или контакта

  * [Список пользователей](Список_пользователей.md "Список пользователей") — предназначен для хранения в поле нескольких пользователей (контактов или сотрудников)

  * [Вычисляемые поля](Вычисляемые_поля.md "Вычисляемые поля") — предназначены для вычисления значений по [ формулам](Стандартные_функции.md "Стандартные функции"), оперирующим данными из других полей, [ справочников](Справочники.md "Справочники"), системных переменных и т.п.

  * [Итоги аналитик](Итоги_аналитик.md "Итоги аналитик") — предназначен для хранения итоговых данных конкретной аналитики.

  * [Сумма подзадач](Сумма_подзадач.md "Сумма подзадач") — предназначен для хранения итоговых данных, подсчитываемых по определенному полю подзадач/

  * [Суммарное поле](Суммарное_поле_проекта.md "Суммарное поле проекта") — предназначен для подсчёта и отображения агрегированных данных по задачам проекта.
