Для вашего удобства [пользовательские поля](../Задачи/Пользовательские_поля_задачи.md) на тарифах Бизнес и Корпорация объединяются в **Наборы полей**. 


В каждом аккаунте по умолчанию существует **Общий набор полей**. 


Мы рекомендуем придерживаться таких принципов: 


  * В общем наборе храните поля, которые могут использоваться в разных бизнес-процессах.

  * В отдельном наборе храните поля, которые используются в одном бизнес-процессе.


Для большинства аккаунтов общего набора достаточно и все поля создаются в нем. 


## Создание набора полей


Создание набора происходит в разделе **Управление аккаунтом** — **Настраиваемые поля** — **Поля задач** — **Новый набор полей**. 


  * Созданный набор выглядит как папка.

  * После создания набора создавать новые поля можно непосредственно в нем.

  * Набор можно быстро выбрать или создать прямо в интерфейсе создания поля:

## Перенос полей между наборами


Поля можно переносить между наборами. Для этого выберите поле и в дополнительном меню сверху нажмите кнопку **Перенести в набор полей**. 


## Ограничения наборов полей


  * Наборы полей создаются отдельно для каждой сущности — наборы полей [задач](../Задачи/Пользовательские_поля_задачи.md), [контактов](../Контакты/Как_добавить_новое_поле_в_карточку_контакта.md), [проектов](../Проекты/Пользовательские_поля_проекта.md).

  * Максимальное количество полей в наборе - **150**. Реальное количество для каждого набора зависит от типов входящих в него полей и может быть меньше.

  * В одном шаблоне (например, шаблоне задачи) вы можете использовать только поля из общего набора + одного дополнительного. Таким образом, максимально возможное количество пользовательских полей в одном шаблоне — до 300 (до 150 из общего + до 150 из дополнительного). При этом общее количество полей в аккаунте определяется вашим тарифом.

  * Поля можно переносить в другой набор при условии, что не нарушается правило из предыдущего пункта. То есть, перенос будет возможен только если в результате в существующих задачах и шаблонах будут находиться поля из базового + не более чем одного дополнительного набора полей.

  * Наборы полей есть только на тарифах «Бизнес» и «Корпорация».

  * Массово перенести несколько полей сразу в один набор система не позволяет, т.к. вы можете выбрать одновременно несколько полей, среди которых могут быть поля, которые уже находятся в задачах вместе с полями принадлежащими к разным наборам. Список доступных наборов полей, куда их можно перенести в таком случае будет разный для этих полей, поэтому это не может быть сделано массово для нескольких полей.
