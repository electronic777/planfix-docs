Для работы в [ПланФиксе](https://planfix.ru/) регистрируется отдельный **Аккаунт** \- выделенное информационное пространство для работы компании, обладающее определенными [ресурсами](https://planfix.ru/prices/). 


Аккаунт регистрируется человеком, который автоматически становится [ управляющим аккаунта](Доступ и права/Управляющий_аккаунтом.md). Мы настоятельно рекомендуем придерживаться правила "Управляющим аккаунта должен быть один из руководителей компании". Если аккаунт был зарегистрирован другим лицом, права управляющего следует [ передать](Доступ и права/Передача_прав_управляющего_аккаунтом.md) руководителю. 


## Название аккаунта


Название аккаунта задается управляющим аккаунта при регистрации аккаунта. 


Как правило, название аккаунта соотносится с названием компании или торговой марки. Аккаунт в том числе представляет собой домен 3 уровня на одном из доменов ПланФикса (planfix.com, planfix.ru, planfix.ua), поэтому название аккаунта может содержать только символы, допустимые для имени домена: латинские буквы, цифры и некоторые спецсимволы. 


**Пример:**


  * Для **ООО "Оза"** зарегистрирован аккаунт с названием **oza**

  * Домен для входа сотрудников в аккаунт: **<https://oza.planfix.ru/>**


Впоследствии название аккаунта может быть изменено по запросу управляющего (см. ниже) 


## Управление аккаунтом


Полное управление аккаунтом доступно только одному человеку - [ управляющему аккаунтом](Доступ и права/Управляющий_аккаунтом.md). Пользователи с [ правами](Доступ и права/Роли_в_ПланФиксе.md) администратора и технического администратора имеют ограниченный доступ к управлению аккаунтом. 


Ссылка на управление аккаунтом находится во всплывающем окне, которое открывается при нажатии на ваш аватар.


  

В разделе **Управление аккаунтом** размещены следующие вкладки: 


  * **Объекты** — позволяет создавать и настраивать [объект](Прочее/Объекты.md) — цифровой двойник физического объекта или процесса из реального мира внутри ПланФикса.

  * **Процессы** — позволяет создавать новые [процессы](Прочее/Процессы.md) и редактировать существующие.

  * **Статусы** — позволяет создавать и настраивать [ статусы](Задачи/Какой_смысл_в_статусах_задач_.md) задач.

  * **Аналитики** — позволяет настраивать работу с аналитикой и создавать новые [ аналитики](Аналитика/Аналитика.md).

  * **Настраиваемые поля** — для работы с полями [ задач](Задачи/Пользовательские_поля_задачи.md), [проектов](Проекты/Пользовательские_поля_проекта.md), [ контактов](Контакты/Как_добавить_новое_поле_в_карточку_контакта.md), [ сотрудников](Контакты/Пользовательские_поля_сотрудника.md).

  * **Рабочее время** — позволяет настраивать [рабочее время](Планирование/Рабочий_календарь.md) компании.

  * **Системные справочники**.

  * **Системные настройки** — позволяет устанавливать [ настройки](Настройка/Системные_настройки.md), которые будут распространяться на весь аккаунт.

  * **Конфигурации** — позволяет управлять доступными [ конфигурациями](Настройка/Конфигурации.md) ПланФикса.

  * **Интеграции** — позволяет подключить [интеграции](Интеграции/Интеграции.md) ПланФикса с внешними сервисами.

  * **Работа с e-mail** — содержит общие настройки аккаунта [ по работе с электронной почтой](Коммуникации/Настройка_исходящей_почты.md).

  * **Доступ к API** — позволяет управлять ключами, необходимыми для использования [ API ПланФикса](Интеграции/Разработчикам__API.md).

  * **Управляющий аккаунтом** — позволяет [ передать права управляющего](Доступ и права/Передача_прав_управляющего_аккаунтом.md) другому пользователю аккаунта.

  * **Безопасность аккаунта** — позволяет управлять использованием [ двухфакторной авторизации](Доступ и права/Двухфакторная_аутентификация.md) на уровне аккаунта и [настроить правила доступа к аккаунту](Доступ и права/Ограничение_доступа_по_IP.md) для разных пользователей.

  * **Экспорт данных** — позволяет экспортировать все данные аккаунта.

  * **Удалить аккаунт** — позволяет управляющему аккаунтом самостоятельно удалить аккаунт.


## Смена названия аккаунта


Процедура смены названия аккаунта производится бесплатно и доступна для: 


  * аккаунтов, находящихся в пробном периоде (в первые 14 дней после регистрации аккаунта)

  * платных и премиум аккаунтов


Проверить доступность желаемого имени вы можете, набрав адрес с ним в строке браузера: **_желаемое_имя_.planfix.ru**. Если вы увидите строку для ввода логина и пароля, значит это имя занято.


Если вы решили изменить название своего аккаунта, управляющему аккаунтом необходимо обратиться в [Службу поддержки ПланФикса](https://planfix.ru/support/) напрямую из аккаунта с указанием желаемого имени. Мы получим ваш запрос и изменим название аккаунта. 


После окончания процедуры переименования, все сотрудники вашего аккаунта получат уведомления на email с новым адресом для входа. 


Все прежние внутренние адреса с предыдущим названием аккаунта будут недоступны, почту на них получить вы не сможете. Вам также необходимо сделать следующее: 


  * Все настройки почтовой переадресации, правил обработки по почте и другое, где использовалось старое название аккаунта, необходимо будет сменить в соответствии с новым названием.

  * В случае использования API или интеграций (включая, но не только интеграцию с телефониями) — надо будет перенастроить заново.


## Полезно


  * Вы можете работать с данными из разных аккаунтов в одном, основном аккаунте, установив [ связь между аккаунтами](Общие/Связь_аккаунтов.md). Это позволит собрать в одном месте все задачи и события и обрабатывать их, не переключаясь в разные аккаунты.

  * Вы можете работать с пользователями других аккаунтов ПланФикса и ставить им задачи [ прямо из своего аккаунта](Задачи/Как_поставить_задачу_пользователю_другого_аккаунта_ПланФикса_.md) или [ при помощи email](Задачи/Как_поставить_задачу_по_e-mail.md).


## Важно


  * Неиспользуемые аккаунты ПланФикса регулярно удаляются. Обязательно ознакомьтесь с процедурой [ автоматического удаления аккаунта](Общие/Создание_и_удаление_аккаунта.md)
