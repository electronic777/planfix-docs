**Фильтры** в [ пользовательских полях](Пользовательские_поля.md "Пользовательские поля") некоторых типов позволяют выводить в списке доступных для выбора значений только те, которые соответствуют заданным условиям. 


Фильтры доступны в полях следующих типов: 


  * [Контакт](Контакт.md "Контакт")

  * [ Контрагент](Контрагент_-_Тип_поля.md "Контрагент - Тип поля")

  * [ Задача](Задача_-_Тип_поля.md "Задача - Тип поля")

  * [Набор задач](Набор_задач.md "Набор задач")


## Настройка фильтра в пользовательском поле


Для активации фильтра необходимо в настройках поля выбрать соответствующий вариант опции **Отображать в контроле выбора** и задать набор условий, которым должны соответствовать объекты, чтобы отобразиться в списке для выбора значения в этом поле.
