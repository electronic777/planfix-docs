Интеграция с сервисом [YouScore](https://youscore.com.ua/ru/) позволяет дополнить информацию в ПланФиксе открытыми данными о компаниях и контактах Украины. 


## Первые шаги


Для работы с сервисом необходимо получить ключ API. Для его подключения необходимо выполнить следующий порядок действий: 


  * Перейдите на [сайт сервиса](https://youscore.com.ua/ru/);

  * Нажмите на кнопку «Тестовый доступ»:


![XZSiZb.png](images/XZSiZb.png)


  


  * заполните форму:


![wlGeNZ.png](images/wlGeNZ.png)


  


  * когда представитель сервиса выйдет на связь, сообщите ему о том, что вам нужен API ключ для подключения в ПланФикс;

  * полученный тестовый ключ API введите в поле настройки интеграции в ПланФиксе при ее подключении;

  * после окончания срока действия ключа, при необходимости, приобретите подходящий пакет в сервисе YouScore.


  


## Настройка интеграции


  * Перейдите в раздел: **Управление аккаунтом** — **Интеграции** — **Сервисы автозаполнения данных** :


![0PsESV.png](images/0PsESV.png)


  


  * Кликните на значок YouScore — затем на кнопку добавления новой интеграции:


![zFLjrw.png](images/zFLjrw.png)


  


  * Откроется окно настроек интеграции:


![VLbV0X.png](images/VLbV0X.png)


  

Заполните следующие поля: 


  * **Название интеграции** — его вы указываете самостоятельно.

  * **Ключ API** — получаете по приложенной выше инструкции, вводите в это поле:

  * **Справочник / База данных для интеграции** — это база данных сервиса YouScore, по которому интеграция будет производить отбор.

  * **Применение автозаполнения** — выбираете, к контакту или к задаче будет производиться автозаполнение.

  * **Ключевое поле в ПланФиксе** — это поле, при изменении значения которого, интеграция начинает работу. Новое значение поля будет использовано интеграцией как значение параметра поиска.

  * **Соответствие полей для заполнения** — это блок, в котором производится сопоставление полей из базы данных YouScore с полями ПланФикса: 

    * **Источник данных** — это поле справочника/базы данных сервиса YouScore.

    * **Поле для заполнения** — поле задачи или контакта в ПланФиксе, в которое должна быть помещена информация из сервиса YouScore.
