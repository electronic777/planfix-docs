В ПланФиксе существуют следующие типы [ пользовательских полей](Пользовательские_поля.md): 


  * [Строка](../Общие/Строка.md) — предназначен для хранения отдельных коротких текстовых значений (обычно отдельных слов или коротких словосочетаний)

  * [Число](../Общие/Число.md) — предназначен для хранения числовых значений

  * [Текст](../Общие/Текст.md) — предназначен для хранения объемных текстовых значений

  * [Дата](../Общие/Дата.md) — предназначен для хранения указаний на конкретную календарную дату

  * [Время](../Время и учет/Время.md) — предназначен для хранения указаний на конкретное время в течение дня (но не объем времени — для него лучше использовать тип [Число](../Общие/Число.md))

  * [Дата и время](../Время и учет/Дата_и_время.md) — сочетает в одном поле указание на конкретную дату и время

  * [Период времени](../Время и учет/Период_времени.md) — предназначен для хранения временного отрезка — начала и окончания какого-то конкретного временного периода в течение дня

  * [Чекбокс](../Общие/Чекбокс.md) — предназначен для хранения логических данных типа Да/Нет (True/False)

  * [Список](../Интерфейс/Список.md) — предназначен для хранения списка значений, которые потом будут выбираться из выпадающего списка

  * [Набор значений](../Общие/Набор_значений.md) — предназначен для ввода и хранения наборов данных, подразумевающих возможность выбора нескольких значений одновременно

  * [Файлы](../Документы/Файлы_-_Тип_поля.md) — предназначен для хранения ссылок на файлы, загруженные в ПланФикс.

  * [Местоположение](../Общие/Местоположение.md) — предназначены для преобразования адресов в геоданные и хранения этих данных.

  * [Проект ](../Проекты/Проект_-_Тип_поля.md) — предназначен для хранения ссылок на проекты.

  * [Задача](../Задачи/Задача_-_Тип_поля.md) — предназначен для хранения ссылок на задачу.

  * [Набор задач](../Задачи/Набор_задач.md) — предназначен для хранения ссылок на несколько задач.

  * [Запись справочника](../Аналитика/Запись_справочника_-_Тип_поля.md) — предназначен для хранения ссылки на конкретную запись выбранного справочника

  * [Набор значений справочника](../Аналитика/Набор_значений_справочника.md) — предназначен для хранения нескольких ссылок на записи выбранного справочника

  * [Контакт](../Контакты/Контакт.md) — предназначен для хранения указания на конкретный [ контакт](../Контакты/Контакты.md)

  * [Сотрудник](../Контакты/Сотрудник.md) — предназначен для хранения указания на конкретного сотрудника

  * [Контрагент](../Общие/Контрагент_-_Тип_поля.md) — предназначен для хранения указания на конкретного [ контрагента](../Общие/Контрагент.md)

  * [Группа, сотрудник или контакт](../Контакты/Группа_сотрудник_или_контакт.md) — предназначен для хранения конкретной группы, сотрудника или контакта

  * [Список пользователей](../Интерфейс/Список_пользователей.md) — предназначен для хранения в поле нескольких пользователей (контактов или сотрудников)

  * [Вычисляемые поля](../Общие/Вычисляемые_поля.md) — предназначены для вычисления значений по [ формулам](../Общие/Стандартные_функции.md), оперирующим данными из других полей, [ справочников](../Аналитика/Справочники.md), системных переменных и т.п.

  * [Итоги аналитик](../Аналитика/Итоги_аналитик.md) — предназначен для хранения итоговых данных конкретной аналитики.

  * [Сумма подзадач](../Задачи/Сумма_подзадач.md) — предназначен для хранения итоговых данных, подсчитываемых по определенному полю подзадач/

  * [Суммарное поле](../Проекты/Суммарное_поле_проекта.md) — предназначен для подсчёта и отображения агрегированных данных по задачам проекта.
