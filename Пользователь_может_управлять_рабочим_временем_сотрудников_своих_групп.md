Администратор аккаунта может [ разрешить пользователю](Дополнительные_возможности_пользователя.md "Дополнительные возможности пользователя") управлять рабочим временем сотрудников своих рабочих групп. Для этого: 


  * зайдите в карточку нужного пользователя в разделе **Сотрудники** ;

  * нажмите кнопку **Редактировать** ;

  * разверните панель **Возможности** ;

  * активируйте признак **Может управлять рабочим временем сотрудников своих групп**.
