[Trello](https://trello.com/home) — это визуальный инструмент для управления работой в команде. Но когда хочется немного больше свободы действий и автоматизации процессов, работу можно перенести в ПланФикс. Для удобной и быстрой миграции в разделе [интеграции](../Интеграции/Интеграции.md) есть импорт данных из других сервисов. 


Для начала импорта перейдите в «Управление аккаунтом» — «Интеграции» — «Импорт данных из других сервисов»: 


![lEw1lF.png](images/lEw1lF.png)


  


  * Выберите «Trello». Откроется окно настройки импорта, в котором необходимо указать токен авторизации. Получить его можно по специальной ссылке:


![0owU8J.png](images/0owU8J.png)


    По клику на ссылку откроется всплывающее окно. Возможно ваш браузер блокирует такие окна, разрешите его открыть и повторите процедуру, если с первого раза не получилось.


  


  * Разрешите ПланФиксу доступ к вашему аккаунту Trello:


![bkASUV.png](images/bkASUV.png)


  


  * Токен авторизации будет добавлен автоматически:


![6mj0eA.png](images/6mj0eA.png)


  


  * На последнем шаге выберите доску, которая будет импортирована в ПланФикс:


![kEkxbe.png](images/kEkxbe.png)


  


  * Когда импорт завершится, в ПланФиксе вы увидите соответствующий планировщик:

## Важно


  * В ПланФиксе для задач из Trello создается новый процесс: «Trello — Имя доски».

  * В нём создаются статусы с именем равным столбцам в Trello. Если одноименный статус есть — используется он.

  * В новом процессе есть системный статус «Черновик» и статусы из доски Trello. Переходы доступны из всех статусов во все для постановщика, аудиторов, исполнителей.

  * По новому процессу создается планировщик.

  * Карточки Trello импортируются в качестве задач.

  * Участники импортируются в качестве контактов с базовым доступом в ПланФикс (ограниченный доступ через личный кабинет), либо используются созданные ранее контакты. Поиск существующего контакта происходит по e-mail, указанному в аккаунте Trello, при условии что у владельца токена авторизации есть к ним доступ в Trello. Если e-mail доступен, но не найден в ПланФиксе, будет создан новый контакт с базовым доступом. Если такой e-mail найден, будет использован ранее созданный контакт.

  * Для создания задач и контактов в ПланФиксе используются стандартные системные шаблоны.

  * В задачу ПланФикса импортируются только описание и прочие системные поля карточки Trello. Если будут запросы на импорт в том числе и пользовательских полей, реализуем это в следующей итерации.

  * В задачу также импортируются: файлы, прикрепленные к карточке Trello, чек-листы и комментарии.

  * По окончании импорта автоматически откроется созданный планировщик.

  * Токен авторизации сохраняется в ПланФиксе для удобного и быстрого запуска следующего импорта. Токен доступен только тому, кто его сохранял.
