На этом шаге вы настраиваете внешний вид отчета - таблицу, в которой должны отображаться полученные данные. Вы добавляете столбцы этой таблицы, которые будут заполняться данными. 


![5SQyp1.png](images/5SQyp1.png)


  

Для каждого столбца необходимо задать как минимум: 


  * Заголовок столбца

  * Тип объекта для вывода


В зависимости от выбранного типа объекта, может понадобиться дальнейшая детализация: система предложит вам доступные реквизиты для выбранного объекта, вам необходимо будет выбрать из них нужный. Пример доступных реквизитов для объекта **Задача** : 


![hA4WPh.png](images/hA4WPh.png)


  

**Примечание:** при выборе для столбца объекта **Аналитика** необходимо задать параметры отображения [ для каждой из аналитик, участвующих в отборе](Аналитика/Отчеты__настройка_вида_отображения_аналитик.md). 


## Использование формул


Один из доступных типов объекта для вывода в столбце - **Формула**. Подробнее об использовании этого типа объекта вы можете прочитать в справочной статье [ Формулы](Аналитика/Отчеты__Формулы.md). 


## Итоговый вид отчета


В итоге, у вас получится форма с набором столбцов отчета, которые и будут определять его вид и выводимые в нем данные: 


![yERqdK.png](images/yERqdK.png)


  

Прямо из окна настройки вы можете сохранить и запустить отчет, проанализировать полученные данные и вернуться в режим редактирования, чтобы внести в форму отчета нужные коррективы.
