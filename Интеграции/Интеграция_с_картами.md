ПланФикс может отображать на картах адреса и маршруты. Адреса (или названия объектов) вводятся в текстовом виде в [ пользовательские поля](Задачи/Пользовательские_поля_задачи.md) типа [Местоположение](Прочее/Местоположение.md). Отображение на карте производится в карточке [задачи](Задачи/Задачи.md), карточке [ контакта](Контакты/Контакты.md), а также [ фильтрах задач](Задачи/Фильтры_задач.md) и [ контактов](Контакты/Фильтры_контактов.md). 


## Поддерживаемые карты


На текущий момент поддерживается отображение адресов на картах **Google Maps** и **OpenStreet Maps**. При этом в обоих источниках карт для обработки текстовых адресов и перевода их в геоданные для отображения используется API Google Maps, поэтому обязательным условием подключения интеграции с картами является наличие у вас учетной записи Google. 


  * [Инструкция по подключению API Google Maps](Интеграции/Инструкция_по_подключению_API_Google_Maps.md)


## Включение интеграции


  * Перейдите в **Управление аккаунтом** — **Интеграции** — **Картографические сервисы**.

  * Нажмите **Включить интеграцию**.

  * Выберите одну из поддерживаемых карт и введите ключ **Google Geocoding API Key** , полученный при [ подключении API Google Maps](Интеграции/Инструкция_по_подключению_API_Google_Maps.md).

  * Сохраните настройки.


## Работа с картами


  * После включения интеграции, адреса и названия объектов, введенные в пользовательские поля типа [Местоположение](Прочее/Местоположение.md), будут помимо текстового вида также отображаться на выбранной карте:


![sxbnql.png](images/sxbnql.png)


  


  * Фильтры задач, содержащие задачи с полем типа [Местоположение](Прочее/Местоположение.md), могут отображать эти данные на карте. Для этого необходимо переключить вид отображение фильтра на **Местоположение** :


![SSdXDh.png](images/SSdXDh.png)


  

**Важно:** Для того, чтобы данные отображались на карте, поле с данными местоположения объекта должно присутствовать в списке [ отображаемых в фильтре полей](Задачи/Фильтры_задач.md). 


  


  * Данные задач в фильтре могут отображаться как в виде точек на карте, так и в виде маршрутов. Переключиться между этими вариантами можно тут:


![Kf7RHj.png](images/Kf7RHj.png)


  

Такая настройка возможна только для Google Maps, OpenStreet Maps предоставляет статичную карту. 


  * Порядок точек маршрута на карте определяется порядком расположения задач в фильтре.


## Обратите внимание


На текущий момент можно подключить только один картографический сервис.
