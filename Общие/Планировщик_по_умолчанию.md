Планировщик по умолчанию это планировщик, который загружается при клике на название раздела [Планировщик](Планировщик.md) в главном меню ПланФикса. 


Установить планировщик по умолчанию можно в выпадающем меню, которое открывается по клику на название этого планировщика:
