Вы можете вывести в формируемый по [ шаблону](Документы/Шаблоны_документов.md) документ все [файлы](Документы/Файлы.md), прикрепленные к задаче на момент его формирования. Для этого необходимо использовать переменную **{{Задача.файл задачи}}**


Для корректной работы, эту переменную необходимо вставить в [ табличную часть шаблона](Документы/Таблицы_в_шаблонах_документов.md), аналогично подходу который используется при вставке данных из прикрепленных к задаче аналитик. 


При формировании документа по шаблону, такая строка будет [ автоматически размножена](Автоматизация/Автоматическое_размножение_строк_документа.md) на количество файлов в задаче. 


У переменной **{{Задача.файл задачи}}** доступны следующие поля: 


  * порядковый номер записи

  * идентификатор

  * имя

  * имя без расширения

  * изображение - это сам файл физически, то есть в это место в документе будет вставляться картинка


## Полезно


  * Если в шаблоне формата MS Word вставить переменную **{{Задача.файл задачи}}** вне таблицы, в итоговый документ будет включена информация только по одному из файлов задачи.

  * Дополнительное форматирование изображений описано в справочной статье [Использование изображений в шаблонах документов](Документы/Использование_изображений_в_шаблонах_документов.md)
