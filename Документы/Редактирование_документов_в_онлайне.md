Документы форматов .XLS и .DOC можно редактировать прямо из интерфейса ПланФикса. У документа, для которого поддерживается онлайн-редактирование, на панели доступна иконка **Редактирование** : 


![Jmm9Rf.png](images/Jmm9Rf.png)


  


  * Чтобы воспользоваться этим функционалом, Вы должны иметь учетную запись Google.

  * После нажатия на иконку **Редактирование** ПланФикс предложит выбрать учетную запись Google, от имени которой будет происходить редактирование документа.

  * После авторизации откроется отдельное окно с редактируемым документом.

  * По окончанию работы просто закройте окно и подтвердите сохранение изменений в документе.


## Важно


  * Есть ограничение на размер файлов для онлайн-просмотра: для документов форматов .XLS — 5 Мб, .DOC — 10 Мб.

  * Для документов, загруженных в ПланФикс, совместное редактирование недоступно. Для совместного редактирования необходимо использовать ссылку на документ, не загружая его.

  * Шаблоны документов нельзя редактировать в Google Docs.
