Пирамида — это тип диаграммы в [отчетах](../Аналитика/Отчеты__Графики_в_отчетах.md) ПланФикса. Данные отображаются по иерархии в форме треугольника. Ширина каждого уровня отображает значение категории относительно других. 


Например, так может отображаться соотношение выполненных задач по проектам за месяц: 


![QMD0SK.png](images/QMD0SK.png)


  

Пирамида строится на основе данных из отчета и состоит из слоев. Каждый слой представляет определенную категорию и показывает ее числовое значение. 


Для того, чтобы отображались нужные данные — используется группировка и сортировка. 


## Как настроить


  * В «Вид отчета» добавьте столбцы с данными, по которым будет собрана пирамида.

  * Далее в разделе «Графики и диаграммы» добавьте новый график типа «Пирамида»:


![jVmVfz.png](images/jVmVfz.png)


  


  * В «Данных для отображения» укажите значения: 

    * Столбцы, по которым будет строиться график: «Выполненные задачи».

    * Группировка: По столбцу «По проектам» и по возрастанию.

    * Сортировка: По столбцу «Выполненные задачи» по месяцам и по возрастанию.


![KlNIUJ.png](images/KlNIUJ.png)


  


  * В «Внешнем виде» можно настроить: 

    * Высота графика — определяет высоту отображаемого графика в пикселях.


![BoZYTD.png](images/BoZYTD.png)


  * Сохраните график и запустите отчет для просмотра.
