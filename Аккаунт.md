Для работы в [ПланФиксе](https://planfix.ru/) регистрируется отдельный **Аккаунт** \- выделенное информационное пространство для работы компании, обладающее определенными [ресурсами](https://planfix.ru/prices/). 


Аккаунт регистрируется человеком, который автоматически становится [ управляющим аккаунта](Управляющий_аккаунтом.md "Управляющий аккаунтом"). Мы настоятельно рекомендуем придерживаться правила "Управляющим аккаунта должен быть один из руководителей компании". Если аккаунт был зарегистрирован другим лицом, права управляющего следует [ передать](Передача_прав_управляющего_аккаунтом.md "Передача прав управляющего аккаунтом") руководителю. 


## Название аккаунта


Название аккаунта задается управляющим аккаунта при регистрации аккаунта. 


Как правило, название аккаунта соотносится с названием компании или торговой марки. Аккаунт в том числе представляет собой домен 3 уровня на одном из доменов ПланФикса (planfix.com, planfix.ru, planfix.ua), поэтому название аккаунта может содержать только символы, допустимые для имени домена: латинские буквы, цифры и некоторые спецсимволы. 


**Пример:**


  * Для **ООО "Оза"** зарегистрирован аккаунт с названием **oza**

  * Домен для входа сотрудников в аккаунт: **<https://oza.planfix.ru/>**


Впоследствии название аккаунта может быть изменено по запросу управляющего (см. ниже) 


## Управление аккаунтом


Полное управление аккаунтом доступно только одному человеку - [ управляющему аккаунтом](Управляющий_аккаунтом.md "Управляющий аккаунтом"). Пользователи с [ правами](Роли_в_ПланФиксе.md "Роли в ПланФиксе") администратора и технического администратора имеют ограниченный доступ к управлению аккаунтом. 


Ссылка на управление аккаунтом находится во всплывающем окне, которое открывается при нажатии на ваш аватар.


  

В разделе **Управление аккаунтом** размещены следующие вкладки: 


  * **Объекты** — позволяет создавать и настраивать [объект](Объекты.md "Объекты") — цифровой двойник физического объекта или процесса из реального мира внутри ПланФикса.

  * **Процессы** — позволяет создавать новые [процессы](Процессы.md "Процессы") и редактировать существующие.

  * **Статусы** — позволяет создавать и настраивать [ статусы](Какой_смысл_в_статусах_задач_.md "Какой смысл в статусах задач?") задач.

  * **Аналитики** — позволяет настраивать работу с аналитикой и создавать новые [ аналитики](Аналитика.md "Аналитика").

  * **Настраиваемые поля** — для работы с полями [ задач](Пользовательские_поля_задачи.md "Пользовательские поля задачи"), [проектов](Пользовательские_поля_проекта.md "Пользовательские поля проекта"), [ контактов](Как_добавить_новое_поле_в_карточку_контакта.md "Как добавить новое поле в карточку контакта"), [ сотрудников](Пользовательские_поля_сотрудника.md "Пользовательские поля сотрудника").

  * **Рабочее время** — позволяет настраивать [рабочее время](Рабочий_календарь.md "Рабочий календарь") компании.

  * **Системные справочники**.

  * **Системные настройки** — позволяет устанавливать [ настройки](Системные_настройки.md "Системные настройки"), которые будут распространяться на весь аккаунт.

  * **Конфигурации** — позволяет управлять доступными [ конфигурациями](Конфигурации.md "Конфигурации") ПланФикса.

  * **Интеграции** — позволяет подключить [интеграции](Интеграции.md "Интеграции") ПланФикса с внешними сервисами.

  * **Работа с e-mail** — содержит общие настройки аккаунта [ по работе с электронной почтой](Настройка_исходящей_почты.md "Настройка исходящей почты").

  * **Доступ к API** — позволяет управлять ключами, необходимыми для использования [ API ПланФикса](Разработчикам__API.md "Разработчикам: API").

  * **Управляющий аккаунтом** — позволяет [ передать права управляющего](Передача_прав_управляющего_аккаунтом.md "Передача прав управляющего аккаунтом") другому пользователю аккаунта.

  * **Безопасность аккаунта** — позволяет управлять использованием [ двухфакторной авторизации](Двухфакторная_аутентификация.md "Двухфакторная аутентификация") на уровне аккаунта и [настроить правила доступа к аккаунту](Ограничение_доступа_по_IP.md "Ограничение доступа по IP") для разных пользователей.

  * **Экспорт данных** — позволяет экспортировать все данные аккаунта.

  * **Удалить аккаунт** — позволяет управляющему аккаунтом самостоятельно удалить аккаунт.


## Смена названия аккаунта


Процедура смены названия аккаунта производится бесплатно и доступна для: 


  * аккаунтов, находящихся в пробном периоде (в первые 14 дней после регистрации аккаунта)

  * платных и премиум аккаунтов


Проверить доступность желаемого имени вы можете, набрав адрес с ним в строке браузера: **_желаемое_имя_.planfix.ru**. Если вы увидите строку для ввода логина и пароля, значит это имя занято.


Если вы решили изменить название своего аккаунта, управляющему аккаунтом необходимо обратиться в [Службу поддержки ПланФикса](https://planfix.ru/support/) напрямую из аккаунта с указанием желаемого имени. Мы получим ваш запрос и изменим название аккаунта. 


После окончания процедуры переименования, все сотрудники вашего аккаунта получат уведомления на email с новым адресом для входа. 


Все прежние внутренние адреса с предыдущим названием аккаунта будут недоступны, почту на них получить вы не сможете. Вам также необходимо сделать следующее: 


  * Все настройки почтовой переадресации, правил обработки по почте и другое, где использовалось старое название аккаунта, необходимо будет сменить в соответствии с новым названием.

  * В случае использования API или интеграций (включая, но не только интеграцию с телефониями) — надо будет перенастроить заново.


## Полезно


  * Вы можете работать с данными из разных аккаунтов в одном, основном аккаунте, установив [ связь между аккаунтами](Связь_аккаунтов.md "Связь аккаунтов"). Это позволит собрать в одном месте все задачи и события и обрабатывать их, не переключаясь в разные аккаунты.

  * Вы можете работать с пользователями других аккаунтов ПланФикса и ставить им задачи [ прямо из своего аккаунта](Как_поставить_задачу_пользователю_другого_аккаунта_ПланФикса_.md "Как поставить задачу пользователю другого аккаунта ПланФикса?") или [ при помощи email](Как_поставить_задачу_по_e-mail.md "Как поставить задачу по e-mail").


## Важно


  * Неиспользуемые аккаунты ПланФикса регулярно удаляются. Обязательно ознакомьтесь с процедурой [ автоматического удаления аккаунта](Создание_и_удаление_аккаунта.md "Создание и удаление аккаунта")
