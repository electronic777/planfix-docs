Круговая диаграмма — это тип диаграммы в [отчетах](Отчеты__Графики_в_отчетах.md) ПланФикса. Данные отображает в виде сегментов круга. Каждый сегмент соответствует определенной категории и показывает её долю в общем объеме. 


Например, так могут отображаться выполненные задачи сотрудников за месяц: 


![y3Dti2.png](images/y3Dti2.png)


  

Круговая диаграмма строится на основе данных из отчета и состоит из круга, разделенного на секторы: 


  * Каждый сектор представляет категорию (например, отдел, проект, статус задачи).

  * Размер сектора пропорционален числовому значению (например, количество задач, процент бюджета, доля времени).


Для того, чтобы отображались нужные данные — используется группировка и сортировка. 


## Как настроить


  * В «Вид отчета» добавьте столбцы с данными, по которым будет собрана круговая диаграмма.

  * Далее в разделе «Графики и диаграммы» добавьте новый график типа «Круговая диаграмма».


    ![SyqVS4.png](images/SyqVS4.png)


  


  * В «Данных для отображения» укажите значения: 

    * Столбцы, по которым будет строиться график: «Выполненные задачи».

    * Группировка: По столбцу «Исполнитель».

    * Сортировка: По столбцу «Выполненные задачи» по месяцам и по возрастанию.


    ![i7GACJ.png](images/i7GACJ.png)


  


  * Во «Внешнем виде» можно настроить: 

    * Высота графика — определяет высоту отображаемого графика в пикселях.


    ![52BRgt.png](images/52BRgt.png)


  


  * Сохраните график и запустите отчет для просмотра.
