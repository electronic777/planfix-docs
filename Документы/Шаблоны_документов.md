**Шаблон документа** — это файл, в котором вы можете использовать поля из [задач](../Задачи/Задачи.md) и [аналитик](../Аналитика/Аналитика.md) ПланФикса. Для этого в шаблон документа вставляются специальные [ переменные](Переменные_в_шаблонах_документов.md). 


Когда по шаблону формируется документ, в эти переменные автоматически подставляются значения из текущей задачи или комментария. В результате вы получаете готовый документ с нужными данными, который сразу прикрепляется к задаче. 


Поддерживаются следующие типы шаблонов документов: 


  * Word

  * Excel

  * Текстовые файлы, например: txt, xml, html, mmd и другие.


  

Обратите внимание: 


  * Шаблон документа должен быть не более 15 МБ.


## Создание шаблона документа


  * Для создания и редактирования шаблонов документов Excel или Word мы рекомендуем использовать соответствующие программные пакеты или бесплатный пакет [LibreOffice](https://ru.libreoffice.org/).


    **Важно:** использование других редакторов, например онлайн-редакторов Google, не гарантирует работоспособность получившегося шаблона.


  * Список [ переменных](Переменные_в_шаблонах_документов.md), содержащих данные задач, аналитик и других сущностей ПланФикса, которые можно использовать в шаблонах, доступен в разделе **Документы** — **Шаблоны документов**.

  * Для начала использования готовый шаблон необходимо загрузить в ПланФикс. Это можно сделать в разделе **Документы** — **Шаблоны документов** — **Новый шаблон**.


## Работа с шаблонами документов


  * По готовому шаблону можно создать документ, который будет содержать данные из задачи и сразу прикрепится к ней. Создание документа по шаблону происходит **в интерфейсе добавления файла к задаче** :


![hyQJlD.png](images/hyQJlD.png)


  

или же в интерфейсе [ автоматического сценария](../Автоматизация/Автоматические_сценарии.md): 


![cKoPC0.png](images/cKoPC0.png)


  

При создании документа по шаблону есть возможность: 


  * выбрать формат итогового файла, который будет прикреплен к задаче;

  * изменить номер документа (он автоматически увеличивается на 1 при каждом новом формировании, но иногда нужно перескочить на другую нумерацию);

  * изменить дату документа (если это необходимо).


## Задание имени документа, созданного по шаблону


  * В имени создаваемого по шаблону документа можно использовать различные переменные, их список можно открыть кликом по специальному значку:


![kRCA8h.gif](images/kRCA8h.gif)


  * С помощью переменных составляется имя файла любой степени сложности.

  * При создании документа в название создаваемого файла подставятся реальные значения указанных переменных.


## Указание типа файла


  * В шаблоне можно указать тип файла, который будет присваиваться по умолчанию:


![TkOieY.png](images/TkOieY.png)


  


  * После этого документ будет создаваться согласно указанному типу, если при создании не указать иной тип вручную:


![mwTyh2.png](images/mwTyh2.png)
