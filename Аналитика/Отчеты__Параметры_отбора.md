На этом шаге вы можете настроить: 


  * Фильтр для отбора данных

  * Группировку и сортировку для отобранных данных


## Фильтр для отбора данных


Внешний вид настройки условий отбора данных для отчета точно такой же, как при настройке [ фильтров задач](Задачи/Фильтры_задач.md). Вы можете задать произвольное количество условий и указать логическую связь между ними в специально отведенном для этого поле: 


![qytz6N.jpg](images/qytz6N.jpg)


  

Подробнее о настройке фильтров и логической связи между ними можно прочитать [ по ссылке](Задачи/Фильтры_задач.md). 


**Примечание:** в случае отбора данных по аналитикам, вы можете настроить уникальные критерии отбора для каждой аналитики — это придает отчетам дополнительную [гибкость](Аналитика/Отбор_данных__дата_аналитики_или_дата_действия_.md). 


## Группировка и сортировка отобранных данных


Группировка и сортировка отобранных данных может быть задана сразу, либо после того, как будет настроен [ вид отчета](Аналитика/Отчеты__Вид_отчета.md). Тогда появляется возможность добавить сортировку и группировку по дополнительным столбцам:
