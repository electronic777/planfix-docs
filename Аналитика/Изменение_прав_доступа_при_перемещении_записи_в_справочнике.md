При перемещении записи [ справочника](Справочники.md) между группами, система предложит автоматически изменить права доступа для этой записи на те, которые установлены для группы, в которую она перемещается: 


  * Если вы согласитесь, права доступа к записи будут изменены на те, которые установлены для группы, в которую она перемещается.

  * Если вы нажмете "Не применять", то права доступа к записи останутся без изменений.

  * Если вы активируете чекбокс "Запомнить мой выбор", то в рамках текущего сеанса работы с этим справочником этот вопрос больше задаваться не будет и ко всем перемещаемым записям будет применен ваш выбор в текущей ситуации.


## Важно


  * Если вы перемещаете запись на высший уровень иерархии справочника (в корневую группу), то система предложит установить для нее права, аналогичные правам по умолчанию для новых записей, добавляемых в корневую группу справочника.
