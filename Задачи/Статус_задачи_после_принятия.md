Когда [ задача](Задачи/Задачи.md) находится в [ статусе "Новая"](Новая__(статус_задачи_.md) "Новая \(статус задачи\)"), в ней отображаются [ кнопки Принять и Отклонить](Поля и формы/Кнопки_Принять_и_Отклонить.md), при помощи которых исполнитель может взять задачу в работу, либо отказаться от нее: 


![7sTYrQ.png](images/7sTYrQ.png)


  

Настройка **Статус, в который будет переводиться задача по нажатию на кнопку "Принять"** позволяет выбрать статус, в который задача будет автоматически переведена после принятия. 


Эта настройка расположена в верхней части окна [ дополнительных параметров набора статусов](Задачи/Дополнительные_параметры_набора_статусов.md): 


![edZOPS.png](images/edZOPS.png)


  


## Дополнительная информация


  * В настройках по умолчанию, после принятия задача будет переведена в статус **В работе**.

  * Для каждого [ процесса](Прочее/Процессы.md) существует свой [ набор статусов](Задачи/Создание_набора_статусов.md), поэтому вы можете управлять тем, в какой статус будет переводиться задача после принятия, на уровне конкретного процесса.
