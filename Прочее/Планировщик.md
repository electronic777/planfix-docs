**Планировщик** — это основной инструмент ПланФикса для планирования работы над задачами. Главная особенность Планировщика ПланФикса: с его помощью можно не только отобрать и увидеть на одном экране нужные задачи, но и быстро изменить их параметры простым перетаскиванием из столбца в столбец. 


  * _**Пример.** В планировщике есть два столбца: задачи Петрова и задачи Сидорова. Если перетащить задачу из первого столбца во второй, то у нее изменится исполнитель с Петрова на Сидорова._


__


Какие именно параметры будут изменены, зависит от условий отбора задачи в столбце, в который вы перетащите задачу. Это дает очень широкие возможности по манипулированию задачами простым перетаскиванием из столбца в столбец. 


  * _**Полезно:** Подробнее о базовых принципах Планировщика можно узнать из **[расширенного видео](https://www.youtube.com/watch?v=AhR6BtSFMGQ)**._


## Устройство раздела


В разделе **Планировщик** вы можете создавать любое количество рабочих областей-планировщиков. Каждый такой планировщик может содержать произвольное количество списков задач. Перемещая задачи из одного списка в другой, вы изменяете параметры задачи.
