Для включения интеграции с сервисом почтовых рассылок [SendGrid](https://sendgrid.com/), перейдите в 


**Управление аккаунтом / Интеграции / Интеграции с сервисами почтовых рассылок** : 


![EGOLOp.jpg](images/EGOLOp.jpg)


  


  * В появившемся списке нажмите галочку активации в блоке SendGrid:


![HkPB2j.jpg](images/HkPB2j.jpg)


  


  * Откройте в соседней вкладке [SendGrid](https://sendgrid.com/) и войдите в него под своей учетной записью. В меню откройте **Settings / API Keys** и нажмите **Create API Key** :


![DjkiL9.jpg](images/DjkiL9.jpg)


  


  * В появившейся панели **дайте название ключу** и нажмите **Create &View**:


![0hBuu9.jpg](images/0hBuu9.jpg)


  


  * Скопируйте сгенерированный ключ:


![KXDkSL.jpg](images/KXDkSL.jpg)


  


  * Перейдите на страницу интеграции с SendGrid в ПланФиксе и вставьте его в соответствующее поле:


![j9GZdQ.jpg](images/j9GZdQ.jpg)


  

Сохраните изменения. После этого в каждом [ фильтре контактов](../Контакты/Фильтры_контактов.md) появится возможность включать автоматический экспорт отобранных контактов в SendGrid (см. [ справку](../Коммуникации/Сервисы_рассылки_e-mail.md)).
