Состав конфигурации [Согласование отпуска](Согласование_отпуска.md "Согласование отпуска") может отличаться для каждого установленного в вашем аккаунте экземпляра этой конфигурации, так как они содержат разные статусы задач (этапы согласования). Удобнее всего посмотреть состав нужного экземпляра конфигурации в списке установленных конфигураций по пути: **Управление аккаунтом** — **Конфигурации** — **Установленные**. 


Клик на названии конфигурации откроет панель с общим описанием конфигурации и перечнем объектов, автоматически созданных системой в процессе установки и настройки данного экземпляра конфигурации. 


Тип объекта | Название | Описание   

---|---|---  

Поле задачи | Дата начала отпуска | Содержит планируемую дату начала отпуска   

Поле задачи | Дата окончания отпуска | Содержит планируемую дату окончания согласуемого отпуска.   

Статус задачи | Этап согласования | Первый статус из цепочки статусов согласования отпусков, в который переходит задача, после ее создания автором   

Статус задачи | Оформление в бухгалтерии | Статус, в котором находится задача на очередном этапе согласования отпуска, в частности - Оформление в бухгалтерии   

Статус задачи | Не согласовано | Статус, в который переходит задача в случае отказа в согласовании по той или иной причине, после этого задача попадает на доработку   

Процесс | Согласование отпуска | [ Процесс](Процессы.md "Процессы"), который устанавливается вместе с конфигурацией   

Шаблон задачи | Согласование отпуска | Служит для создания задачи, инициализирующей процесс согласования отпуска   

  

  

Ссылки на процесс и шаблон задач позволяют быстро перейти к нужному объекту для его [ редактирования](Настройка_конфигурации_Согласование_отпуска.md "Настройка конфигурации Согласование отпуска").
