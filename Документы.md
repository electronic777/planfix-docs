В разделе **Документы** хранятся все документы, которые попали в ПланФикс [ одним из доступных способов](Файлы.md "Файлы"). 


Для удобства каталогизации документы, находящиеся в проектах, группируются по папкам [ проектов](Проекты.md "Проекты"), [ справочников](Справочники.md "Справочники") или [ контактов](Контакты.md "Контакты"), к которым они прикреплены. 


Остальные документы располагаются в папке **Документы без проекта**.
