Интерфейс импорта задач вызывается кнопкой **Импорт** в разделе [Задачи](Задачи/Задачи.md): 


  * поддерживаются исходные файлы форматов csv, xls, xlsx, а также файлы Microsoft Project.

  * после выбора файла система анализирует его и просит сопоставить имеющиеся в файле столбцы с реквизитами задачи в ПланФиксе. При этом, **если значение в первой строке столбца совпадает с названием реквизита задачи в ПланФиксе, сопоставление происходит автоматически** — это позволяет подготовить файл определенного формата, чтобы при частом импорте не приходилось каждый раз делать сопоставление вручную:


![67EJqO.png](images/67EJqO.png)


  


  * для корректного импорта данных в [ пользовательские поля задач](Задачи/Пользовательские_поля_задачи.md) необходимо выбрать в интерфейсе импорта шаблон задачи, содержащий эти поля.


![B3GQRe.png](images/B3GQRe.png)


  


  * если в качестве реквизитов задачи указаны несуществующие в системе **Контрагент** или **Проект** , в процессе импорта они будут созданы. Для более полного заполнения данных Контрагента рекомендуется перед импортом задач выполнить [импорт контактов](Контакты/Импорт_контактов.md).

  * при заполнении реквизитов **Исполнитель** , **Участник** , **Аудитор** будет произведена попытка найти среди существующих пользователей ПланФикса человека с именем и фамилией, совпадающими с данными из файла. Если такой человек найден не будет, реквизит останется незаполненным. В связи с этим рекомендуется вначале добавить нужных сотрудников в систему, а затем проверить точное совпадение имен в системе и в импортируемом файле.

  * при импорте есть возможность контролировать дублирование задач: 

    * **считать все задачи новыми** : создаются новые задачи без контроля дублей;

    * **игнорировать существующие задачи** : ПланФикс не создаст задачу если обнаружит совпадение по указанному полю;

    * **обновлять данные существующих задач при импорте** : ПланФикс обновит данные задачи если обнаружит совпадение по указанному полю.


![ZAz2rt.png](images/ZAz2rt.png)


  


  * ограничений на общее количество импортируемых задач отсутствует, но в одном файле может быть обработано не более 1000 строк.


## Важно


  * При импорте в поля, содержащие дату и время: 

    * для типа [Дата](Прочее/Дата.md) данные должны быть представлены в формате: **дд-мм-гггг**

    * для типа [Дата и время](Время и учет/Дата_и_время.md) данные должны быть представлены в формате **дд-мм-гггг мм:чч**

    * номер дня в виде одной цифры и год в виде двух цифр не поддерживаются

    * вместо символа "-" могут использоваться символы "." (точка) или "/" (слэш).

  * Для корректного импорта файлов формата CSV кодировка файла должна быть **UTF-8​**

  * Увидеть только что импортированные задачи можно, сделав новый [ фильтр](Задачи/Фильтры_задач.md) по условию **Дата создания задачи = Сегодня**

  * Из [Microsoft Project](Интеграции/Microsoft_Project.md) в ПланФикс импортируются только связи и даты.
