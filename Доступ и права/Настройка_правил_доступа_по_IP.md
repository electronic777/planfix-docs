## Название


Название необходимо, чтобы удобно оперировать им в списке правил. Используйте понятные названия, отражающие суть правила. 


  


## Правило действует


  * Для любой сети — используется для создания разрешительных правил «для всех», а также для правил, которые ограничивают работу сотрудников в нерабочее время.

  * Для выбранных хостов — позволяет задать перечень IP-адресов, с которых разрешена работа в аккаунте.

  * Для подсети — позволяет задать маску подсети, с которой разрешен доступ пользователей к аккаунту.


  


## Доступы


  * Для сотрудников

  * Для клиентов


В этом блоке управляющий аккаунтом задает список пользователей, на которых распространяется правило. 


  


## Важно


Если пользователь не попадет под действие ни одного из правил, он не получит доступа в ПланФикс. Будьте внимательны при создании правил. Напишите в [Службу поддержки](https://planfix.ru/support/), если испытываете затруднения.
