## Добавление кнопки


Для добавления кнопки необходимо указать: 


1\. В каких процессах может отображаться кнопка. 


2\. Набор условий, при которых она будет отображаться. 


3\. Кто сможет нажимать кнопку. Даже если условия будут выполняться, кнопка будет видна только указанным в этом поле пользователям. 


4\. Настройки внешнего вида кнопки. 


5\. Набор действий, которые будут выполнены по нажатию на кнопку.

## Добавление кнопки на основе сценария


Существует быстрый способ создать кнопку на основе существующего [ сценария](Автоматические_сценарии.md "Автоматические сценарии"). Для этого необходимо выбрать нужный сценарий и выбрать пункт **Скопировать как кнопку** из меню операций: 


![XeYFuF.png](images/XeYFuF.png)


  

При этом будет открыт интерфейс создания кнопки, в котором будут автоматически загружены соответствующие данные из выбранного сценария. При необходимости их можно отредактировать и дополнить.
