Поля данного типа служат для хранения информации о нескольких пользователях — сотрудниках или контактах. 


## Параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения.


## Параметры для поля этого типа в аналитиках:


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.


За отображение контакта в этом поле отвечает настройка [ Отображать в списке выбора участников задачи](Контакты/Как_работать_с_клиентом_по_e-mail_.md) в карточке контакта. 


## Важно


  * Поле может содержать максимум 250 значений.
