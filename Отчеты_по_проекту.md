На панели **Отчеты** [ карточки проекта](Карточка_проекта_.md "Карточка проекта:") отображается список всех доступных пользователю отчетов. Клик по названию отчета приведет к его запуску, при этом в параметры отбора данных отчета будет добавлено условие **Проект = <текущий проект>**. Это позволяет быстро запускать любой универсальный отчет по конкретному проекту, не изменяя его параметров. 


Аналогичные механизмы [ действуют в карточках задачи и контакта](Панель_Отчеты.md "Панель Отчеты").
