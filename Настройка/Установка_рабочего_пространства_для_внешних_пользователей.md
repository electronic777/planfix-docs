[ Внешние пользователи](../Контакты/Внешний_пользователь.md), как и сотрудники, могут работать в разных [ рабочих пространствах](../Общие/Рабочие_пространства.md), к которым им дан [ доступ](../Доступ и права/Доступ_к_рабочему_пространству.md). 


При этом действуют следующие правила: 


  * По умолчанию, все внешние пользователи работают с базовым рабочим пространством.

  * В случае, если внешнему пользователю дан доступ и установлено по-умолчанию другое рабочее пространство, вместо базового, он работает в нем, не видит пункта меню выбора рабочих пространств ("ракета") и не может переключиться на базовое рабочее пространство.


## Установка рабочего пространства по умолчанию для внешнего пользователя


Существует 3 способа установить рабочее пространство, в котором он будет оказываться по умолчанию, сразу при входе в ПланФикс: 


1\. В карточке внешнего пользователя (контакта), выбрав его в соответствующем параметре: 


![VgnE2t.jpg](images/VgnE2t.jpg)


  

2\. Массово установить рабочее пространство по умолчанию для списка нужных контактов: 


![1ev71x.jpg](images/1ev71x.jpg)


![Xg9NYg.jpg](images/Xg9NYg.jpg)


  

3\. В режиме создания или редактирования рабочего пространства установить его как рабочее пространство по умолчанию для всех контактов с доступом: 


![7Q9xV1.jpg](images/7Q9xV1.jpg)


**Важно:** этот вариант прописывает данное рабочее пространство во всех карточках контактов с доступом в ПланФикс и во всех [ шаблонах контактов и компаний](../Контакты/Шаблоны_контактов.md), чтобы созданные по ним впоследствии контакты автоматически получали это же рабочее пространство по умолчанию.. Прописанные таким образом значения можно изменить впоследствии любым из 3-х вышеописанных способов.
