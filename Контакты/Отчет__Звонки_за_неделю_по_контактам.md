Этот отчет служит для получения сводных данных о количестве и времени звонков за определенный промежуток времени, в разрезе по контактам. Он доступен после подключения интеграции с одной из [виртуальных АТС](Прочее/Виртуальные_АТС.md). 


Для запуска отчета перейдите в раздел "Отчеты" — "Отчеты по звонкам" — "Отчет: Звонки за неделю по контактам":
