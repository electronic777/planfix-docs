## Установка конфигурации


[ Установить конфигурацию](Настройка/Инструкция_по_установке_конфигурации.md) может администратор аккаунта. После этого планировщик «Сводные данные» станет доступным всем пользователям аккаунта. Для того, чтобы этот планировщик стал стартовым экраном при входе в ПланФикс, необходимо: 


1\. Установить раздел [Планировщик](Прочее/Планировщик.md) разделом по умолчанию в карточке пользователя: 


![AKS460.png](images/AKS460.png)


  

2\. Установить планировщик **Сводные данные** планировщиком по умолчанию:
