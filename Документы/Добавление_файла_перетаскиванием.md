[Файлы](Документы/Файлы.md) и изображения в [ задачу](Задачи/Задачи.md) можно добавить перетаскиванием. Для этого нужный файл достаточно перетащить в задачу. При этом: 


  * Если файл перетащить в карточку задачи в режиме просмотра, откроется интерфейс добавления нового комментария, к которому уже будет прикреплен файл.

  * Также можно добавить файл прямо в процессе написания комментария:


![kLx7Rb.png](images/kLx7Rb.png)


  


  * Если перетаскивать файл с изображением прямо в текст комментария, изображение будет вставлено прямо в текст:


![reFknY.png](images/reFknY.png) _Аналогичным образом можно вставить изображение в описание задачи в ходе ее создания или редактирования._


  


  * Также в текст можно вставлять картинку и прямо из буфера обмена — скопировать (Ctrl+С) ее на какой-то странице или в программе и вставить (Ctrl+V).


## Дополнительная информация


  * Можно перетаскивать в текст картинки из других источников (например, с веб-страницы, открытой в другой вкладке браузера).

  * Можно перетаскивать файлы внутри задачи, из комментария в комментарий или с панели **Документы** в комментарий.

  * Можно перетаскивать файлы из других задач, проектов или контактов, открытых в соседней вкладке браузера.
