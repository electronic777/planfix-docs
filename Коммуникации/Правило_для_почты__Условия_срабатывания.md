**Условия срабатывания** \- первый блок [ правила обработки почты](Задачи/Правила_создания_задач_по_e-mail.md), который необходимо заполнить при [ его создании](Доступ и права/Создание_и_редактирование_правил.md). Блок содержит условия, которым должно соответствовать письмо для срабатывания этого правила: 


![JFPsGB.jpg](images/JFPsGB.jpg)


  

При этом: 


  * Блок может содержать одно или несколько условий, соединенных логическими выражениями

  * Блок может не содержать ни одного условия, тогда правило будет срабатывать для всех писем, пришедших на адрес, на котором оно настроено. Но для большей наглядности мы рекомендуем в таких случаях использовать конструкцию **Любое письмо** :


    ![gzfgfl.jpg](images/gzfgfl.jpg)


  

Типы условий, используемых в блоке условий срабатывания: 


  * [Содержит текст](Прочее/Содержит_текст.md)

  * Не содержит текст

  * = (Равно)

  * [Содержит слово с меткой](Прочее/Содержит_слово_с_меткой.md) (устаревшее)
