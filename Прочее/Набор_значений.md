Поля данного типа служат для хранения любых текстовых значений, например тегов. Отличительной чертой этого типа поля является то, что в нем одновременно может быть указано несколько значений. При этом есть два способа добавлять значения в поле: 


  * Свободный ввод

  * Фиксированный список


## Свободный ввод


С помощью значения «Свободный ввод» все сотрудники компании могут самостоятельно вносить новые значения в список и изменять их. Например, в существующей задаче поле выглядит следующим образом:

## Фиксированный список


При «Фиксированном списке» сотрудник сможет выбрать значения только из заранее добавленного набора:

## Важно


Создать поле типа **Набор значений** можно в качестве [ пользовательского поля](Поля и формы/Пользовательские_поля.md), поля [ справочника](Аналитика/Справочники.md) и поля [ аналитики](Аналитика/Аналитика.md). 


## Параметры поля в справочниках


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения;

  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.


## Параметры поля в аналитиках


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.
