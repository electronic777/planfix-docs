Эта настройка находится в разделе: **Управление аккаунтом** — **Работа с помощью e-mail** — [ Настройка входящих писем](Коммуникации/Настройка_входящей_почты.md). 


Данная настройка позволяет разбивать общение с внешним контактом (например, клиентом) на несколько задач. Активировав ее, вы получите возможность работы следующим образом: 


  * По завершении обсуждения текущего вопроса, завершаете задачу, созданную по обращению контакта

  * При поступлении следующего письма в эту задачу (часто бывает, что клиент вместо написания нового письма находит старую переписку и пишет в ней), ПланФикс автоматически создаст новую задачу для общения с ним.

  * В новую задачу будет добавлена ссылка на предыдущее общение с этим контактом. Она помогает быстро перейти в него и ознакомиться с историей общения.
