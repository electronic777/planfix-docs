**Дополнительные операции** \- четвертый и последний блок [ правила обработки почты](Задачи/Правила_создания_задач_по_e-mail.md), который необходимо заполнить при [ его создании](Доступ и права/Создание_и_редактирование_правил.md). Дополнительные операции позволяют проводить нужный набор операций в процессе обработки письма. По внешнему виду и набору возможных операций они очень похожи на список операций [ автоматических сценариев для задач](Задачи/Автоматические_сценарии_для_задач.md): 


![Ev8aoL.jpg](images/Ev8aoL.jpg)


  

При этом, дополнительные операции отличаются возможностью оперировать [ инфоблоками](Коммуникации/Правило_для_почты__Извлечение_данных.md), извлеченными при обработке письма. Инфоблоки используются как для поиска нужных объектов (например, задачи или контакта), так и для заполнения полей задач, контактов и аналитик значениями.
