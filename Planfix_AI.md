## AI-кредиты


[AI кредиты](AI_кредиты.md "AI кредиты") — это единицы расчета для использования AI-инструментов в ПланФиксе. 


  * Для пробного использования AI-инструментов ежемесячно и бесплатно доступно ограниченное число кредитов.

  * Для постоянной работы докупите нужное количество кредитов на месяц в разделе «Подписка» по следующей тарификации:


     AI-инструмент | Стоимость   

---|---  

Переводчик | 4 кредита за 1000 символов ( текста для перевода )   

Помощник для комментариев | 5 кредитов за 1000 символов ( исходного текста комментария )   

Распознавание речи | 2 кредита за 1 минуту   

Анализ распознанного разговора | 2 кредита за 1000 символов (разговор + инструкция + результат анализа)   

Действие сценария Запрос к Planfix AI | 2 кредита за 1000 символов ( запрос + ответ )   

Действие сценария Квалифицировать запрос | 2 кредита за 1000 символов ( запрос + ответ )   

AI-агент Датамайнер  | 2 кредита за 1000 символов (дополнительных инструкций + входного текста + выходного текста)   

AI-агент Суммаризатор  | 0.5 кредитов за 1000 символов (дополнительных инструкций + входного текста + выходного текста)   

AI-агент Интервьюер  | 2 кредита за 1000 символов (дополнительных инструкций + входного текста + выходного текста)
