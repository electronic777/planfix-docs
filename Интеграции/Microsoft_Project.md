**Microsoft Project** — это программа управления [проектами](Проекты/Проекты.md), разработанная корпорацией Microsoft. Задачи из проекта Microsoft Project можно импортировать в ПланФикс. 


Импорт поддерживает следующий формат файлов: csv, xls, xlsx, mpp, mpx, mpd, mspdi, pod. 


## Импорт задач Microsoft Project в ПланФикс


  * Допустим в Microsoft Project у вас есть проект с задачами, которые необходимо перенести в ПланФикс:


![AfqKRp.png](images/AfqKRp.png)


  


  * Сохраните файл проекта с задачами, например, в формате .mpp:


![x1sEbX.png](images/x1sEbX.png)


  


  * Создайте [новый проект](Проекты/Как_создать_проект_.md) в ПланФиксе и нажмите кнопку **Импорт** :


![fbyoot.png](images/fbyoot.png)


  


  * Загрузите сохранённый файл Microsoft Project в ПланФикс:


![BydeuC.png](images/BydeuC.png)


  


  * Укажите шаблон для импортируемых задач, а также выберите для каждого столбца поле задачи, в которое будут импортированы данные:


![lSvcf1.png](images/lSvcf1.png)


  


  * После завершения импорта переключите отображение задач проекта на Диаграмму Ганта, чтобы увидеть даты начала и завершения задач, а также их связи между собой:


![ACRg03.png](images/ACRg03.png)


  


## Важно


  * Из Microsoft Project в ПланФикс импортируются только даты начала и завершения задач, а также связи между ними.

  * Длительность задач не импортируется. Это связано с некоторыми особенностями: 

    * В Microsoft Project бывают дробные длительности (например, 8,23 дня). В ПланФиксе такой формат времени не поддерживается.

    * Рабочее время в Microsoft Project и ПланФиксе должно полностью совпадать. Этого добиться практически невозможно, в том числе из-за наличия дробной длительности в Microsoft Project.


    Всё это приводит к некорректным вычислениям даты завершения задачи в ПланФиксе. Именно поэтому в ПланФикс импортируются только даты начала и завершения задач, а также связи между ними.
