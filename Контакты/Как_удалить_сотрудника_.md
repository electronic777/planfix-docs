**Важно:**


  * При увольнении мы рекомендуем не удалять сотрудника, а провести процедуру [ одним из доступных способов](Контакты/Как_уволить_сотрудника.md).

  * Перед удалением сотрудника [ передайте его активные задачи на другого сотрудника](Задачи/Передача_задач_другому_сотруднику.md).


  

Удалить сотрудника может [ администратор аккаунта](Доступ и права/Роли_в_ПланФиксе.md) либо [ сотрудник, управляющий оплатой аккаунта](Контакты/Как_дать_сотруднику_права_на_оплату_аккаунта.md). 


  

Для того, чтобы удалить сотрудника, необходимо: 


1\. Зайти в раздел [ Сотрудники](Контакты/Сотрудники_и_компания.md)


2\. Кликнуть на сотрудника, которого собираетесь удалить 


3\. В карточке сотрудника, в меню дополнительных операций выбрать пункт **Удалить** :
