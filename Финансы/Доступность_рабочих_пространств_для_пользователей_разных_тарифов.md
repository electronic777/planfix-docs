Количество [ рабочих пространств](Прочее/Рабочие_пространства.md) для пользователей различных тарифных планов ограничено. С действующими на текущий момент ограничениями можно ознакомиться на [странице тарифов](https://planfix.ru/prices/). Пользователи старой линейки тарифов могут узнать о действующих для них ограничениях по запросу в [Службу поддержки](https://planfix.ru/support/).
