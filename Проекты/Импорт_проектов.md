Есть 2 способа импортировать в ПланФикс [проекты](Проекты/Проекты.md): 


1\. Если при [ импорте задач](Задачи/Импорт_задач.md) одна из колонок в исходном файле содержит название проекта, будет автоматически создан проект с соответствующим названием и задача будет помещена в него. 


2\. Можно импортировать только список проектов с нужными данными, без задач. Этот способ предпочтительнее использовать в случаях, если есть необходимость на уровне проекта сразу заполнить какие-то данные, накопленные за пределами ПланФикса, например, установить контрагента проекта или его описание. Подробно этот способ описан ниже. 


## Импорт списка проектов


  * В разделе **Проекты** нажмите на кнопку **Импорт**.

  * Выберите исходный файл в одном из поддерживаемых форматов.

  * Укажите, в какие поля проекта необходимо импортировать данные из файла и проведите импорт:

## Полезная информация


  * Вы можете заранее добавить в [ шаблон проекта](Проекты/Шаблоны_проектов.md) дополнительные [ поля](Проекты/Пользовательские_поля_проекта.md), чтобы разместить в них накопленные данные любого типа. Обратите внимание на поля "Бюджет проекта", "Дата завершения проекта" и "Процент" из примера выше - они были добавлены именно таким образом.

  * Можно вначале импортировать проекты описанным выше способом, а затем произвести [импорт задач](Задачи/Импорт_задач.md). Если названия проектов в файле импорта задач будут совпадать с названиями проектов, ранее импортированных в ПланФикс, задачи автоматически попадут в нужный проект.
