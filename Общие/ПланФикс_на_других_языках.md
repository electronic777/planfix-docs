ПланФикс — полиязычная система: 


  * [Языки, на которые переведен интерфейс ПланФикса](../Интерфейс/Языки_интерфейса_ПланФикса.md).

  * Язык интерфейса устанавливается на уровне конкретного пользователя.

  * В одном аккаунте ПланФикса могут совместно работать пользователи, использующие разные языки интерфейса.


## Подключение и настройка интерфейса для разных категорий пользователей


Включение для пользователя того или иного языка интерфейса влияет только на сам интерфейс — контент (содержимое), внесенное в систему, автоматически не переводится. 


Для того, чтобы создать комфортные условия работы для ваших коллег, клиентов и партнеров, использующих другие языки, рекомендуем следовать данным рекомендациям: 


  * [Язык интерфейса для контакта без доступа в ПланФикс](../Контакты/Язык_интерфейса_для_контакта_без_доступа_в_ПланФикс.md)

  * [Язык интерфейса для контакта с доступом в ПланФикс](../Контакты/Язык_интерфейса_для_контакта_с_доступом_в_ПланФикс.md)

  * [Язык интерфейса для сотрудника](../Контакты/Язык_интерфейса_для_сотрудника.md)

  * [Язык аккаунта по умолчанию](../Прочее/Язык_аккаунта_по_умолчанию.md)
