Настройка отвечает за язык по умолчанию при создании нового сотрудника. Находится в Управление аккаунтом — Системные настройки — Язык и время. 


## Для чего используется


В большинстве случаев интерфейс ПланФикса отображается на [языке](Язык_интерфейса_для_сотрудника.md "Язык интерфейса для сотрудника"), указанном у сотрудника в его карточке. 


Настройка «Язык по умолчанию» определяет: 


  * Какой язык будет автоматически назначаться новым сотрудникам при их создании.

  * Какой язык интерфейса устанавливается по умолчанию управляющему аккаунтом.

  * Базовый язык интерфейса для системных сообщений и некоторых элементов, если у сотрудника не задан собственный язык.


## Важно


  * Язык аккаунта по умолчанию не меняет язык всем существующим сотрудникам — они продолжают работать с тем языком, который выбран у них в карточке.

  * Если у вас возникает вопрос, почему новые сотрудники создаются с определённым языком интерфейса — проверьте значение в настройке «Язык по умолчанию».
