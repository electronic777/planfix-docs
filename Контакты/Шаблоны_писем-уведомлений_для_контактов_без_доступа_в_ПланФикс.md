ПланФикс позволяет настраивать шаблоны писем-уведомлений, которые отправляются внешним контактам без доступа в систему. 


Существует 5 уведомлений: 


  * Основные: 

    * Новая задача, поставленная контакту

    * Новый комментарий, о котором уведомлен контакт


  * Дополнительные, которые включаются опционально: 

    * Создание задачи по входящему письму от внешнего контакта (автоответ)

    * Принятие исполнителем задачи, в которой внешний контакт является постановщиком

    * Изменение статуса задачи, в которой внешний контакт является постановщиком


**Обратите внимание:** Дополнительные уведомления по умолчанию отключены. 


  

Также есть возможность создавать дополнительные шаблоны уведомлений для отправки писем при помощи [ автоматических сценариев](Автоматизация/Автоматические_сценарии.md). 


  


## Создание шаблонов писем-уведомлений


Создание шаблонов происходит в разделе **Управление аккаунтом** \- **Работа с e-mail** : 


![LtXUJs.png](images/LtXUJs.png)


  

Во вкладке **Шаблоны уведомлений** можно добавить новый шаблон письма-уведомления. 


По умолчанию системные шаблоны не отображаются в этой вкладке, так как их нельзя редактировать: 


![v0BXYc.png](images/v0BXYc.png)


  

После нажатия на кнопку **Новый шаблон** открывается интерфейс создания шаблона: 


![J3U8I0.png](images/J3U8I0.png)


  

В новом шаблоне письма-уведомления необходимо задать нужные параметры: 


  * **Название** (можно задать вручную или использовать переменные из списка):


![MVGVXn.png](images/MVGVXn.png)


  


  * **Язык шаблона**. Под названием шаблона расположен переключатель языков. ПланФикс позволяет настраивать шаблоны для каждого языка, который используется в системе – ваши контакты будут получать уведомления по шаблонам, язык которых соответствует [ языку интерфейса](Контакты/Язык_интерфейса_для_контакта_без_доступа_в_ПланФикс.md), выбранному в карточке [ контакта](Контакты/Контакты.md):


![m6bDp8.png](images/m6bDp8.png)


  


  * **Тема письма** (По умолчанию, в нее сразу подставляется название задачи):


![hDoIJq.png](images/hDoIJq.png)


  


  * **Содержание письма** (для отображения в письме доступны различные переменные, которыми вы можете оперировать в ПланФиксе. Каждая такая представляет собой заключенное в двойные фигурные скобки название реквизита):


![tJdJxT.png](images/tJdJxT.png)


  

Полный список доступных переменных приведен тут же, по ссылке в справочном блоке: 


![LeWoP3.png](images/LeWoP3.png)


  

По клику на ссылку открывается список доступных реквизитов: 


![uKnM6d.png](images/uKnM6d.png)


  


  * **Дополнительная информация в письме** включает в себя пункты: 

    * Добавлять [ подпись](Автоматизация/Автоматическая_подпись_в_уведомлениях.md) к шаблону

    * Добавлять ссылку на [личный кабинет](Прочее/Личный_кабинет.md)

    * [Добавлять прикрепленные файлы](Документы/Добавлять_прикрепленные_файлы_если_шаблон_используется_для_уведомлений.md), если шаблон используется для уведомлений


При необходимости можно отключить ссылку на [Личный кабинет](Прочее/Личный_кабинет.md) клиента и лишние реквизиты, оставив только [ автоматическую подпись](Автоматизация/Автоматическая_подпись_в_уведомлениях.md) \- таким образом шаблон будет выглядеть как обычное письмо: 


![oIRSRu.png](images/oIRSRu.png)


  


## HTML-шаблоны писем


Вы можете использовать HTML-верстку в шаблоне, вставив ее в исходный текст шаблона:

## Подключение шаблонов писем-уведомлений


Для подключения шаблонов в разделе **Уведомление внешних контактов** нужно установить их вместо стандартных: 


![I1zGwP.png](images/I1zGwP.png)


  

При подключении дополнительных уведомлений не забудьте активировать эти уведомления: 


![zYjmlS.png](images/zYjmlS.png)


  

**Обратите внимание:** Настройка шаблонов писем-уведомлений для контактов без доступа в ПланФикс доступна только платным и премиум-аккаунтам.
