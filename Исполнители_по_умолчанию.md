**Исполнители по умолчанию** это сотрудники или контакты, которые будут добавлены в список исполнителей любой новой задачи проекта при ее создании. Список исполнителей по умолчанию можно задать при создании или редактировании [ карточки проекта](Карточка_проекта.md "Карточка проекта"). В процессе создания новой задачи в проекте, ПланФикс автоматически добавит в нее исполнителей из заданного списка. Автор задачи до ее сохранения сможет согласиться с этим списком исполнителей, либо отредактировать его. 


## Когда это удобно


Если в большинстве задач по проекту участвуют одни и те же исполнители, удобно будет задать их список на уровне проекта: это сэкономит время на создание задач в будущем - нужные люди будут подключены к задаче по умолчанию. 


## Важно


  * Наличие в проекте исполнителей по умолчанию не гарантирует того, что они будут видеть все задачи проекта, ведь при создании задачи ее автор может исключить их из списка исполнителей.
