Поле типа **Число** — это поле, которое служит для хранения числовой информации. Создать поле типа **Число** можно в качестве [ пользовательского поля](Пользовательские_поля.md "Пользовательские поля"), поля [ справочника](Справочники.md "Справочники") и поля [ аналитики](Аналитика.md "Аналитика"). 


## Параметры поля:


  * **Количество десятичных знаков** — позволяет указать, какое количество знаков выводить после разделителя;

  * **Запрещать дублирование значений** — не даст сохранить значение поля, если в нем уже есть такое значение для данного поля в другой задаче/контакте;

  * **Автонумерация** — при добавлении новой записи автоматически предложит значение, равное максимальному до этого момента +1;

  * **Разбивать на группы разрядов** — позволяет разбивать числа на группы разрядов.


## Дополнительные параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения;

  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.


## Дополнительные параметры для поля этого типа в аналитиках:


  * **Итоги аналитик** — позволяет суммировать данные по строкам аналитики.


## Важно


Числа в ПланФиксе хранятся в формате с [двойной точностью](https://ru.wikipedia.org/wiki/Число_двойной_точности).
