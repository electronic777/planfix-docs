В конфигурацию «CRM: Управление сделками 3.0» добавлено несколько специальных отчетов. Они позволяют увидеть сводную информацию по запросам за определенный период. Параметры отчетов можно менять. 


## Отчеты


  * **Объем продаж за текущий месяц** — показывает объем продаж по каждому сотруднику:


![IfkJ5x.png](images/IfkJ5x.png)


  


  * **Объем продаж за прошедший месяц** — идентичен предыдущему отчету, показывает объем продаж по каждому сотруднику за предыдущий месяц.


  * **Продажи по Т/У за прошедший месяц** — показывает объем продаж по каждому товару или услуге:


![VD5wg6.png](images/VD5wg6.png)


  


  * **Конверсия сделок по этапам за прошедший месяц** — показывает, сколько сделок было на каждом этапе и сколько из них в итоге завершилось успешно.


  * **Конверсия по источникам за прошедший месяц** — демонстрирует какой канал дает больше продаж и конверсию по каждому из них.


  * **История сделок по клиенту(ам)** — показывает историю сделок и общую сумму по каждому клиенту:


Если запустить отчет из карточки клиента — отобразятся только его сделки: 


![GiEAVH.png](images/GiEAVH.png)


  


  * **Отказы за последние 90 дней** — демонстрирует подробный анализ работы с отказами (сколько было отказов от сделки, сколько возращено на новый круг обработки и сколько в итоге потеряно):


![8WXJ4L.png](images/8WXJ4L.png)


## Обратите внимание


Помимо специальных отчетов вы можете [ сформировать любой отчет](Как_создать_новый_отчет_.md), который будет учитывать специфические показатели именно вашей компании.
