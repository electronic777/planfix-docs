В разделе [Задачи](Задачи.md) есть возможность просмотра списка задач в разных видах отображения: 


  * **Таблица** — позволяет увидеть все нужные данные задачи в виде столбцов таблицы. Состав этих столбцов можно [настроить](Фильтры_задач.md).

  * **Дерево** — отличается от табличного тем, что позволяет отображать иерархию вложенных задач ([подзадач](Подзадачи.md)) в виде дерева.

  * **Хроника** — позволяет видеть задачи в разрезе событий по ним. Подробнее о Хронике можно прочитать в [этой справочной статье](../Общие/Хроника.md).

  * **Гант** — позволяет отобразить отобранные в список задачи в виде [диаграммы Ганта](../Аналитика/Диаграмма_Ганта.md).

  * **Карта** — позволяет отобразить отобранные в список задачи на карте. Для этого в задачах должно быть поле типа [Местоположение](../Общие/Местоположение.md), которое содержит адресную информацию с привязкой к карте.


## Важно


  * Переключить вид просмотра можно над списком задач в дополнительном меню вверху справа.

  * В других разделах ПланФикса переключение вида отображения происходит аналогичным образом. Доступные виды отображения зависят от сути конкретного раздела.
