[360dialog](https://www.360dialog.com/) — официальный поставщик бизнес-решений WhatsApp Business API. Для того, чтобы связать ПланФикс и 360dialog для получения возможности получать и отправлять сообщения в WhatsApp, вам необходимо провести несколько несложных действий в обоих сервисах.

  * Перейдите в **Управление аккаунтом** — **Интеграции** — **Мессенджеры** :


![85NOVD.png](images/85NOVD.png)


  


  * Выберите WhatsApp:


![AhvIqq.png](images/AhvIqq.png)


  


  * Включите настройку интеграции:


![4IlLmP.png](images/4IlLmP.png)


  


  * Добавьте номер телефона и укажите имя для этого канала связи:

  * Произведите настройки номера и добавьте API key из 360dialog:
