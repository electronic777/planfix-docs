Список допустимых значений для параметра **sortType** (указывает в каком порядке будет произведена сортировка и по какому полю): 


Значение | Описание | Примечание   

---|---|---  

**NAME_ASC** | по названию группы (алфавит) |   

**NAME_DESC** | по названию группы (обратный порядок) |   

**USERCOUNT_ASC** | по количеству участников в группе (возрастание) |   

**USERCOUNT_DESC** | по количеству участников в группе (убывание) |
