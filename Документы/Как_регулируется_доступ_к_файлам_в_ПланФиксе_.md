Идеология ПланФикса подразумевает определение прав доступа к файлу на уровне задачи. Даже если Вы загрузили файл в проект (Карточка проекта — Файлы — Новый файл), другие сотрудники его не увидят *****. 


Доступ к файлу задается непосредственно в задаче, для работы над которой он нужен. Для этого при создании задачи или добавлении действия используется один из вариантов: 


  * Для файла, который был загружен в проект ранее: Файл — Прикрепить — Использовать файлы проекта.

  * Для нового файла: Файл — Прикрепить — Прикрепить новый файл.


В результате все участники этой задачи получат доступ к прикрепленному файлу, в том числе они будут видеть его во вкладке «Файлы» карточки проекта. 


## Пример


Компания ведет проект по разработке сайта для крупного клиента. В проекте уже хранятся файлы «Коммерческое предложение клиенту», «Контракт» и «Техническое задание». Для разработки дизайна сайта компания привлекает дизайнера-фрилансера. Создается новая задача «Дизайн сайта», исполнителем выбирается фрилансер, к задаче прикрепляется файл «Техническое задание» (Файл — Прикрепить — Использовать файлы проекта). 


В итоге, у фрилансера есть доступ к файлу «Техническое задание», который нужен ему для работы над сайтом, но не имеет доступа к файлам «Коммерческое предложение клиенту» и «Контракт». 


## Ручное управление доступом


Вы можете отрегулировать доступ к файлу вручную. Для этого зайдите в просмотр файла и нажмите иконку доступа. 


В появившемся окне есть возможность: 


  * дать доступ пользователям, не имеющим доступа к задачам, к которым прикреплен файл;

  * сгенерировать публичную ссылку для скачивания файла людьми, не являющимися пользователями вашего аккаунта ПланФикса.


## Важно


  * У Администраторов аккаунта и Аудиторов проекта есть доступ (просмотр и редактирование) ко всем файлам, загруженным в проект.

  * У Клиент-менеджеров проекта также есть доступ (просмотр) ко всем файлам, загруженным в проект.

  * Руководитель [по структуре компании](../Контакты/Структура_компании.md) может редактировать файлы сотрудников-подчинённых.
