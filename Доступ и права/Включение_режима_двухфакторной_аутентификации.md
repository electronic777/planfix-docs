Политика режима [ двухфакторной аутентификации](Доступ и права/Двухфакторная_аутентификация.md) для всего аккаунта задается управляющим аккаунтом в разделе: **Управление аккаунтом** — **Безопасность аккаунта** — **Двухфакторная аутентификация**. 


## Режимы работы


**Для сотрудников**


  * Никто из сотрудников не может использовать такой тип аутентификации;

  * Все сотрудники могут использовать использовать такой тип аутентификации _(этот вариант выбран по умолчанию)_ ;

  * Все сотрудники обязаны использовать такой тип аутентификации;

  * Только выбранные сотрудники или группы обязаны использовать такой тип аутентификации.


**Для клиентов**


  * Никто из клиентов не может использовать такой тип аутентификации _(этот вариант выбран по умолчанию)_ ;

  * Все клиенты обязаны использовать такой тип аутентификации;

  * Только выбранные клиенты и контакты обязаны использовать такой тип аутентификации.


  

При выборе одного из пунктов, обязывающих сотрудников или клиентов использовать двухфакторную аутентификацию, при следующем входе, ПланФикс попросит провести установку мобильного приложения Google Authenticator и регистрацию приложения в сервисе ПланФикс. 


## Важно


  * Управляющий аккаунтом может включать двухфакторную аутентификацию для всего аккаунта.

  * Администратор — только для сотрудника.
