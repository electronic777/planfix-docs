ПланФикс поддерживает возможность редактирования [документов](Документы/Документы.md) при помощи программного обеспечения, установленного на локальном устройстве (ПК). При этом используется следующая логика: 


  * При выборе опции **Редактировать** — **на компьютере** документ автоматически скачивается на ПК пользователя и открывается в программе, ассоциированной на данном устройстве с документами этого типа.

  * После завершения редактирования, документ автоматически загружается в ПланФикс в виде новой версии исходного документа.


Для обеспечения функционирования этой логики используется специальный программный агент **Planfix Docs Editor**. Программа существует в двух версиях: для операционных систем macOS и Windows. В зависимости от операционной системы при первой попытке редактирования документа на компьютере ПланФикс предложит автоматически установить нужную версию Planfix Docs Editor. 


Принцип работы Planfix Docs Editor в операционных системах macOS и Windows одинаков и ничем не отличается. Ниже продемонстрировано редактирование документа в операционной системе Windows. 


## Описание работы


  * В интерфейсе просмотра документа нажмите "Редактировать" — "На компьютере":

  * При первой попытке редактирования система предложит установить Planfix Docs Editor.

  * Установите программу и начните редактирование, либо повторно нажмите "Редактировать" — "На компьютере".

  * Документ автоматически будет скачан на ваш компьютер и открыт в приложении, с которым ассоциирован в настройках операционной системы.

  * После завершения редактирования, сохраните изменения и закройте программу, при помощи которой редактировался этот файл. Planfix Docs Editor автоматически загрузит его обратно в ПланФикс.

  * Отредактированный файл станет доступен в виде [новой версии](Документы/Ведение_версий_документа.md) первоначального документа:

## Ограничения


  * Planfix Docs Editor работает в операционной системе Windows, начиная с версии 7. Для работы необходим пакет .Net Framework 3.5.
