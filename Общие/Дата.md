Поля данного типа служат для хранения дат. При работе с полями данного типа используется стандартный интерфейс выбора дат ПланФикса. Создать поле типа **Дата** можно в качестве [пользовательского поля](../Поля и формы/Пользовательские_поля.md), поля [справочника](../Аналитика/Справочники.md) и поля [ аналитики](../Аналитика/Аналитика.md). 


## Параметры поля


  * **Запрещать дублирование значений** — не даст сохранить значение поля, если в нем уже есть такое значение для данного поля, например, в другой задаче.

  * **Автоматически заполнять текущей датой** — позволяет заполнять поле текущей датой, например, при создании задачи.


## Дополнительные параметры для поля этого типа в справочниках


  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.

  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения.


## Дополнительные параметры для поля этого типа в аналитиках


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.
