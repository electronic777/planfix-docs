Мобильное приложение позволяет работать с несколькими [аккаунтами](../Прочее/Аккаунт.md) ПланФикса, быстро переключаясь между ними. 


## Выход из аккаунта


Откройте меню **Еще** и нажмите на иконку **Выйти** : 


![wV6VVU.png](images/wV6VVU.png)


## Добавление нового аккаунта


  * Выйдите из основного аккаунта и нажмите кнопку **Другой аккаунт**.

  * Введите адрес нужного аккаунта ПланФикса.

  * Кликните **Далее**.

  * Введите логин и пароль.

  * Нажмите **Войти**.


После ввода данных приложение автоматически откроет аккаунт, который вы только что добавили. 


Обратите внимание, добавленные аккаунты будут отображаться в списке для быстрого переключения между ними, который отображается при выходе из аккаунта. 


## Смена аккаунта


Откройте меню **Еще** и нажмите на иконку **Пользователи** , чтобы перейти в другой аккаунт. 


![m4Udqm.png](images/m4Udqm.png)


  


  * Нажмите на нужный аккаунт, приложение автоматически его откроет.


## Удаление аккаунта


Для удаления добавленного ранее аккаунта из списка быстрого доступа: 


  * Выйдите из аккаунта.

  * На экране входа проведите пальцем влево на названии аккаунта, чтобы удалить его из списка быстрого доступа.
