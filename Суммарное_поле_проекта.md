Суммарное поле проекта — это инструмент для автоматического подсчёта и отображения агрегированных данных по задачам проекта. Доступно в пакетах «Бизнес» и «Корпорация». 


## Назначение


Суммарное поле используется для: 


  * Подсчёта суммы — например, по выбранному числовому полю в задачах проекта — бюджета, прибыли, часов.

  * Подсчёта количества — например, задач, соответствующих заданным условиям — количество выполненных, просроченных, заявок в работе.


## Преимущества


  * **Гибкость в настройке** — можно создать любое количество суммарных полей и привязать их к нужным параметрам задач. Например, настроить подсчёт прибыли, количества активных задач или рейтинга.

  * **Автоматизация** — после настройки поле обновляется автоматически при изменении данных в задачах. Это исключает ручной пересчёт и экономит время, особенно в крупных проектах.

  * **Удобство в аналитике и отчётности** — Суммарные поля позволяют быстро оценивать состояние проекта: видеть общие суммы, динамику и ключевые показатели без создания отдельных отчётов.


## Как добавить


  * Перейдите в меню Управление аккаунтом — Настраиваемые поля — Поля проектов.

  * Нажмите "Добавить поле" — "Суммарное поле".

  * Укажите, по какому полю задач нужно считать сумму или количество, и задайте условия отбора задач (если нужно).

  * Сохраните изменения.


После этого суммарное поле появится в карточке проекта и будет автоматически обновляться при изменении данных в задачах. 


## Примеры использования


  * Подсчёт общей суммы по полю «Прибыль» для всех задач проекта.

  * Подсчёт количества задач в статусе «Выполнено».

  * Подсчёт часов, затраченных исполнителями в рамках задач проекта.
