**Шаблон задачи** позволяет быстро создавать типовые [задачи](Задачи/Задачи.md) и другие [объекты](Прочее/Объекты.md) с заранее определенными параметрами: проектом, исполнителями, описанием и т.п. 


По умолчанию, в каждом [ аккаунте](Общие/Аккаунт.md) есть как минимум один шаблон под названием **Стандартный шаблон**. По этому шаблону создаются задачи, если не выбран другой шаблон. Этот шаблон можно отредактировать, но нельзя удалить. 


Вы можете создавать любое количество своих шаблонов, а также определять, какой из них будет использоваться по умолчанию.

## Дополнительно


  * Если в шаблон будут внесены изменения (например, добавлен исполнитель), то это никак не отразится на уже созданных по данному шаблону задачах. Но все задачи, создаваемые по шаблону начиная с этого момента, уже будут содержать установленного исполнителя.


  * В ПланФиксе есть возможность создавать шаблоны для деревьев задач. Создав дерево шаблонов в разделе "Шаблоны", вы получаете возможность быстро создавать аналогичное дерево задач, выбрав при создании задачи корневой (самый первый) шаблон этого дерева.


  * До появления [Объектов](Прочее/Объекты.md), шаблоны задач использовались для задания формы задачи. В шаблон можно было добавить нужные пользовательские поля и тем самым получить нужный объект: Сделку, Встречу, Заявку и так далее. На текущий момент этой функции у шаблонов задач нет. В старых аккаунтах возможность добавления полей в шаблоны задач сохраняется для обратной совместимости, но даже там мы рекомендуем использовать Объекты, как более функциональный способ настройки ПланФикса.
