[ Конфигурация](../Настройка/Конфигурации.md) **«Учет рабочего времени»** — это готовое решение ПланФикса, позволяющее вести учет рабочего времени сотрудников. 


Конфигурация позволяет: 


  * учитывать фактическое время работы сотрудников над задачами и проектами;

  * работать по тайм-трекингу;

  * контролировать виды выполняемых работ;

  * определять стоимость работ для расчетов заработной платы сотрудников;

  * анализировать в отчете трудозатраты по задачам и проектам за выбранный период.

## Если конфигурация не подошла


Если описанный подход к организации работы вам не подходит, есть несколько вариантов дальнейших действий: 


  * Установите другую конфигурацию из списка [готовых решений](https://planfix.ru/configurations/).

  * Настройте свой вариант работы, используя [ возможности ПланФикса](../Прочее/Описание_возможностей.md).

  * Обратитесь к [партнерам ПланФикса](https://planfix.ru/ourpartners/), которые помогут настроить систему наиболее подходящим для вас образом.
