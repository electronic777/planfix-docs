Поля данного типа служат для хранения суммарных итогов аналитик. Создать поле типа **Итоги аналитик** можно в качестве [ пользовательского поля](../Поля и формы/Пользовательские_поля.md). 


## Параметры поля


  * **Аналитика** \- указывает, на какую аналитику ссылается поле;

  * **Поле** \- указывает, из какого поля аналитики брать данные для суммирования;

  * **Разбивать на группы разрядов** — позволяет разбивать числа на группы разрядов.


## Дополнительная информация


  * Для удобства использования в формулах и расчетах, итоги аналитики по реквизиту типа [Период времени](../Время и учет/Период_времени.md) подсчитываются в минутах
