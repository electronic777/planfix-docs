Администратор аккаунта может настроить доступ к любому [ рабочему пространству](Рабочие_пространства.md "Рабочие пространства"). Управление правами доступа производится внутри рабочего пространства, в режиме его создания или редактирования: 


![gFpw8B.jpg](images/gFpw8B.jpg)


Клик в это поле вызывает стандартный для ПланФикса интерфейс задания прав доступа к объекту: 


![Q80P2W.jpg](images/Q80P2W.jpg)


  

Он позволяет настроить доступ к рабочему пространству как для сотрудников, так и для клиентов ([контактов с доступом в ПланФикс](Внешний_пользователь.md "Внешний пользователь")). 


  


## Быстрое назначение рабочего пространства по умолчанию


Ниже параметра **Доступ к рабочему пространству** расположены две ссылки для быстрого назначения рабочего пространства по умолчанию: 


![wI7uPv.jpg](images/wI7uPv.jpg)


Они позволяют в один клик установить текущее рабочее пространство для выбранной категории пользователей. При этом: 


  * Рабочее пространство будет прописано в соответствующем параметре карточки сотрудника или контакта с доступом в ПланФикс и может быть изменено там вручную или нажатием аналогичной ссылки в другом рабочем пространстве.

  * При установке рабочего пространства по умолчанию для всех контактов с доступом, оно также будет прописано во всех шаблонах контактов и компаний, чтобы созданные по ним впоследствии контакты автоматически получали это же рабочее пространство по умолчанию.


  


## Важно


  * Доступ, который вы предоставляете к рабочему пространству, определяет только то, какие пользователи увидят его в списке рабочих пространств. То, какие данные увидит конкретный пользователь в данном рабочем пространстве, определяется его [ правами доступа](Права_доступа_в_ПланФиксе.md "Права доступа в ПланФиксе").

  * Для [ контактов с доступом в ПланФикс](Внешний_пользователь.md "Внешний пользователь") доступ к базовому рабочему пространству можно отключить. Для сотрудников такая возможность доступна, если по умолчанию выбрано не базовое пространство.
