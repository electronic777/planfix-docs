Вы можете удалить неиспользуемый [ статус задачи](Какой_смысл_в_статусах_задач_.md) из конкретного набора статусов или полностью удалить его из системы. 


  


## Удаление статуса из набора статусов


Если статус не используется в определенном [ процессе](../Общие/Процессы.md), вы можете удалить его из набора статусов этого процесса, при этом сохранив этот статус в системе для дальнейшего использования. 


Удаление происходит в интерфейсе [ настройки набора статусов](Настройка_правил_перехода_между_статусами.md). Из набора можно удалить как [ пользовательские статусы](Пользовательские_статусы_задачи.md), добавленные вами в систему, так и [ стандартные статусы ПланФикса](Какой_смысл_в_статусах_задач_.md). 


  


## Удаление статуса из системы


Отредактировать или удалить пользовательский статус задачи можно во вкладке **Статусы** в меню **Управление аккаунтом**. Для редактирования просто кликните на нужном статусе: 


![gqeoD6.png](images/gqeoD6.png)


  

Откроется окно редактирования. В нем кроме опций редактирования, доступно также удаление статуса. 


![3GG96m.png](images/3GG96m.png)


  

**Важно** : 


  * При удалении таким образом произойдет **полное удаление** статуса из системы.

  * Описанным образом можно удалить только [ пользовательские статусы задач](Пользовательские_статусы_задачи.md), которые вы добавили в систему ранее.
