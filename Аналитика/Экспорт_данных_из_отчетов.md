Вы можете экспортировать данные, полученные в результате выполнения отчетов ПланФикса, в виде файла формата MS Excel произвольного вида. Это позволяет формировать по данным из ПланФикса различные финансовые документы, сводки и т.п. 


Для задания формы получаемого документа используются файлы MS Excel, в которых производится нужное форматирование и размещаются переменные из отчетов. Эти файлы называются **Шаблонами экспорта данных**. При экспорте данных выбирается нужный шаблон: 


![7DGobS.png](images/7DGobS.png)


  

ПланФикс создает новый файл по этому шаблону и размещает в ячейках, в которых в шаблоне расположены переменные, соответствующие этим переменным данные из отчета. Итоговый файл предлагается сразу скачать и использовать по дальнейшему назначению. 


## Переменные в названиях шаблонов


В названиях шаблонов для экспорта можно использовать переменные. При генерации документа в имя файла на место переменных подставятся нужные данные. Так можно автоматически задавать документам уникальные имена.

![ZmdyeI.png](images/ZmdyeI.png)

## Важно


  * При экспорте данных отчета применяется ограничение по времени: если экспорт отчета занимает более 10 минут, процесс автоматически прерывается. Для уменьшения времени экспорта используйте более простые шаблоны, с меньшим количеством реквизитов, либо экспортируйте данные за меньший период.


## Полезно


  * Вы также можете формировать документы произвольной формы на основании данных отдельных задач при помощи [ Шаблонов документов](Документы/Шаблоны_документов.md).

  * [Пример использования шаблонов экспорта данных из отчетов](https://planfix.com/ru/blog/shablony-eksporta-dannyx-iz-otchetov/) для формирования счета и приложения к счету по выполненным для клиента работам.

  * Для того, чтобы в шаблоне формата Excel ячейка автоматически подстраивала свою высоту в зависимости от длины текста, необходимо соблюсти следующие условия: 

    * строка должна иметь высоту по умолчанию для документа Excel;

    * ячейка не должна содержать объединение;

    * для ячейки надо поставить опцию "Перенос текста" в Excel.

  * Если отчет содержит большое количество строк, можно [ применить условное форматирование](Документы/Шаблоны__Выделение_строк_полосами_фона.md) для выделения их "зеброй":
