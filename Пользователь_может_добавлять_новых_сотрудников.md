Администратор аккаунта может [ разрешить пользователю](Дополнительные_возможности_пользователя.md "Дополнительные возможности пользователя") добавлять новых сотрудников, делать контактов сотрудниками и редактировать существующих сотрудников своих групп. Для этого: 


  * зайдите в карточку нужного пользователя в разделе **Сотрудники** ;

  * нажмите кнопку **Редактировать** ;

  * разверните панель **Возможности** ;

  * активируйте признак, который указан на картинке ниже:


![nTu3zb.png](images/nTu3zb.png)


  

После этого пользователь сможет добавлять новых сотрудников и редактировать уже существующих в тех [ группах](Рабочие_группы.md "Рабочие группы"), в которых состоит сам.
