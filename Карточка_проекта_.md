**Карточка проекта** открывается по клику на название [ проекта](Проекты.md "Проекты"). Она состоит из двух панелей: основной (слева) и дополнительных панелей с информацией (справа). 


На основной панели отображается список задач проекта, а в панели с дополнительной информацией могут отображаться разнообразные данные: 


  * [Детали проекта](Детали_проекта.md "Детали проекта")

  * [Документы проекта](Документы_проекта.md "Документы проекта")

  * [Лог проекта](Лог_проекта.md "Лог проекта")

  * [Контакты проекта](Контакты_проекта.md "Контакты проекта")

  * [Отчеты по проекту](Отчеты_по_проекту.md "Отчеты по проекту")

  * [Уведомления по проекту](Уведомления_по_проекту.md "Уведомления по проекту")
