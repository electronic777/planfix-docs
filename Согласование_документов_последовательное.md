Конфигурация **«Согласование документов: последовательное»** позволяет согласовать документ по заранее настроенной цепочке согласования. В процессе установки конфигурации вы задаете этапы, которые должен пройти документ для согласования, и ответственных сотрудников или группы сотрудников для каждого этапа. 


Для установки требуется тариф "Профессионал". 


Конфигурация может устанавливаться в аккаунте многократно, для каждого типа документов — это имеет смысл, если они отличаются цепочкой согласования.
