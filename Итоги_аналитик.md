Поля данного типа служат для хранения суммарных итогов аналитик. Создать поле типа **Итоги аналитик** можно в качестве [ пользовательского поля](Пользовательские_поля.md "Пользовательские поля"). 


## Параметры поля


  * **Аналитика** \- указывает, на какую аналитику ссылается поле;

  * **Поле** \- указывает, из какого поля аналитики брать данные для суммирования;

  * **Разбивать на группы разрядов** — позволяет разбивать числа на группы разрядов.


## Дополнительная информация


  * Для удобства использования в формулах и расчетах, итоги аналитики по реквизиту типа [Период времени](Период_времени.md "Период времени") подсчитываются в минутах
