Управляющий [аккаунтом](Общие/Аккаунт.md) может дать доступ к управлению [ оплатой за использование ПланФикса](Финансы/Как_оплатить_пакет_.md) любому сотруднику (или нескольким). Для этого: 


  * зайдите в карточку нужного пользователя в разделе **Сотрудники** ;

  * нажмите кнопку **Редактировать** ;

  * разверните панель **Возможности** ;

  * активируйте признак **Может управлять подпиской на сервис** :


![dHuS96.png](images/dHuS96.png)


  

После этого у данного пользователя в главном меню появится пункт **Подписка на сервис** и он сможет проводить оплату за использование ПланФикса без участия управляющего аккаунтом. 


## Права на удаление сотрудников


Сотрудник, имеющий право управлять подпиской аккаунта, может удалять любых сотрудников, кроме управляющего аккаунтом. Эта возможность дана ему для того, чтобы он мог таким образом регулировать основной параметр, по которому происходит оплата сервиса.
