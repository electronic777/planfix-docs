ПланФикс дает возможность автоматически разделить [ задачу](Задачи.md "Задачи") на набор [ подзадач](Подзадачи.md "Подзадачи") или пунктов [ чек-листа](Чек-лист.md "Чек-лист") для каждого исполнителя. Эта возможность активируется при создании задачи во вкладке: Дополнительные параметры — [Подключение исполнителей](Подключение_исполнителей_к_задаче.md "Подключение исполнителей к задаче") — У задачи одновременно несколько исполнителей. 


Как это работает: 


  * при создании задачи вы выбираете перечень (или группу) исполнителей;

  * выбираете один из этих вариантов подключения исполнителей;

  * в зависимости от выбранного варианта, ПланФикс создает в этой задаче по одному пункту чеклиста или по подзадаче для каждого исполнителя (участника группы).


Если выбран вариант с подзадачами, то каждый исполнитель видит только свою подзадачу и работает в ней с постановщиком, не мешая другим. При этом все подзадачи собраны в одну надзадачу, что позволяет постановщику в одном месте видеть и контролировать процесс работы всех исполнителей. 


В варианте с пунктами чеклиста все исполнители могут общаться в общей задаче, а наличие персонального пункта чеклиста для каждого исполнителя позволяет им в индивидуальном режиме отработать задачу и перевести ее в статус “Выполненная”, не влияя на коллег. 


## Обратите внимание


ПланФикс позволяет автоматически разделить задачу на подзадачи. Для каждой подзадачи можно назначить своего исполнителя. Такой способ подключения исполнителей происходит по принципу «Отдельная задача каждому исполнителю». Если вы выбираете такое подключение исполнителей к подзадачам, обратите внимание на следующую особенность. 


Например, существует задача A с подзадачами B и C. Если к такому набору подзадач подключать исполнителей по описанному выше принципу, подзадачи B и C удалятся, а вместо них для каждого исполнителя будут созданы новые подзадачи: А1, В1, С1; А2, В2, С2 (и так далее по числу исполнителей). 


Обращая внимание на такую особенность, **мы не рекомендуем вызывать действие автоматического сценария или кнопки «Создать отдельную подзадачу для каждого исполнителя» на уже существующих задачах с подзадачами**.
