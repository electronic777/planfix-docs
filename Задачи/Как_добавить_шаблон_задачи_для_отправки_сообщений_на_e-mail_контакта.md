Задачу-общение с клиентом по почте можно создать в ручную. Делается это через карточку клиента по нажатию на кнопку **Написать** — **На почту** : 


![BCPj3x.png](images/BCPj3x.png)


  

Открывается форма создания задачи. Тут важно указать правильный шаблон, чтобы в задаче были нужные поля — это шаблон **Диалог с клиентами** : 


![qeFm66.png](images/qeFm66.png)


  

Этот шаблон не является настройкой по умолчанию. Его необходимо указать в настройках работы с e-mail: 


  * Переключитесь в **Базовое рабочее пространство** :


![1o0rOU.png](images/1o0rOU.png)


  


  * Пройдите по пути: **Управление аккаунтом** — **Работа с e-mail** — **Уведомление внешних контактов** :


![KB3mHq.png](images/KB3mHq.png)


  


  * Укажите шаблон **Диалог с клиентами** в шаблоне задачи для отправки сообщений на e-mail контакта:


![iIXmfN.png](images/iIXmfN.png)
