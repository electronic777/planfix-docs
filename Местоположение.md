Поля данного типа служат для хранения адресной информации. Создать поле типа **Местоположение** можно в качестве [пользовательского поля](Пользовательские_поля.md "Пользовательские поля"), поля [справочника](Справочники.md "Справочники") и поля [аналитики](Аналитика.md "Аналитика"). 


Адрес, введенный в такое поле, будет преобразован в геоданные, содержащие координаты введенного адреса на карте. 


  * В поле этого типа одновременно хранится текстовый адрес и геоданные, которые могут отображаться на карте.

  * Также в поле можно вводить координаты нужной точки в формате, который используется поисковыми системами.


## Параметры для поля этого типа в справочниках


  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.


## Параметры для поля этого типа в аналитиках


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.


## Важно


  * Для корректной работы с данным типом поля необходимо включить [интеграцию с картографическими сервисами](Интеграция_с_картами.md "Интеграция с картами").
