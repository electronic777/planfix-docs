Суммарное количество непрочитанных событий отображается в табе браузера, в котором открыт ПланФикс: 


![Iep0WM.png](images/Iep0WM.png)


  

В [ Хронике](../Общие/Хроника.md) отображается счетчик непрочитанных событий по каждой из задач: 


![FohKJ7.png](images/FohKJ7.png)


  

Счетчик помогает следить за выполнением основного принципа Хроники: 


**Хорошая Хроника — это пустая Хроника.**


  


Если вдруг по каким-то причинам ваш счетчик событий имеет такой вид: 


![EEiA1p.png](images/EEiA1p.png)


  

И вам нужно сбросить этот счетчик, отметив прочтенными все события в вашей хронике, воспользуйтесь специальным пунктом для отметки прочитанными всех событий в меню вида отображения Хроники: 


![h3Xg5R.png](images/h3Xg5R.png)


  


## Обратите внимание


  * Счетчик учитывает события по задачам и по контактам. Поэтому, для корректного отображения показаний счетчика, можно сделать настройки Хроники отдельно для задач и для контактов. Еще вариант — использовать настройку Хроники по задачам и контактам:


![eNUga5.png](images/eNUga5.png)
