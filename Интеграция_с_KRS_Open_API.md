Интеграция с KRS Open API (Krajowy Rejestr Sądowy) позволяет получать официальные сведения о юридических лицах, зарегистрированных в Польше, по их KRS-номеру. 


Обычно используется для: 


  * Поиска и проверки польских компаний по KRS.

  * Автоматического заполнения карточек компании.

  * Снижения количества ручного ввода данных.

  * Использования в бизнес-процессах, например, при регистрации нового контрагента.


## Что делает


  * Получает зарегистрированное название компании.

  * Извлекает юридический адрес.

  * Определяет тип организации.

  * Подставляет основные регистрационные данные.


## Как использовать


  * Укажите номер KRS в карточке контакта.

  * ПланФикс отправит запрос в KRS Open API.

  * Затем автоматически подставит полученные данные в указанные поля.


## Настройка интеграций


  * Перейдите в Управление аккаунтом — Интеграции — Автозаполнение данных.

  * Включите интеграцию с KRS Open API.

  * Добавьте название интеграции.

  * Укажите поля, где в ПланФиксе содержатся значения KRS или NIP.

  * Сопоставьте поля для автозаполнения.

  * Сохраните настройки.
