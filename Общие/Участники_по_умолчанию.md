**Участники по умолчанию** это сотрудники или контакты, которые будут добавлены в список участников любой новой задачи проекта при ее создании. Список участников по умолчанию можно задать при создании или редактировании [ карточки проекта](../Проекты/Карточка_проекта.md). В процессе создания новой задачи в проекте, ПланФикс автоматически добавит в нее участников из заданного списка. Автор задачи до ее сохранения сможет согласиться с этим списком участников, либо отредактировать его. 


## Когда это удобно


Если необходимо обеспечить доступ к просмотру задач проекта одним и тем же людям, удобно будет задать их список на уровне проекта: это сэкономит время на создание задач в будущем - нужные люди будут подключены к задаче по умолчанию. 


## Важно


**Наличие в проекте участников по умолчанию не гарантирует того, что они будут видеть все задачи проекта** , ведь при создании задачи ее автор может исключить их из списка участников.
