Срок, за который доступны [ данные об отправке и прочтении писем получателями](Отслеживание_отправки_и_прочтения_письма.md "Отслеживание отправки и прочтения письма"), зависит от используемого вами [тарифного плана](https://planfix.ru/prices/): 


  * Для бесплатных аккаунтов статус отправки и прочтения писем получателями недоступен

  * Для аккаунтов, работающих по тарифу **Профессионал** данные доступны за последние **30 дней**

  * Для аккаунтов, работающих по тарифу **Бизнес** данные доступны за последние **180 дней**

  * Для аккаунтов, работающих по тарифу **Корпорация** данные доступны **без ограничения срока хранения**


  


## Важно


  * Переход на другой пакет изменяет доступность данных. В том числе, вы можете **перейти на один из старших пакетов для того, чтобы получить доступ к данным за более длительный период**.

  * Данные в системе хранятся начиная с 15.04.2019 г. До этой даты данные недоступны, вне зависимости от пакета.

  * Приведенные выше данные о периоде доступности информации являются сравочными, точные данные на текущий момент см. в [таблице тарифов](https://planfix.ru/prices/).
