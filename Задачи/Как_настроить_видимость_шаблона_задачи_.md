Если у сотрудника есть [ доступ к шаблону задачи](Задачи/Доступ_к_шаблонам_задач.md), но ему не нужно в ходе текущей работы создавать задачи по этому шаблону, он может отключить его отображение в списках шаблонов. 


По умолчанию новые доступные шаблоны сразу становятся видимыми в списке шаблонов при создании задачи. Об этом сообщает включенная лампочка возле названия шаблона. 


При необходимости шаблон можно скрыть из списка, отключив лампочку. 


Настройка видимости шаблона — это индивидуальная настройка для каждой учетной записи, каждый пользователь настраивает ее только для себя.
