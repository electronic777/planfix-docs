В названии и описании [ задач](Задачи.md) можно использовать различные **переменные**. Переменная представляет собой название поля задачи или другого объекта, заключенное в двойные фигурные скобки. Пример переменной, в которой хранится текущая дата: 

    

    

    {{Системные.Сегодня}}


  

Переменные добавляются в [объекты](../Общие/Объекты.md) и [шаблоны задач](Шаблоны_задач.md) во время их создания или редактирования. В момент создания задачи вместо переменной подставляется ее текущее значение. 


## Добавление переменных


В интерфейсе создания объекта есть специальная пиктограмма, нажмите на неё, появится список доступных переменных: 


![jaoKiY.png](images/jaoKiY.png)


  

Среди переменных доступны: 


  * около 30 системных переменных, позволяющих отобразить в разных форматах время, дату, день недели, месяц и год;

  * все переменные, относящиеся к задаче, включая переменные для каждого [ пользовательского поля](Пользовательские_поля_задачи.md);

  * переменные для каждого реквизита любого [ справочника](../Аналитика/Справочники.md), который используется в [ полях задачи](../Аналитика/Запись_справочника.md);

  * все переменные, относящиеся к [ проекту](../Проекты/Проекты.md), в котором создается задача – включая [ пользовательские поля проекта](../Проекты/Пользовательские_поля_проекта.md);

  * все переменные, относящиеся к [контрагенту](../Общие/Контрагент.md) задачи, в том числе любые данные из [ карточки клиента](../Контакты/Карточка_контакта.md);

  * все переменные, относящиеся к надзадаче текущей задачи, включая данные из карточки ее контрагента или используемых в ней справочников.


В момент создания задачи по шаблону, вместо переменной подставляется ее текущее значение.

## Обратите внимание


  * Замена переменной на конкретное значение происходит однократно в момент создания задачи по объекту.

  * Если задача создается по объекту в статусе [Черновик](../Общие/Черновик.md), замена переменных на значения в ней произойдет в момент перевода задачи в один из рабочих [ статусов](Какой_смысл_в_статусах_задач_.md).

  * Дату в переменной можно [форматировать](../Поля и формы/Форматирование_даты.md).

  * Переменные в задачах доступны только в платных и премиум-аккаунтах.
