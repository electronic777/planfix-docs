Раздел [Документы](Документы/Документы.md) состоит из двух взаимосвязанных панелей: 


  * Панели фильтров (слева)

  * Панели навигации по документам (справа)


## Панель фильтров


На панели фильтров доступны следующие разделы: 


  * Использование облачного хранилища

  * Документы по проектам

  * Документы без проекта

  * Документы из справочников

  * Документы контактов

  * Избранное

  * [Шаблоны документов](Документы/Шаблоны_документов.md)

  * [Шаблоны экспорта отчетов](Аналитика/Шаблоны_экспорта_отчетов.md)

  * Корзина


## Панель навигации по документам


На панели навигации по документам отображается информация выбранного на панели фильтров раздела. Как правило, это перечень папок и файлов с дополнительной информацией о них. 


Отметив галочкой нужные документы, сверху можно увидеть кнопки меню массовых операций над ними.
