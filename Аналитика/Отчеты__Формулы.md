Отчеты ПланФикса позволяют использовать в столбцах простые формулы для расчета значений. Формулы оперируют данными, выводящимися в столбцах, находящихся **левее столбца с формулой** (т.е. до столбца с формулой), при помощи стандартных операций. Данные из столбцов адресуются по букве, обозначающей столбец: 


![SkTY1f.png](images/SkTY1f.png)


  

Формулы рассчитываются по порядку слева направо в один проход: строка заполняется значениями из базы для всех столбцов, которые не формулы, потом начинают рассчитываться формулы. Соответственно, формула нормально оперирует значениями из базы до неё, но не может оперировать значениями других формул. 


В формулах можно использовать [стандартные операции и функции](Прочее/Функции.md) ПланФикса.
