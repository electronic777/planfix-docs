В карточке каждого [ контакта](Контакты.md "Контакты") указан процесс, в котором вы можете задать необходимые для работы с этим контактом автоматические сценарии: 


![0Qy7SC.png](images/0Qy7SC.png)


  

Для создания автоматического сценария контакта выберете соответствующий раздел: Управление аккаунтом — Процессы контактов — Название нужного процесса — Автоматические сценарии. 


## Создание нового сценария


Изначально список сценариев процесса пуст: 


  * Нажмите на кнопку "Новый сценарий".

  * Откроется окно добавления нового сценария.


## Настройка сценария


  * Задаем говорящее название сценарию.

  * Выбираем событие, при котором будет срабатывать сценарий для контактов.

  * Когда срабатывает выбранное в сценарии событие, контакт проверяется на соответствие заданным в сценарии условиям и запускается выполнение автоматических действий.

  * Перечень операций, которые можно производить над контактами:


![G52S4p.png](images/G52S4p.png)


  

При выборе операции разворачивается дополнительное меню настроек, соответствующее типу операции. 


## Обратите внимание


  * Вы можете отлаживать автоматические сценарии в [ режиме тестирования](Режим_тестирования_сценариев_задач_и_контактов.md "Режим тестирования сценариев задач и контактов").

  * Автоматические сценарии массовым образом могут изменять не более 20 контактов. Такое ограничение введено для предотвращения отправки спама из системы.
