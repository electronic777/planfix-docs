[1msg](https://1msg.io/) — это инструмент управления и автоматизации [WhatsApp](WhatsApp.md) для вашего бизнеса. Для того, чтобы настроить интеграцию ПланФикса и 1msg, проведите несколько несложных операций. 


Перед началом настройки зарегистрируйте аккаунт в [1msg](https://1msg.io/), добавьте новый проект и активируйте канал связи. 


  


## Процедура настройки


  * В ПланФиксе перейдите в **Управление аккаунтом** — **Интеграции** — **Мессенджеры** — **WhatsApp** — **Chat API**.

  * В интеграции активируйте переключатель «Использовать API 1msg.io» (1) и перейдите на сайт 1msg (2):


![vWeV4w.png](images/vWeV4w.png)


  


  * В профиле 1msg скопируйте userToken:


![pt5x2p.png](images/pt5x2p.png)


  


  * Вернитесь в ПланФикс и добавьте скопированный userToken в поле «API user token» (1) и нажмите ссылку «Обновить», чтоб получить список добавленных аккаунтов WhatsApp из личного кабинета 1msg.

  * Кликнув на добавленный аккаунт (2), настройте его, указав шаблоны для создания новых задач и контактов, а также подключив дополнительных исполнителей при необходимости:


![6LAulx.png](images/6LAulx.png)


  


  * Не забудьте сохранить настройки интеграции, нажав кнопку **Сохранить**.


  

С этого момента все новые сообщения из WhatsApp будут попадать в ПланФикс, а для каждого чата будет создаваться отдельная задача. 


  


## Интеграция c WhatsApp поддерживает


  * Получение и отправку файлов.

  * Получение локаций и контактов.
