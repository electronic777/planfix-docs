Для перевода задачи в разные статусы можно использовать кнопки. Для каждого статуса используется отдельная кнопка. Настроить перевод в статусы при помощи кнопок можно в разделе **Управление аккаунтом** — **Объект** — **Нужный Объект** : 


![f5gIgC.png](images/f5gIgC.png)


  


  * Переходим в **Набор статусов**.

  * Кликаем на нужный статус и активируем опции:


![mYckm0.png](images/mYckm0.png)


  


  * Так отображается «Кнопка для смены статуса» в задаче:


![PbO0DE.png](images/PbO0DE.png)


  


  * Опция «Отображать в меню» добавляет кнопку в дополнительное меню:


![m8rIF1.png](images/m8rIF1.png)
