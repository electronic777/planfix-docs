**Вычисляемые поля** — это вид [ пользовательских полей](Пользовательские_поля.md "Пользовательские поля") ПланФикса. В настройках вычисляемого поля указывается формула, по которой происходит вычисление значения поля. 


Значение пересчитывается только при изменении полей, от которых оно зависит. Принудительно пересчитать значение вычисляемого поля можно автоматическим сценарием или массовым действием. 


Вычисляемые поля поддерживаются в следующих объектах системы: 


  * [Вычисляемые поля задач](Вычисляемые_поля_задач.md "Вычисляемые поля задач")

  * [Вычисляемые поля в аналитиках](Вычисляемые_поля_в_аналитиках.md "Вычисляемые поля в аналитиках")

  * [Вычисляемые поля контактов](Вычисляемые_поля_контактов.md "Вычисляемые поля контактов")
