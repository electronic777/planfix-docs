## Создать задачу


Интерфейс создания новой задачи откроется по клику на пункт **Создать задачу** , при этом: 


  * пользователь, который инициировал создание задачи, становится ее автором;

  * проект указывается таким же, как проект, в котором находится текущая задача;

  * задача, в которой находится этот комментарий, становится надзадачей для создаваемой задачи;

  * текст комментария подставляется в описание задачи;

  * исполнителями устанавливаются те, кто был уведомлен об этом комментарии.


Значения добавляются в форму создания задачи по-умолчанию исключительно для удобства, чтобы избежать лишней ручной работы по заполнению полей. Любой из параметров задачи можно изменить перед сохранением. 


## Создать задачу+


Альтернативный способ создать задачу из комментария — выбрать пункт **Создать задачу+**. Так в создаваемую задачу дополнительно будут добавлены ещё и данные по самому комментарию: 


  * дата и время комментария;

  * автор;

  * ссылка на комментарий.
