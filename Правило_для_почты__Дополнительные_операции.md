**Дополнительные операции** \- четвертый и последний блок [ правила обработки почты](Правила_создания_задач_по_e-mail.md "Правила создания задач по e-mail"), который необходимо заполнить при [ его создании](Создание_и_редактирование_правил.md "Создание и редактирование правил"). Дополнительные операции позволяют проводить нужный набор операций в процессе обработки письма. По внешнему виду и набору возможных операций они очень похожи на список операций [ автоматических сценариев для задач](Автоматические_сценарии_для_задач.md "Автоматические сценарии для задач"): 


![Ev8aoL.jpg](images/Ev8aoL.jpg)


  

При этом, дополнительные операции отличаются возможностью оперировать [ инфоблоками](Правило_для_почты__Извлечение_данных.md "Правило для почты: Извлечение данных"), извлеченными при обработке письма. Инфоблоки используются как для поиска нужных объектов (например, задачи или контакта), так и для заполнения полей задач, контактов и аналитик значениями.
