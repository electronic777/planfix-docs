ПланФикс поддерживает интеграцию с онлайн коннектором сервисов и приложений [**Make**](https://www.make.com/en/register?utm_source=planfix-app&utm_medium=partner&utm_campaign=planfix-app-partner-program). Подключить его можно по следующему пути: Управление аккаунтом — Интеграции — Сервисы обработки данных: 


![YNx7UG.png](images/YNx7UG.png)


  


## Возможности интеграции


Интеграция срабатывает при наступлении события, выбранного в автоматическом сценарии ПланФикса. Она поддерживает следующие действия: 


  * Создать контакт

  * Создать комментарий

  * Создать задачу

  * Удалить контакт

  * Удалить комментарий

  * Получить контакт

  * Получить комментарий

  * Получить задачу

  * Сделать вызов API

  * Обновить контакт

  * Обновить комментарий

  * Обновить задачу

  * Поиск комментариев

  * Поиск контактов

  * Поиск задач


С какими сервисами ПланФикс можно интегрировать через Make смотрите на этой [странице](https://www.make.com/en/integrations/planfix?utm_source=planfix-app&utm_medium=partner&utm_campaign=planfix-app-partner-program). 


## Подключение


Для подключения ПланФикса к Make используйте XML API: 


  * Перейдите в Управление аккаунтом — Доступ к API — XML API, чтобы получить ключ.

  * Затем по кнопке ниже создайте токен авторизации.


Ключ и токен используйте в MAke для подключения.
