## Ввод данных об услугах


Для того, чтобы распечатать счет или акт выполненных работ, необходимо внести данные для него в [ задачу](Задачи/Задачи.md). Это может быть любая существующая задача, в которой идет работа с клиентом, или задача, специально созданная для выставления счета. 


Шаблоны документов, входящие в состав конфигурации, используют [ контрагента задачи](Прочее/Контрагент.md) — он подставляется в качестве Покупателя (Заказчика). Убедитесь, что в задаче указан контрагент: 


![rrN6Dn.png](images/rrN6Dn.png)


  

Табличная часть счета и акта берется из входящей в состав конфигурации [ аналитики](Аналитика/Аналитика.md) **Услуга** , прикрепленной к задаче. Соответственно, вам необходимо добавить к задаче столько записей этой аналитики, сколько строк должно быть в итоговом документе. 


  * Вы можете добавлять аналитику как в тело самой задачи при ее создании:


![yr6prV.png](images/yr6prV.png)


  


  * Так и отдельным комментарием к существующей задаче:


![IVZkuI.png](images/IVZkuI.png)


  


  * Выберите из списка аналитику **Услуга** :


![uwyqo3.png](images/uwyqo3.png)


  


  * Добавьте нужное количество записей аналитики, по одной записи для каждой услуги:


![AQq6VF.png](images/AQq6VF.png)


  


  * Сохраните введенные данные.


  


## Создание документа


Для создания документа (счета или акта): 


  * Убедитесь, что необходимые данные (контрагент и аналитики) добавлены в задачу.

  * Добавьте к задаче новый файл (пиктограмма с изображением скрепки), выберите опцию создания документа по шаблону и укажите нужный шаблон:


![4JUp2Z.png](images/4JUp2Z.png)


  


  * При необходимости откорректируйте номер и дату документа:


![qtArVv.png](images/qtArVv.png)


  


  * Сохраните комментарий. К задаче будет автоматически прикреплен документ, созданный по выбранному шаблону и наполненный данными из задачи:


![GCf666.png](images/GCf666.png)


  

Его можно просмотреть, скачать, распечатать, отправить клиенту и т.п. 


  

**Примечания:**


  * При создании задачи или комментария можно одновременно добавить аналитику и создать документ:


![En694f.png](images/En694f.png)


  


  * Вы можете сразу отправить документ клиенту или подрядчику, которому он предназначен. Для этого: 

    * если документ создается при создании задачи: 

      * укажите клиента в роли исполнителя этой задачи (у клиента должен быть указан email);

      * дайте задаче название, которое станет темой письма клиенту;

      * в описании задачи изложите сопроводительный текст, адресованный клиенту;

      * добавьте аналитику и документ, создаваемый по нужному шаблону.


    В итоге клиент получит письмо с приложенным к нему документом. Его ответ на это письмо попадет в задачу в виде нового комментария и вы сможете продолжить с ним переписку прямо из ПланФикса.


  *     * если документ создается при написании комментария: 

      * отметьте клиента в блоке "Уведомить"

      * в тексте комментария напишите обращение к клиенту, которое станет текстом письма;

      * прикрепите к комментарию аналитику и документ, создаваемый по нужному шаблону:


![eIhrvf.png](images/eIhrvf.png)


  


    В итоге клиент получит письмо с темой, соответствующей названию задачи, в которой вы добавили комментарий, с приложенным к нему документом. Его ответ на это письмо попадет в задачу в виде нового комментария и вы сможете продолжить с ним переписку прямо из ПланФикса.


  * При генерации файла есть возможность выбрать формат PDF - этот вариант удобно использовать при отправке клиенту документов с подписью и печатью.

  * Если вы добавляете документ к задаче, то в его табличную часть войдут все аналитики, внесенные в эту задачу, даже если они были внесены в разное время в разных действиях. Если вам нужно создать документ с данными аналитики только из одного добавленного ранее комментария, можно сделать это через меню дополнительных операций с действием:


![MCqYef.png](images/MCqYef.png)


  


  * Одни и те же данные, внесенные в задачу, могут использоваться для генерации разных документов. Например, и счет, и акт выполненных работ могут использовать одну и ту же табличную часть (данные из аналитики) с перечнем оказанных услуг. Таким образом вы можете по одной и той же задаче сгененировать счет и акт для клиента, ничего не добавляя в нее дополнительно.


  


## Нумерация документов


При добавлении документа конкретного типа (например, счета) в первый раз, система автоматически предложит для него номер 1. Вы можете изменить этот номер на другой: 


![ZHcV33.png](images/ZHcV33.png)


  

При добавлении следующего счета ПланФикс автоматически прибавит к этому номеру единицу. Вы также сможете поменять его, если это будет необходимо.
