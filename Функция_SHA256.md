Функция SHA256() — принимает сообщения произвольной длины и вычисляет уникальный хэш-код фиксированной длины. Используется в работе с API различных сервисов для их интеграции в ПланФикс. 


## Формат


**SHA256(** текст**)**


## Пример


SHA256("Доброе утро") 


Результат: 607decee1e0350c692d8eafaaa0570642436b5abb761dddda489528b7b2a9aa3 


## Важно


  * На текущий момент функция не работает для формул, вычисляемых только для строк итогов или первых группировок в отчетах.
