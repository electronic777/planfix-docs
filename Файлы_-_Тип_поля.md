Поля данного типа служат для хранения ссылок на файлы, хранящиеся в ПланФиксе. Создать поле типа **Файлы** можно в качестве [ пользовательского поля](Пользовательские_поля.md "Пользовательские поля") и поля [ справочника](Справочники.md "Справочники"). 


## Параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** \- не даст сохранить запись справочника, пока это поле не содержит значения


## Важно


У поля типа Файл нет настроек доступа к полю. Файлы, прикрепленные в задаче любым способом видят все, кто имеют доступ к этой задаче.
