Поля данного типа служат для хранения произвольного количества ссылок на задачи. Создать поле типа **Набор задач** можно в качестве [ пользовательского поля](Пользовательские_поля.md "Пользовательские поля"), поля [ справочника](Справочники.md "Справочники") и поля [ аналитики](Аналитика.md "Аналитика"). 


## Дополнительные параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения.


## Дополнительные параметры для поля этого типа в аналитиках:


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.


## Полезно


  * При помощи [ фильтров в пользовательских полях](Фильтры_в_пользовательских_полях.md "Фильтры в пользовательских полях") вы можете выводить в списке для выбора в поле только задачи, соответствующие заданным условиям.


## Важно


  * В поле типа «Набор задач» может содержаться максимум 250 значений.
