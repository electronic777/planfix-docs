Основной принцип вывода данных из аналитик в отчетах: **Одна строка отчета может содержать только одну запись аналитики**. 


При этом нередко существует необходимость выводить в одном отчете данные из нескольких аналитик. Возьмем для примера простой отчет План/Факт, в котором для каждого дня выведем: 


  * запланированное время работы;

  * фактическое время работы;

  * разницу между ними.


Результат выполнения отчета выглядит так (фрагмент): 


![9QcTmR.jpg](images/9QcTmR.jpg)


В развернутом виде видно, что итоговые цифры собираются из записей аналитик - каждая строка содержит одну запись аналитики **Планируемое время работы** или **Фактическое время работы**. По понятным причинам, каждая из этих аналитик заполняет либо графу **План** , либо графу **Факт**. 


Вот как выглядят настройки этих столбцов отчета: 


![8kNRKh.jpg](images/8kNRKh.jpg)


  

Обратите внимание на выделенные красным блоки. Как ПланФикс интерпретирует эти настройки: 


  * В столбце **Планируемое время работы** выводятся данные соответствующей аналитики, а для строк с аналитикой **Фактическое время работы** в нем ничего не выводится.

  * В столбце **Время работы** также выводятся данные соответствующей аналитики, а для строк с аналитикой **Планируемое время работы** в нем ничего не выводится.


А теперь посмотрите на выделенный зеленым блок настройки столбца **Дата**. В нем данные выводятся и для аналитики **Планируемое время работы** , и для аналитики **Фактическое время работы**. И в том, и в другом случае это реквизиты аналитик, которые называются "Дата" и указывают на дату, к которой относится введенная аналитика. Такой подход позволяет сгруппировать данные из разных аналитик и получить свернутые данные: в одной строке мы можем увидеть суммарные данные по планируемому и фактически затраченному времени работы. 


Описанный пример иллюстрирует правило, которое вам необходимо взять на вооружение при создании отчетов, содержащих несколько аналитик: **В одну строку отчета данные из нескольких аналитик могут быть собраны только в случае, если вы их осознанно сгруппируете.**


Столбец **Дата** в описанном случае как раз является примером такой осознанной группировки: 


  * выбраны совпадающие по сути реквизиты в каждой из аналитик;

  * они выведены в одном столбце;

  * по этому столбцу задана группировка.
