Создание [ пользовательских полей](Поля и формы/Пользовательские_поля.md) происходит в разделе **Управление аккаунтом** — **Настраиваемые поля** — **Поля задач** — по кнопке **Новое поле** : 


  * При создании поля можно сразу выбрать, в каком [наборе](Поля и формы/Наборы_пользовательских_полей.md) оно будет создано.

  * После создания поля, его можно добавить в нужные [Объекты](Прочее/Объекты.md), где оно будет отображаться следующим образом:


![D4RiqD.png](images/D4RiqD.png)


  

Подробнее об использовании полей в Объектах можно прочитать в этой статье: [Карточка объекта](Прочее/Карточка_объекта.md). 


## Важно


Вы можете задать [ права доступа](Доступ и права/Доступ_к_пользовательским_полям.md) к пользовательскому полю, чтобы ограничить его видимость для пользователей вашего аккаунта. 


## Альтернативный способ создания пользовательского поля


Создать пользовательское поле можно непосредственно в процессе создания или редактирования [объекта](Прочее/Объекты.md).
