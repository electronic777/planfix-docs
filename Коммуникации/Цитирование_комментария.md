Цитирование применяется для удобства ведения переписки. Cуществует несколько способов процитировать [ комментарий](Действия__(комментарии_.md) "Действия \(комментарии\)") пользователя: 


  * Иконка **Ответить** расположена в комментарии внизу слева. Нажав на неё, вы можете быстро ответить, процитировав весь текст комментария:


![TfFqhJ.png](images/TfFqhJ.png)


  


  * Для того, чтобы сделать цитату только из части комментария, выделите мышкой необходимую часть текста и нажмите на ссылку **Цитировать** в появившемся окне:


![yEtyji.png](images/yEtyji.png)


  


  * При наведении курсора мыши на комментарий появляется иконка **Дополнительных действий**. Нажмите на неё, чтобы вызвать меню:


![SSHBbD.png](images/SSHBbD.png) _Изображение может отличаться в зависимости от выбранного[вида отображения](../Задачи/Лента_комментариев_задачи.md) комментариев._


  

Выберите пункт **Цитировать+** для создания нового комментария. В поле для ввода вашего ответа появится весь текст цитируемого комментария с дополнительной информацией о нем: 


![VgcFpr.png](images/VgcFpr.png)
