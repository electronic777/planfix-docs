Поля данного типа служат для хранения информации о времени начала и окончания какого-то конкретного временного периода в течение дня. Создать поле типа **Период времени** можно в качестве [ пользовательского поля](Поля и формы/Пользовательские_поля.md), поля [ справочника](Аналитика/Справочники.md) и поля [ аналитики](Аналитика/Аналитика.md). 


## Параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения;

  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.


## Параметры для поля этого типа в аналитиках:


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку;

  * **Итоги аналитик** — позволяет суммировать временные затраты по строкам аналитики.
