[ Конфигурация](Конфигурации.md "Конфигурации") **Управление курсами** — это готовое решение для управления оффлайновыми курсами, кружками и секциями. Его можно изменить под специфику вашей компании. 


В стартовом виде конфигурация позволяет: 


  * принимать обращения потенциальных клиентов по разным каналам связи (электронная почта, социальные сети, мессенджеры,заявки с сайта и прочее);

  * фиксировать интерес пользователей к определенному курсу или мастер-классу;

  * вести клиентов по всем стадиям цепочки продаж;

  * управлять календарным расписанием курсов, бронировать места проведения занятий, видеть занятость преподавателей;

  * вести базу клиентов с историей покупок (какой курс уже покупался и когда).

## Первые шаги


  * [ Добавьте преподавателей](Добавление_сотрудников.md "Добавление сотрудников") для управления их расписанием

  * Добавьте пробный курс, как это показано в видео

  * [ Добавьте каналы общения с клиентами](Как_настроить_прием_заявок_из_разных_каналов.md "Как настроить прием заявок из разных каналов")

  * Попробуйте общаться с клиентами из ПланФикса, как это показано в видео

  * Проведите пробный заказ от запроса до успешного окончания обучения

  * Если общением с клиентами будут заниматься другие сотрудники, [ добавьте их](Настройка_шаблона__Диалог_с_клиентом_.md "Настройка шаблона "Диалог с клиентом"")

## Если конфигурация не подошла


Если описанный подход к организации работы вам не подходит, есть несколько вариантов дальнейших действий: 


  * Установите другую конфигурацию из списка [готовых решений](https://planfix.ru/configurations/).

  * Настройте свой вариант работы, используя [ возможности ПланФикса](Описание_возможностей.md "Описание возможностей")

  * Обратитесь к [партнерам ПланФикса](https://planfix.ru/ourpartners/), которые помогут настроить систему наиболее подходящим для вас образом.
