**Общие поля** — это один из видов настраиваемых пользовательских полей, наряду с [ полями задач](../Задачи/Пользовательские_поля_задачи.md), [ контактов](../Контакты/Как_добавить_новое_поле_в_карточку_контакта.md) и [ проектов](../Проекты/Пользовательские_поля_проекта.md). Они служат для хранения информации, которую планируется использовать на уровне всего аккаунта, например в [ шаблонах документов](../Документы/Шаблоны_документов.md).
