**Шаблон документа** — это файл в формате Microsoft Excel или Microsoft Word, в котором вы можете использовать поля из задач и аналитик ПланФикса. Для этого в шаблон документа вставляются специальные переменные. Как создать собственный шаблон документа и пользоваться переменными, смотрите в справке про [шаблоны документов](Шаблоны_документов.md). 


  * Шаблоны документов расположены в базовом рабочем пространстве:


![rTJrGQ.png](images/rTJrGQ.png)


  


  * По пути: раздел **Документы** — **Шаблоны документов**.


  

В конфигурации "CRM: Управление сделками 3.0" используется 3 шаблона документов: 


  * **Договор** — шаблон документа в формате Microsoft Word, по которому формируются договоры.

  * **КП** — шаблон документа в формате Microsoft Word, по которому формируются коммерческие предложения.

  * **Счет** — шаблон документа в формате Microsoft Excel, по которому формируется счёт на оплату.
