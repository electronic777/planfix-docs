Комбинированная диаграмма — это тип диаграммы в [отчетах](Отчеты__Графики_в_отчетах.md) ПланФикса, которая объединяет два типа графиков: 


  * Столбчатая гистограмма — отображает данные в виде столбцов разной высоты, показывая количественные значения.

  * Линейный график — соединяет точки данными линией, что помогает отследить изменения во времени.


Этот тип диаграммы позволяет сравнивать разные показатели, даже если они измеряются в разных единицах. 


Например, так могут отображаться выполненные задачи сотрудников за месяц и время их выполнения по аналитикам: 


![4SDUgC.png](images/4SDUgC.png)


  

График строится на основе данных из отчета и состоит из четырех основных элементов: 


  * Горизонтальная ось (X) — показывает категории (например, сотрудников, проекты или дни недели).

  * Вертикальная ось (Y) — отображает числовые значения (например, количество выполненных задач, затраченное время или объем продаж).

  * Столбцы — представляют один из показателей (на примере это объем выполненной работы).

  * Линия — отображает другой показатель (на примере это сумма затраченного времени, указанного в аналитике).


Для того, чтобы отображались нужные данные — используется группировка и сортировка. 


## Как настроить


  * В «Вид отчета» добавьте столбцы с данными, по которым будет собрана комбинированная диаграмма.

  * Далее в разделе «Графики и диаграммы» добавьте новый график типа «Комбинированная диаграмма»:


![e5Lwdv.png](images/e5Lwdv.png)


  


  * В «Данные для отображения» укажите значения: 

    * Столбцы, по которым будет строиться график: «Выполненные задачи» и «Затраченное время».

    * Группировка: По столбцу «Исполнитель(общим списком)» и по возрастанию.

    * Сортировка: По столбцу «Выполненные задачи» по месяцам и по возрастанию.


    ![OCmxfz.png](images/OCmxfz.png)


  


  * Во «Внешнем виде» можно настроить: 

    * Высота графика — определяет высоту отображаемого графика в пикселях.


    ![8kpGoQ.png](images/8kpGoQ.png)


  


  * Сохраните график и запустите отчет для просмотра.
