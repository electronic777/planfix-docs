Воронка продаж, которую обычно организуют в ПланФиксе, поделена на этапы. Каждый этап — определенный статус задачи. Таким образом, меняя статус, задачу можно перемещать из одного этапа воронки продаж в другой. 


Статусы настраиваются в [ процессах задач](../Общие/Процессы.md). По какому процессу живут задачи, можно узнать во вкладке **Детали** : 


![zsLaGy.png](images/zsLaGy.png)


  


## Добавление нового этапа воронки продаж


Для добавления нового этапа воронки: 


  * создайте новый статус задачи в нужном процессе — подробности в этой [ справке](../Задачи/Добавление_нового_статуса.md).


  * добавьте очередной этап, который будет соответствовать новому статусу. Для этого перейдите в планировщик, где настроена воронка продаж, и создайте новый список:


![CnGKcv.png](images/CnGKcv.png)


  


  * назовите новый этап воронки продаж;

  * задайте подходящий цвет;

  * активируйте опцию "Отбирать по условиям";

  * укажите, с каким статусом отбирать в список задачи;

  * сохраните настройки:


![BN23NR.png](images/BN23NR.png)


  


  * укажите нужную вам последовательность этапов:


![m79KzZ.gif](images/m79KzZ.gif)
