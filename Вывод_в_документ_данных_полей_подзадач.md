Вы можете вывести в формируемый по [ шаблону](Шаблоны_документов.md "Шаблоны документов") документ данные полей подзадач. Для этого необходимо использовать переменную **{{Задача.Подзадачи}}**. 


Для корректной работы, эту переменную необходимо вставить в [ табличную часть шаблона](Таблицы_в_шаблонах_документов.md "Таблицы в шаблонах документов"), аналогично подходу, который используется при вставке данных из прикрепленных к задаче аналитик. 


При формировании документа по шаблону, такая строка будет [ автоматически размножена](Автоматическое_размножение_строк_документа.md "Автоматическое размножение строк документа") на количество данных в подзадачах. 


## Полезно


  * Если в шаблоне формата MS Word вставить переменную **{{Задача.Подзадачи}}** вне таблицы, в итоговый документ будет включена информация только из первой подзадачи.
