Вы можете вывести в формируемый по [ шаблону](Шаблоны_документов.md "Шаблоны документов") документ все [файлы](Файлы.md "Файлы"), прикрепленные к пользовательскому полю типа Файл. Для этого необходимо использовать переменную **{{Задача.Поле.Файлы}}**. 


Для корректной работы, эту переменную необходимо вставить в [ табличную часть шаблона](Таблицы_в_шаблонах_документов.md "Таблицы в шаблонах документов"), аналогично подходу, который используется при вставке данных из прикрепленных к задаче аналитик. 


При формировании документа по шаблону, такая строка будет [ автоматически размножена](Автоматическое_размножение_строк_документа.md "Автоматическое размножение строк документа") на количество файлов в пользовательском поле типа Файл. 


У переменной **{{Задача.Поле.Файлы}}** доступны следующие поля: 


  * порядковый номер записи

  * идентификатор

  * имя

  * имя без расширения

  * изображение - это сам файл физически, то есть в это место в документе будет вставляться картинка


## Полезно


  * Если в шаблоне формата MS Word вставить переменную **{{Задача.Поле.Файлы}}** вне таблицы, в итоговый документ будет включена информация только по одному из файлов задачи.

  * Дополнительное форматирование изображений описано в справочной статье [Использование изображений в шаблонах документов](Использование_изображений_в_шаблонах_документов.md "Использование изображений в шаблонах документов").
