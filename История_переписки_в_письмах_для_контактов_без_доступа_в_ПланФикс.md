В ПланФиксе есть возможность добавлять историю переписки в [ письма-уведомления](Уведомления_для_контактов_без_доступа.md "Уведомления для контактов без доступа") для [ контактов без доступа](Как_работать_с_клиентом_по_e-mail_.md "Как работать с клиентом по e-mail?"). 


  * Включить опцию:


    **Управление аккаунтом** — **Работа с e-mail** — **Уведомления внешних контактов**


  * И активировать добавление последних сообщений из истории переписки в текст письма, отправляемого ПланФиксом (количество сообщений, которые могут быть включены в письмо, зависит от используемого [пакета](https://planfix.ru/prices/)).


После этого каждое письмо, которое получит любой из контактов, будет дополнительно содержать выбранное количество последних сообщений из переписки. 


## Важно


  * Если контакты, с которыми вы переписываетесь, используют современные почтовые программы, которые умеют собирать письма в цепочки, использование этой опции нецелесообразно. Цепочки удобнее в работе и лучше отображают историю переписки, а добавляемая при помощи этой опции дополнительная история переписки будет только мешать восприятию информации..

  * Если в задаче идет параллельное общение с разными контактами (или сотрудниками), каждый из контактов получит в качестве истории только те сообщения, которые писал он сам или которые были ему адресованы.

  * Опция доступна только пользователям платных и премиум-аккаунтов.
