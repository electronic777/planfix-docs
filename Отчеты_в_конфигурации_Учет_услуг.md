В состав [конфигурации](Учет_услуг_выставление_счетов_закрытие_актов_отчеты.md "Учет услуг выставление счетов закрытие актов отчеты") входит группа отчётов «Учет услуг». Они оперируют введенными в ходе работы с конфигурацией аналитиками **Услуга** и позволяют выводить данные за нужный период времени в необходимых разрезах. 


Для того, чтобы воспользоваться отчетами, перейдите в главном меню в раздел **Отчеты** и выберите нужный отчет из группы **Учет услуг** : 


![CiOfxf.png](images/CiOfxf.png)


  

Эти отчеты позволяют ответить на стандартные для любого вида деятельности вопросы: 


  * **Учет услуг: Итоги по клиентам за период** — сколько денег принес нам каждый из клиентов.

  * **Учет услуг: Итоги по ответственным за период** — на какую сумму были внесены услуги каждым из менеджеров.

  * **Учет услуг: Распределение по видам услуг** — какие услуги приносят больше всего валового дохода.


Помимо использования по назначению, вы можете сделать копию любого из отчётов и построить нужный вам отчет, используя многочисленные возможности [ конструктора отчетов ПланФикса](Отчеты.md "Отчеты").
