**Синхронизация** — это режим, в котором изменение условий в одном списке текущего [ планировщика](../Общие/Планировщик.md) приводит к аналогичным изменениям условий в других списках. Например, можно изменить день в календарном списке — и другие календарные списки текущего планировщика автоматически переключатся на этот же день. 


  * **Режим синхронизации** включается в выпадающем меню текущего планировщика:

## Как происходит синхронизация


Синхронизируются только те изменения, которые пользователь добавляет в фильтр списка задач. Редактируя один список, изменения автоматически добавятся и в другие списки, которые участвуют в синхронизации в рамках одного планировщика. 


**В параметрах отбора задач синхронизируется** : 


  * добавление нового условия;

  * изменение типа условия;

  * изменение оператора условия;

  * изменение значения условия;

  * удаление условия.


**В виде карточки задач и настройке меню карточки синхронизируется** : 


  * добавление полей и операций;

  * изменение полей и операций;

  * удаление полей и операций.


**Важно:** карточка задачи целиком не синхронизируется. Настроить её внешний вид в одном списке и через синхронизацию перенести настройки в другой список нельзя. 


## Практика использования синхронизации


  * **При управлении задачами команды**

    * Например, вы руководитель проекта и используете Планировщик с разными списками: первый — для общего обзора всех задач, второй — для задач конкретной команды, а третий — календарный для ежедневного планирования. При изменении условия фильтра (например, выбор конкретной даты), вам не придется делать изменения в каждом списке отдельно. Благодаря синхронизации изменения в одном списке автоматически применятся и в других списках. В итоге вы увидите задачи за выбранную дату сразу во всех списках, что сэкономит ваше время, если бы вы настраивали каждый список по отдельности.


  * **При ведение нескольких проектов одновременно**

    * Например, вы управляете несколькими проектами и создаете списки задач по каждому из них. Если в какой-то момент вам нужно сосредоточиться только на одном проекте, достаточно добавить условие «Проект» в фильтр одного списка, и благодаря синхронизации остальные списки покажут задачи этого же проекта автоматически.


  * **При личном планировании**

    * Если у вас есть несколько списков для личных задач, например, один — по рабочим задачам, другой — по личным делам, третий — календарь встреч и событий, вам не придется каждый раз вручную устанавливать одинаковые условия. При добавлении фильтра по дате в календарном списке другие синхронизированные списки автоматически покажут задачи именно на эту дату.


## Что важно запомнить


  * Синхронизация распространяется только на условия фильтрации и настройки карточки задач.

  * Не синхронизируются сортировка, группировка, логика фильтра и цветовые настройки.


Используя синхронизацию, вы экономите время и снижаете вероятность ошибки при работе с несколькими списками задач.
