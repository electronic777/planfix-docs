Находясь в любом разделе ПланФикса вы в один клик можете попасть в [Избранное](../Общие/Избранное.md), используя **Панель быстрого доступа к Избранному** , на которой отображены отмеченные вами объекты: 


![wgeOre.png](images/wgeOre.png)


  


## Поиск в избранном


Клик по звездочке в окошке глобального поиска по аккаунту вызывает Панель быстрого доступа к Избранному, на которой отображаются важные для вас объекты, которые отмечены значком Избранного: 


![8C1Wgu.png](images/8C1Wgu.png)


  

По Избранному можно искать: при включенной Панели быстрого доступа к Избранному строка глобального поиска работает только по нему: 


![2Rlrs6.png](images/2Rlrs6.png)


  


## Работа с избранным


Список Избранного в Панели быстрого доступа к Избранному можно сортировать и отображать в удобном для вас виде. Для каждого типа объекта настройки могут быть разными. К примеру, [Задачи](../Задачи/Задачи.md) могут быть отсортированы по дате, а [файлы](../Документы/Файлы.md) — по релевантности: 


![oGOSsT.png](images/oGOSsT.png)


  

Также есть возможность фильтровать объекты, отмеченные различными значками Избранного, внутри каждой группы: 


![MpJJRS.png](images/MpJJRS.png)
