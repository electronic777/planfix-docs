Это тип списка в [Планировщике](../Общие/Планировщик.md), который выводит данные из [аналитик](../Аналитика/Аналитика.md) в разбивке по часам. 


Особенности Расписания почасового: 


  * Почасовая сетка растянута горизонтально, в отличие от [почасового календаря](Календарь_почасовой.md).

  * Данные можно вывести в несколько строк:


![9aRwE8.png](images/9aRwE8.png)


  

Для настройки расписания переходим во вкладку "Общая информация". Выбираем и фильтруем объекты, данные из которых нужно отобразить. 


![VV3XHN.png](images/VV3XHN.png)


А затем в вкладке "Данные для отображения" указываем, что именно нужно выводить из выбранных элементов в календарь. Например, можно отобразить расписание сотрудников, на основе данных из аналитики. 


![EqVeVj.png](images/EqVeVj.png)


  

Помимо [Аналитик](../Аналитика/Аналитика.md) в параметры можно добавить поля задач, в том числе [пользовательские](../Задачи/Пользовательские_поля_задачи.md): 


![O8YVZS.png](images/O8YVZS.png)


## Быстрые фильтры


В расписании почасовом можно быстро отфильтровать данные по дате (1) или объекту (2), по которому ведется расписание. Для этого достаточно кликнуть на соответствующий элемент: 


![25eq3X.png](images/25eq3X.png)


  

В итоге можно получить полное расписание на день: 


![qEtlmw.png](images/qEtlmw.png)


  

Или расписание по конкретному объекту за все дни выбранного временного периода: 


![ZD5RUx.png](images/ZD5RUx.png)


## Множественная фильтрация


В **Расписании почасовом** можно включить фильтр сразу по нескольким объектам. Для этого нажмите на поле объекта, а затем включите фильтр: 


![4Y9bh1.png](images/4Y9bh1.png)


  

Чтобы отключить любой из фильтров, нажмите «Сбросить фильтр»: 


![ZwQMzj.png](images/ZwQMzj.png)


## Важно


  * Список Планировщика "Расписание почасовое" работает только в платных аккаунтах ПланФикса.
