В разделе **Документы** хранятся все документы, которые попали в ПланФикс [ одним из доступных способов](Файлы.md). 


Для удобства каталогизации документы, находящиеся в проектах, группируются по папкам [ проектов](../Проекты/Проекты.md), [ справочников](../Аналитика/Справочники.md) или [ контактов](../Контакты/Контакты.md), к которым они прикреплены. 


Остальные документы располагаются в папке **Документы без проекта**.
