**Раздел Планировщик** мобильного приложения позволяет работать с данными в планировщиках, созданных в разделе [Планировщик](../Общие/Планировщик.md) веб-версии. На текущий момент в мобильной версии поддерживаются списки Планировщика следующих типов: 


  * Хроника

  * Список

  * Карточки

  * Таблица

  * Календарь почасовое

  * Календарь по дням


Ведется работа над добавлением поддержки в приложении остальных [ типов списков Планировщика](../Общие/Виды_списков_планировщика.md). 


## Переключение между планировщиками и списками


При загрузке открывается [планировщик по умолчанию](../Общие/Планировщик_по_умолчанию.md), установленный в веб-версии. Для того, чтобы открыть другой планировщик — нажмите на название текущего планировщика и выберите другой из перечня: 


![xBla1X.png](images/xBla1X.png)


  * Нажмите на иконку **Глаз** , чтобы изменить настройки отображения списка планировщиков.

  * Используйте **Поиск** по планировщикам, если их много.


## Работа с задачами в планировщике


  * Для того, чтобы открыть существующую задачу, нажмите на ее карточку в списке.

  * Внутри задачи можно писать комментарии, изменять значение полей, прикреплять аналитику, файлы и напоминания.

  * Нажмите на **Стрелку** в верхнем левом углу карточки задачи, чтобы вернуться в планировщик.

  * Новая задача, созданная в списке планировщика, будет соответствовать его параметрам.

  * Для того, чтобы создать задачу, нажмите на Плюс и выберите способ постановки: 

    * Голосовой ввод

    * Сокращенная карточка

    * Полная карточка


**Важно**


  * Если в планировщике присутствует группировка, задачу можно создать прямо в ней. Задача, созданная таким способом, будет соответствовать группировке.
