**Страница пользователя (сотрудника)** содержит данные и настройки, относящиеся к конкретному пользователю ПланФикса. Есть два основных способа попасть на страницу пользователя: 


  * Список пользователей, находящихся онлайн, доступен по клику на иконку "Сейчас в онлайне" в верхнем правом углу. На первом месте в списке всегда находится текущий пользователь, просматривающий этот список. Если вам нужно перейти в собственную карточку пользователя, это самый быстрый способ.


  * В разделе [Сотрудники](Сотрудники_и_компания.md "Сотрудники и компания") виден общий список пользователей. Клик на любого пользователя в списке позволяет попасть на его страницу (карточку):


Для обычного пользователя доступна только часть данных других пользователей аккаунта. [ Администраторы аккаунта](Роли_в_ПланФиксе.md "Роли в ПланФиксе"), а также сотрудники, которым дан [ специальный доступ к добавлению и редактированию сотрудников](Пользователь_может_добавлять_новых_сотрудников.md "Пользователь может добавлять новых сотрудников"), могут не только просматривать всю информацию о пользователе, но и редактировать ее. 


## Структура страницы пользователя


На странице пользователя доступны следующие вкладки: 


  * Общая информация

  * [ Настройки](Настройки_в_карточке_пользователя.md "Настройки в карточке пользователя")

  * [Рабочий календарь](Рабочий_календарь.md "Рабочий календарь")

  * [ Уведомления](Подписки_и_уведомления.md "Подписки и уведомления")

  * [ Задачи по почте](Правила_создания_задач_по_e-mail.md "Правила создания задач по e-mail") \- можно задать правила обработки для писем, приходящих на [ внутренний адрес сотрудника](Как_поставить_задачу_по_e-mail.md "Как поставить задачу по e-mail")

  * [Проекты](Проекты.md "Проекты") \- будет выведен список проектов, в которых принимает участие данный сотрудник

  * [Задачи](Задачи.md "Задачи") \- будет выведен список задач, в которых данный сотрудник является исполнителем

  * [Комментарии](Комментарии.md "Комментарии") \- будет выведен список комментариев и действий сотрудника

  * [Файлы](Файлы.md "Файлы") \- будет выведен список файлов, загруженных или созданных данным сотрудником

  * [Отчеты](Отчеты.md "Отчеты") \- будет выведен список отчетов, созданных данным сотрудником

  * [ Внешние адреса](Настройка_внешних_почтовых_адресов_сотрудников.md "Настройка внешних почтовых адресов сотрудников") \- позволяет настроить для пользователя один или несколько адресов электронной почты, с которых будут отправляться сообщения, написанные им в ПланФиксе для внешних адресатов (контактов)

  * [ Безопасность](Включение_двухфакторной_аутентификации_для_сотрудников.md "Включение двухфакторной аутентификации для сотрудников") \- настройка двухфакторной аутентификации

  * [ Управление сессиями](Пароль_приложения.md "Пароль приложения") \- отображает список сессий пользователя на различных устройствах и позволяет удалить скомпрометированную сессию

  * [Связь аккаунтов](Связь_аккаунтов.md "Связь аккаунтов") \- позволяет пользователю связать текущий аккаунт с другим используемым им аккаунтом

  * Лог - отображает лог изменений страницы пользователя


## Редактирование данных пользователя


Нажав на кнопку **Редактировать** на странице пользователя, вы увидите окно, в котором отображаются и могут быть отредактированы все данные пользователя. 


Некоторые данные пользователя (например, [Должность](Должность.md "Должность") или [часовой пояс](Как_изменить_часовой_пояс_.md "Как изменить часовой пояс?")) могут быть изменены только в режиме редактирования. 


Помимо редактирования данных в существующих полях, в этом окне вы можете [ добавить дополнительные поля в карточку сотрудника](Пользовательские_поля_сотрудника.md "Пользовательские поля сотрудника").
