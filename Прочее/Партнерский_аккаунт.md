[ Сертифицированный партнер](Партнерская_программа.md) может зарегистрировать специальный аккаунт для экспериментов с настройками и т.п. служебных надобностей. Параметры аккаунта: 


  * Ограничения по функционалу на уровне тарифного плана **Корпорация**

  * Возможность подключить до **5 сотрудников**


  


## Регистрация партнерского аккаунта


Для регистрации партнерского аккаунта необходимо: 


  * При регистрации указать партнерский e-mail

  * Выбрать из предлагаемых системой вариантов Партнерский аккаунт


![K77o65.png](images/K77o65.png)


  


## Ограничения использования партнерского аккаунта


  * Партнер может зарегистрировать только один аккаунт такого типа

  * Аккаунт не может использоваться для текущей работы компании партнера
