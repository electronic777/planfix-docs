**Автор проекта** не имеет особых полномочий по его управлению, за исключением прав на редактирование или удаление проекта. 


  * Автор проекта не видит всех задач проекта

  * Автор проекта не может завершить любую задачу проекта


Автор проекта может добавить себя в аудиторы. Таким образом он получит все права на управление проектом, которые есть у [ аудитора проекта](Проекты/Аудитор_проекта.md).
