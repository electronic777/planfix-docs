ПланФикс позволяет настраивать свои интерфейсы для разных пользователей системы. Настроенный для определенной группы или конкретного пользователя интерфейс называется **Рабочим пространством**. Рабочие пространства позволяют существенно упростить использование ПланФикса, так как пользователь будет видеть только тот объем функционала системы, который необходим ему в работе. 


Пользователю ПланФикса может быть доступно несколько рабочих пространств. [ Переключение между ними](Переключение_между_рабочими_пространствами.md "Переключение между рабочими пространствами") позволяет быстро перейти от одной операционной обстановки к другой. Это особенно актуально для руководителей и тех сотрудников, которые участвуют в работе разных подразделений компании. 


![3mDH3D.jpg](images/3mDH3D.jpg)


  

В рамках рабочего пространства можно настраивать: 


  * Главное меню ПланФикса (в том числе добавлять в него выпадающие меню)

  * Дополнительные пункты меню

  * Создание объектов

  * Раздел, в который попадает пользователь по умолчанию

  * Цветовую схему рабочего пространства
