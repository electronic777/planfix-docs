График — это тип диаграммы в [отчетах](Отчеты__Графики_в_отчетах.md) ПланФикса. Данные отображаются как точки, соединенные линиями, показывая изменения во времени или связи между величинами. Полезен для анализа изменений, таких как выполнение задач, рост выручки или прогресс проектов. 


Например, так могут отображаться просроченные и выполненные задачи сотрудников за месяц: 


![AZLEU6.png](images/AZLEU6.png)


  

График строится на основе данных из отчета и состоит из двух осей: 


  * Горизонтальная ось (X) — это категории (например, дни недели или проекты).

  * Вертикальная ось (Y) — это числовые значения (например, количество выполненных задач, трудозатраты, объем выручки).


Для того, чтобы отображались нужные данные — используется группировка и сортировка. 


## Как настроить


  * В «Вид отчета» добавьте столбцы с данными, по которым будет собран график.

  * Далее в разделе «Графики и диаграммы» добавьте новый график типа «График»:


![h5MpmT.png](images/h5MpmT.png)


  


  * В «Данных для отображения» укажите значения: 

    * Столбцы, по которым будет строиться график: «Просроченные задачи», «Выполненные задачи».

    * Группировка: По столбцу «Исполнитель».

    * Сортировка: По столбцу «Выполненные задачи» по месяцам и по возрастанию.


    ![qHcqyN.png](images/qHcqyN.png)


  


  * Во «Внешнем виде» настройте отображение графика: 

    * Высота графика — определяет высоту отображаемого графика в пикселях.

    * Расположение подписей — определяет, как будут отображаться подписи осей. Совет: "Вертикально" позволяет компактно разместить длинные подписи, а "Горизонтально" подходит для более коротких значений.

    * Цвета элементов — позволяет различать группы данных на графике с помощью разных цветов.


    ![HIIZx9.png](images/HIIZx9.png)


  


  * Сохраните график и запустите отчет для просмотра.
