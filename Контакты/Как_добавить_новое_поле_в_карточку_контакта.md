Добавить в карточку контакта дополнительное пользовательское поле можно в режиме редактирования [ шаблона контакта](Контакты/Шаблоны_контактов.md) по иконке "плюс": 


  * В открывшейся форме необходимо выбрать тип поля, задать название и настроить дополнительные параметры, зависящие от типа поля.

  * Указать права доступа к полю для сотрудников и клиентов.

  * При необходимости новое поле можно перенести на нужную панель шаблона.


## Важно


  * Если поле было создано ранее, оно будет присутствовать в списке полей.

  * Создавать такое поле еще раз нет необходимости — один раз созданное поле имеет смысл использовать многократно, во всех шаблонах контактов, для которых оно актуально.
