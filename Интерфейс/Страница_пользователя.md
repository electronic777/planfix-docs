**Страница пользователя (сотрудника)** содержит данные и настройки, относящиеся к конкретному пользователю ПланФикса. Есть два основных способа попасть на страницу пользователя: 


  * Список пользователей, находящихся онлайн, доступен по клику на иконку "Сейчас в онлайне" в верхнем правом углу. На первом месте в списке всегда находится текущий пользователь, просматривающий этот список. Если вам нужно перейти в собственную карточку пользователя, это самый быстрый способ.


  * В разделе [Сотрудники](Контакты/Сотрудники_и_компания.md) виден общий список пользователей. Клик на любого пользователя в списке позволяет попасть на его страницу (карточку):


Для обычного пользователя доступна только часть данных других пользователей аккаунта. [ Администраторы аккаунта](Доступ и права/Роли_в_ПланФиксе.md), а также сотрудники, которым дан [ специальный доступ к добавлению и редактированию сотрудников](Контакты/Пользователь_может_добавлять_новых_сотрудников.md), могут не только просматривать всю информацию о пользователе, но и редактировать ее. 


## Структура страницы пользователя


На странице пользователя доступны следующие вкладки: 


  * Общая информация

  * [ Настройки](Настройка/Настройки_в_карточке_пользователя.md)

  * [Рабочий календарь](Планирование/Рабочий_календарь.md)

  * [ Уведомления](Коммуникации/Подписки_и_уведомления.md)

  * [ Задачи по почте](Задачи/Правила_создания_задач_по_e-mail.md) \- можно задать правила обработки для писем, приходящих на [ внутренний адрес сотрудника](Задачи/Как_поставить_задачу_по_e-mail.md)

  * [Проекты](Проекты/Проекты.md) \- будет выведен список проектов, в которых принимает участие данный сотрудник

  * [Задачи](Задачи/Задачи.md) \- будет выведен список задач, в которых данный сотрудник является исполнителем

  * [Комментарии](Коммуникации/Комментарии.md) \- будет выведен список комментариев и действий сотрудника

  * [Файлы](Документы/Файлы.md) \- будет выведен список файлов, загруженных или созданных данным сотрудником

  * [Отчеты](Аналитика/Отчеты.md) \- будет выведен список отчетов, созданных данным сотрудником

  * [ Внешние адреса](Контакты/Настройка_внешних_почтовых_адресов_сотрудников.md) \- позволяет настроить для пользователя один или несколько адресов электронной почты, с которых будут отправляться сообщения, написанные им в ПланФиксе для внешних адресатов (контактов)

  * [ Безопасность](Контакты/Включение_двухфакторной_аутентификации_для_сотрудников.md) \- настройка двухфакторной аутентификации

  * [ Управление сессиями](Доступ и права/Пароль_приложения.md) \- отображает список сессий пользователя на различных устройствах и позволяет удалить скомпрометированную сессию

  * [Связь аккаунтов](Общие/Связь_аккаунтов.md) \- позволяет пользователю связать текущий аккаунт с другим используемым им аккаунтом

  * Лог - отображает лог изменений страницы пользователя


## Редактирование данных пользователя


Нажав на кнопку **Редактировать** на странице пользователя, вы увидите окно, в котором отображаются и могут быть отредактированы все данные пользователя. 


Некоторые данные пользователя (например, [Должность](Прочее/Должность.md) или [часовой пояс](Прочее/Как_изменить_часовой_пояс_.md)) могут быть изменены только в режиме редактирования. 


Помимо редактирования данных в существующих полях, в этом окне вы можете [ добавить дополнительные поля в карточку сотрудника](Контакты/Пользовательские_поля_сотрудника.md).
