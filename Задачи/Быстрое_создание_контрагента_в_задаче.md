В [ задаче](Задачи.md) есть возможность быстро создавать нового [ контрагента](../Общие/Контрагент.md). Это удобно при создании новой задачи (заявки, заказа) "на лету" — вы добавляете нового клиента прямо в процессе ввода данных его заказа: 


  

![atSsuw.png](images/atSsuw.png)


  

При нажатии на пиктограмму **Плюс** появляется окно создания нового [ контакта](../Контакты/Контакты.md) (компании или физического лица): 


  

![dvduQR.png](images/dvduQR.png)


  

В нем можно указать известные вам реквизиты контакта и сохранить его. Минимально необходимо указать название компании / ФИО контакта. После сохранения данных, созданный контакт будет выбран в качестве контрагента создаваемой задачи. 


  


## Полезно


  * По умолчанию, при нажатии на пиктограмму **Плюс** открывается окно создания нового контрагента по шаблону компании. В [ шаблоне задачи](Шаблоны_задач.md) вы можете указать, по какому шаблону будут создаваться контрагенты, добавляемые в этой задаче:


  

![hgOf5N.png](images/hgOf5N.png)


  


    Например, указав в шаблоне задачи шаблон контрагента **Шаблон контакта** (физического лица), вы упростите добавление новых клиентов в процессе создания задач (заказов, заявок), если ваша компания работает с клиентами - физическими лицами.
