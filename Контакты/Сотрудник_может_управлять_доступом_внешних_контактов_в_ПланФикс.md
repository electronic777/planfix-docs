Администратор аккаунта может [ разрешить сотруднику](Общие/Дополнительные_возможности_пользователя.md) управлять доступом внешних контактов в ПланФикс. Для этого: 


  * зайдите в [ карточку](Интерфейс/Страница_пользователя.md) нужного пользователя в разделе **Сотрудники** ;

  * нажмите кнопку **Редактировать** ;

  * разверните панель **Возможности**

  * активируйте признак **Может управлять доступом внешних контактов в ПланФикс** :


![eySfP6.png](images/eySfP6.png)


  

После этого у сотрудника появится возможность предоставлять [ внешним контактам](Контакты/Внешний_пользователь.md) доступ в ПланФикс: 


![kaocyH.png](images/kaocyH.png)
