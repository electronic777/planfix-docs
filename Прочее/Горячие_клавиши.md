## В интерфейсе


  * Esc — закрыть текущую крайне правую панель


## При работе с задачей


  * ALT+T — создание новой задачи

  * ALT+R — добавление нового комментария

  * Ctrl+Enter: 

    * сохранение текста комментария и закрытие интерфейса добавления действия

    * сохранение и закрытие окна задачи при ее создании или редактировании

    * сохранение и закрытие окна аналитики при ее создании или редактировании

    * сохранение комментария во встроенном таймере

    * сохранение напоминания

  * Ctrl+K - вставка ссылки в текст


## При работе с чек-листами


  * Enter — сохранить изменение в строке

  * Esc — отменить редактирование/создание строки

  * Tab — переместить строку на уровень ниже

  * Shift+Tab — переместить строку на уровень выше


## При просмотре документов


  * \+ (плюс) — увеличение масштаба изображения

  * \- (минус) — уменьшение масштаба изображения


## При создании и редактировании текста задачи/комментария


  * Ctrl + C — скопировать выделенный текст

  * Ctrl + V — вставить скопированный текст

  * Ctrl + X — вырезать выделенный текст

  * Ctrl + A — выделить весь текст

  * Ctrl + Z — отмена последнего действия

  * Ctrl + B — включение или отключение полужирного начертания

  * Ctrl + I — включение или отключение курсива

  * Ctrl + U — включение или отключение подчеркивания слов

  * Ctrl + Shift + V — вставка скопированного текста без форматирования


## При редактировании имени контакта


  * Ctrl+Enter — сохранение и закрытие окна контакта при быстром редактировании ФИО


## Встроенный таймер


  * Ctrl + B — запустить / остановить таймер 

    * Если таймер не запущен и открыта задача — запускается таймер без открытия панели

    * Если таймер запущен — открывается панель, таймер останавливается с запросом сохранения
