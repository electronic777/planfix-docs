Интерфейс импорта записей справочника вызывается кнопкой на панели действий в корневой папке или в группе записей [ справочника](Справочники.md): 


![Ylr7n1.jpg](images/Ylr7n1.jpg)


  


  * Поддерживаются исходные файлы форматов csv, xls, xlsx, mpp, mpx, mpd, mspdi, pod.

  * После выбора файла система просит указать, куда импортировать записи: в корень справочника или в одну из его групп.

  * В импортируемом файле можно указать принадлежность каждой записи к конкретной группе записей в справочнике. При этом поиск будет вестись по названию группы. При совпадении с названием существующей группы, записи будет импортированы в нее. Если соответствующая группа не будет найдена, система создаст ее и импортирует записи в созданную группу:


![HTCcng.jpg](images/HTCcng.jpg)


  


  * После выбора файла система анализирует его и просит сопоставить имеющиеся в файле столбцы с данными и реквизиты записей справочника в ПланФиксе:


![Kij4rB.jpg](images/Kij4rB.jpg)


**Важно**


  * Если значение в первой строке столбца совпадает с названием реквизита в ПланФиксе, сопоставление происходит автоматически — это позволяет подготовить файл определенного формата, чтобы при частом импорте не приходилось каждый раз делать сопоставление вручную.

  * Контроль дублей при импорте производится только для полей с активной галочкой "Запрещать дублирование значений".


## Создание файла с записями справочника для импорта


  * Основой импортируемого файла является таблица Excel, в которую внесены данные для импорта:


![Og1QwQ.jpg](images/Og1QwQ.jpg)


  


  * Файл может иметь любую структуру, и название колонок не должно обязательно совпадать с названиями полей в структуре справочника. Вы можете соотнести их с полями, которые используете, при задании условий импорта.

  * Готовый файл сохраняется в формате CSV (разделители — запятые) и импортируется в ПланФикс через кнопку "Импорт" в разделе [Справочники](Справочники.md).


## Важно


  * При импорте в поля, содержащие дату и время: 

    * для типа [Дата](../Общие/Дата.md) данные должны быть представлены в формате: **дд-мм-гггг**

    * для типа [Дата и время](../Время и учет/Дата_и_время.md) данные должны быть представлены в формате **дд-мм-гггг мм:чч**

    * номер дня в виде одной цифры и год в виде двух цифр не поддерживаются

    * вместо символа "—" могут использоваться символы "." (точка) или "/" (слэш).

  * Для корректного импорта файлов формата CSV кодировка файла должна быть **UTF-8 ​​​​​​​**

  * Импорт записей справочника доступен только сотрудникам компании.
