По умолчанию, при просмотре [задачи](Задачи/Задачи.md), большинство ее полей отображается во вкладке **Детали** : 


![jboWtB.png](images/jboWtB.png)


  

Для того, чтобы нужные поля отображались на основной панели карточки задачи, нужно: 


  * зайти в [ шаблон](Задачи/Шаблоны_задач.md), по которому создаются задачи этого типа

  * открыть в нем панель "Детали задачи"

  * перетащить с панели с деталями на основную панель карточки задачи нужные поля:


![KxpR1x.png](images/KxpR1x.png)


  

После этого во всех задачах, созданных по этому шаблону, данные поля будут отображаться на основной панели карточки задачи. 


## Полезно


  * В карточке задачи поля можно увидеть таким образом:


![ftmeoX.png](images/ftmeoX.png)


  


  * Обратите внимание, поля на основной панели можно расположить в нескольких столбцах.
