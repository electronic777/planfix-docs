ПланФикс обеспечивает стандартные для SaaS-сервисов меры по обеспечению безопасности и сохранности пользовательских данных: 


  * транзакционное резервное копирование в реальном режиме времени в географически разнесенные хранилища;

  * ежедневное резервное копирование данных пользовательских аккаунтов за прошедшие сутки;

  * шифрование канала связи по SSL-протоколу;

  * разграничение доступа к данным на базе ролей;

  * изолирование баз данных для разных аккаунтов;

  * дублированное хранение пользовательских файлов на сервисе Amazon S3 в разных датацентрах.
