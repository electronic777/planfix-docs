При интеграции с [ телефонией](../Общие/Виртуальные_АТС.md) ПланФикс позволяет создавать новый контакт для клиента, позвонившего впервые. 


Когда вам звонит новый абонент, которого пока нет в списке [ контактов](Контакты.md) ПланФикса, система автоматически создает для него новую карточку контакта и сохраняет в ней номер телефона, с которого поступил звонок. 


В настройках интеграции вы можете выбрать [ шаблон](Шаблоны_контактов.md), по которому будет создаваться контакт в таком случае:
