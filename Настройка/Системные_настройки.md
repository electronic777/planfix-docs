**Системные настройки** – это настройки, которые требуются для работы [ПланФикса](Общие/Описание_возможностей.md). Их действие распространяется на весь аккаунт. Системные настройки находятся в разделе «Управление аккаунтом» – «Системные настройки». 


## Разделы системных настроек


  * Информация о компании

  * Оформление

  * Дата и время

  * Задачи и проекты

  * Расширенные возможности


## Информация о компании


В этой вкладке задаются основные настройки, влияющие на работу вашей компании в ПланФиксе: 


  * Название организации

  * Логотип


### Название организации


Добавьте название компании, рабочей группы или команды. 


### Логотип


  * Система позволяет заменить стандартный логотип ПланФикса на логотип вашей компании. Добавьте нужный логотип для светлой и тёмной темы интерфейса ПланФикса. При этом возможность установки логотипа поддерживается для [тарифных планов](https://planfix.ru/prices/) Бизнес и Корпорация.


  * Вы также можете добавить свой логотип на страницу входа в аккаунт. Установите [свой домен](Прочее/Свой_домен.md) в ПланФиксе, чтобы активировать эту возможность.


## Оформление


Настройки в этой вкладке касаются внешнего вида вашего аккаунта ПланФикса: 


  * Цветовая схема

  * Изображение при входе в аккаунт

  * Отображение имен

  * Вид ленты комментариев по умолчанию


### Цветовая схема


Эта настройка позволяет задать собственные цвета для интерфейса или использовать готовые сочетания: 


Обратите внимание: 


  * Настройка цветовых схем доступна для всех платных и премиум-аккаунтов ПланФикса.

  * На уровне каждого [рабочего пространства](Прочее/Рабочее_пространство.md) может быть выбрана своя цветовая схема.


### Изображение при входе в аккаунт


Вы можете выбрать картинку для страницы входа в аккаунт. Нажмите на поле настройки, чтобы задать картинку. 


Важно: 


  * Внешний вид страницы входа настраивается администратором на уровне всего аккаунта.

  * Изображение отображается для всех сотрудников и контактов с доступом в ПланФикс.


### Отображение имен


Эта настройка влияет на формат имени, фамилии и отчества [сотрудников](Контакты/Сотрудники.md) и [контактов](Контакты/Контакты.md). Укажите нужные настройки, учитывая негласные правила коммуникации вашей команды. 


### Вид ленты комментариев по умолчанию


Эта настройка влияет на отображение ленты действий в аккаунте по умолчанию. В карточке сотрудника каждый пользователь может настроить для себя вид ленты персонально. 


На данный момент существует два варианта отображения ленты: 


  * **Современный** , в котором лента общения больше похожа на чат.

  * **Чат** , в котором общение по задачам аналогично общению в мессенджерах.


Нажмите на название подходящего варианта, чтобы переключить вид отображения.

### Формат даты для новых сотрудников и контактов


Выбранный формат даты будет установлен по умолчанию в карточках сотрудников и контактов. Настройка влияет на отображение дат в интерфейсе и уведомлениях ПланФикса: 


  * Формат даты **США** : 02/02/2022

  * Формат даты **Европа** : 02-02-2022


**Важно:** сотрудники могут изменить указанный вами формат даты в [своей карточке](Настройка/Настройки_в_карточке_пользователя.md). 


### Формат времени для новых сотрудников


Вы можете установить по умолчанию формат отображения времени в интерфейсе ПланФикса для сотрудников: 13:00 или 1:00PM. 


## Задачи и проекты


Настройки в этой вкладке связаны с датами завершения задач и их отображением на диаграмме Ганта: 


  * Дата завершения задачи

  * Даты последователя в Диаграмме Ганта

  * Завершать пункты чек-листа при выполнении или завершении задачи

  * Разрешить дате завершения подзадачи быть больше даты завершения надзадачи

  * Сдвигать дату завершения надзадачи при изменении даты подзадачи


### Дата завершения задачи


У любой задачи есть две даты завершения: 


  * Дата планируемого завершения

  * Дата фактического завершения


В зависимости от бизнес-процесса может быть важно, менять или не менять дату планируемого завершения на фактическую. Установите подходящий вариант, исходя из своих потребностей. 


### Даты последователя в Диаграмме Ганта


Эта настройка влияет на отображение задач на диаграмме Ганта. При выборе настройки: 


  * **Даты последователя стремятся к заданным вручную** — задача-последователь никуда не сдвинется на диаграмме и начнется в указанное вручную время, даже если ее предшественник завершится раньше указанного срока.

  * **Даты последователя зависят только от дат предшественника** — дата задачи-последователя рассчитается как дата завершения задачи-предшественника плюс запаздывание. Дата, которая ставилась вручную в расчет приниматься не будет.


### Завершать пункты чек-листа при выполнении или завершении задачи


Эта настройка влияет на поведение системы при переводе задачи в [системные статусы](Задачи/Какой_смысл_в_статусах_задач_.md) **Выполненная** и **Завершенная**. По умолчанию автоматическая отметка чек-листа включена. 


Это означает, что: 


  * при переводе в статус **Выполненная** или **Завершенная** будут автоматически отмечены выполненными все пункты [ чек-листа](Прочее/Чек-лист.md) задачи;

  * подзадачи, отображаемые в чек-листе, при этом изменяться не будут;

  * при переводе в статус **Завершенная** будет выдано окно с вопросом, завершать ли подзадачи. Вопрос будет задан только при переводе статуса в интерфейсе системы, там где это возможно. Пользователь сам выберет, завершать подзадачи в данном случае или нет.

  * если отметить пункт чек-листа невыполненным, задача будет автоматически возвращена в статус **В работе** или другой [ статус, который установлен для перехода по кнопке Принять](Задачи/Статус_задачи_после_принятия.md).


  

Если установить значение настройки **Выключено** : 


  * при переводе задачи в статус **Выполненная** будет задан вопрос об отметке выполненными всех пунктов чек-листа;

  * при переводе задачи в статус **Завершенная** будет задан вопрос об отметке выполненными всех пунктов чек-листа и завершении подзадач (если они есть);

  * вопросы в виде диалоговых окон будут заданы только в случае, если перевод в статус производится пользователем в интерфейсе, где есть возможность задать такой вопрос;

  * если отметить пункт чек-листа невыполненным, с задачей не будет производиться никаких автоматических операций.


### Разрешить дате завершения подзадачи быть больше даты завершения надзадачи


Активируйте эту настройку, если для вашего процесса важно, чтобы подзадачи могли завершаться позже надзадачи. 


Например, надзадача «Отчёт по НДС за 3 квартал» должна быть завершена до 25 октября, а ее подзадача «Оплата налога» длится до 25 декабря. 


### Сдвигать дату завершения надзадачи при изменении даты подзадачи


При активации этой настройки дата завершения надзадачи автоматически изменится, если даты завершения ее подзадач вышли за установленные сроки. 


Опция отображается только в случае, если выключена (установлено «Нет») настройка **Разрешить дате завершения подзадачи быть больше даты завершения надзадачи**.

  * Контакты с доступом в ПланФикс

  * Блок быстрого выбора дат

  * Цитирование

  * Лайки

  * [Быстрые ответы](Прочее/Быстрые_ответы.md)


### Контакты с доступом в ПланФикс


Эта настройка напрямую влияет на возможность контактов с доступом в ПланФикс просматривать комментарии в чате задач: 


  * **Видят все нескрытые комментарии в доступных им задачах и карточках контактов.** Контакт будет видеть все не скрытые комментарии, независимо от того уведомили его или нет.

  * **Видят только те комментарии, в которых они были явно уведомлены.** Контакт сможет увидеть только те комментарии, в которых его уведомили.


### Блок быстрого выбора дат


Если возможность активирована, тогда при выборе даты станут доступны быстрые варианты выбора: "Сегодня", "Завтра", "Через неделю" и прочие. 


### Цитирование


По умолчанию при цитировании комментария курсор для ввода текста находится ниже цитаты. 


Вы можете переместить курсор выше текста цитаты, включив настройку **Текст выше цитаты**. Этот вариант удобнее использовать при переписке из ПланФикса с пользователями, получающими ваши комментарии на email. 


### Лайки


Эта настройка позволяет включить или отключить возможность отмечать комментарии значком **Сердечка**. По умолчанию, лайки включены, но при необходимости администратор аккаунта может отключить их. 


### Быстрые ответы


Активация этой функции позволит вам вызывать с помощью **Звездочки** справочник с быстрыми ответами: 


![CzO9z0.gif](images/CzO9z0.gif)


  

Для того, чтобы включить функцию: 


  * Активируйте чек-бокс **Быстрые ответы**

  * Выберите подходящий справочник в поле **Справочник с быстрыми ответами**. При необходимости создайте его.

  * Укажите какое поле справочника нужно отобразить в **Поле с текстом ответа**.
