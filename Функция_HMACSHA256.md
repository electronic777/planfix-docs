Функция HMACSHA256() — принимает сообщения произвольной длины и специальный секретный ключ, затем вычисляется хэш-код с помощью алгоритма SHA-256. Используется в работе с API различных сервисов для их интеграции в ПланФикс. 


## Формат


**HMACSHA256(** value; key**)**


## Пример


HMACSHA256("Привет, мир!";"секрет123") 


Результат: 953f6ff93b32f11a3f9108e74d634aa749cc58a9be94cda394ddf9cbf37dbd64
