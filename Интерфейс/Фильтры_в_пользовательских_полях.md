**Фильтры** в [ пользовательских полях](../Поля и формы/Пользовательские_поля.md) некоторых типов позволяют выводить в списке доступных для выбора значений только те, которые соответствуют заданным условиям. 


Фильтры доступны в полях следующих типов: 


  * [Контакт](../Контакты/Контакт.md)

  * [ Контрагент](../Общие/Контрагент_-_Тип_поля.md)

  * [ Задача](../Задачи/Задача_-_Тип_поля.md)

  * [Набор задач](../Задачи/Набор_задач.md)


## Настройка фильтра в пользовательском поле


Для активации фильтра необходимо в настройках поля выбрать соответствующий вариант опции **Отображать в контроле выбора** и задать набор условий, которым должны соответствовать объекты, чтобы отобразиться в списке для выбора значения в этом поле.
