Функция возвращает логическое значение "истина", если в столбце отчета или анализируемом поле содержится рабочий день. При этом используется [рабочее время](Рабочий_календарь.md "Рабочий календарь") аккаунта по умолчанию (первое в списке рабочих времен аккаунта) Обычно используется в связке с [ функцией ЕСЛИ](Функция_ЕСЛИ.md "Функция ЕСЛИ"). 


  


**ЭТОРАБДЕНЬ(дата)**


**ISWORKDAY(дата)**


  

**Примеры:**


ЕСЛИ(ЭТОРАБДЕНЬ(H);G;G*2) 


  


## Важно


На текущий момент **функция не работает для формул, вычисляемых только для строк итогов или первых группировок в отчетах**.
