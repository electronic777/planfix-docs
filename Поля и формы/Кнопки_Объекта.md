ПланФикс позволяет добавлять в [Объекты](../Общие/Объекты.md) дополнительные кнопки для совершения различных операций. Они настраиваются в «Управление аккаунтом» — «Объекты» — нужный объект — «Кнопки»: 


![Kh2ZLI.png](images/Kh2ZLI.png)


  


## Добавление кнопки


Добавляя кнопку в Объект, укажите: 


  * В каких еще объектах она будет работать (это удобный способ добавить кнопку сразу в несколько объектов, если это необходимо).

  * Набор условий, при которых она будет отображаться.

  * Кто может нажимать кнопку. Даже если условия будут выполняться, кнопка будет видна только указанным в этом поле пользователям.

  * Внешний вид кнопки.

  * Набор операций, которые будут выполнены по нажатию на кнопку.

## Добавление кнопки на основе сценария


Существует быстрый способ создать кнопку на основе существующего сценария Объекта. Для этого: 


  * Выберите нужный сценарий

  * Нажмите «Скопировать как кнопку» в меню операций

  * Откроется интерфейс создания кнопки с уже добавленными данными из выбранного сценария

  * Останется выбрать пользователей, которые могут нажимать на кнопку, и настроить ее внешний вид:
