После создания [Шаблона проекта](../Проекты/Шаблоны_проектов.md), происходит переход к списку задач этого шаблона проекта. В этом списке отображаются не сами задачи, а [ шаблоны](Шаблоны_задач.md), по которым они будут созданы. По умолчанию, список пуст. Его можно наполнить шаблонами задач, чтобы при создании проекта по этому шаблону, одновременно внутри него был создан набор задач по заранее заданным шаблонам. 


## Создание шаблонов задач в рамках шаблона проекта


  * Для создания шаблонов задач нужно зайти в шаблон проекта и нажать кнопку **Новый шаблон задачи**.

  * Откроется стандартная форма [создания шаблона задачи](Шаблоны_задач.md).


## Добавление существующих шаблонов задач в шаблон проекта


В Шаблон проекта можно добавить уже существующие Шаблоны задач. Для этого достаточно в имеющемся [Шаблоне задачи](Шаблоны_задач.md), войдя в режим Редактирования шаблона, выбрать в качестве проекта нужный Шаблон проекта (Вид отображения — Шаблоны): 


![740mgc.png](images/740mgc.png)


## Важно


  * Если в существующем проекте изменить шаблон проекта, то задачи, добавленные в выбранный шаблон проекта, созданы не будут.

  * Создание шаблонных задач в рамках шаблона проектов происходит только при создании нового проекта по шаблону.
