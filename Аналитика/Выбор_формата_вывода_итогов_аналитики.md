В шаблоне документа можно указать [переменную](../Документы/Переменные_в_шаблонах_документов.md) итогов аналитики. 


Она может отображаться в докумене в двух вариантах: 


  * с форматированием

  * без форматирования


Выбрать переменную итогов аналитики можно в списке переменных для шаблона документов: 


![onZGbG.png](images/onZGbG.png)


![7xvV9S.png](images/7xvV9S.png)


  


## Переменная итогов аналитик с форматированием


Шаблон документа с переменной выглядит так: 


![Qokux1.png](images/Qokux1.png)


  

Итоговый документ: 


![CmGvo2.png](images/CmGvo2.png)


  

Обратите внимание, итог аналитики содержит два знака после запятой и обозначение денежной единицы. Т.е. при форматировании применяются указанные настройки поля: количество знаков после запятой, разделение на группы разрядов, единица измерения. 


## Переменная итогов аналитик без форматирования


Вариант переменной без форматирования: 


![cOmseD.png](images/cOmseD.png)


  

Готовый документ выглядит таким образом: 


![Z3Wy5y.png](images/Z3Wy5y.png)


Обратите внимание, итог аналитики содержит только числовое значение. 


  


## Числовое значение пользовательского поля типа «Итоги аналитик»


Значение пользовательского поля типа «Итоги аналитик» в шаблоне документа можно отобразить без форматирования, для дальнейшего его использования в других вычислениях. 


Укажите нужную переменную без форматирования: 


![7A3Zvu.png](images/7A3Zvu.png)


  

Добавьте её в шаблон: 


![HylXOr.png](images/HylXOr.png)


  

Готовый документ выглядит так: 


![kf8Ye1.png](images/kf8Ye1.png)
