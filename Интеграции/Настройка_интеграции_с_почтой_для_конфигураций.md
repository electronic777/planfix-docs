ПланФикс умеет принимать сообщения от клиентов [ по разным каналам связи](Прочее/Как_настроить_прием_заявок_из_разных_каналов.md), в том числе по почте. Для этого необходимо настроить переадресацию писем из корпоративной почты на виртуальный адрес в вашем аккаунте ПланФикса. 


  

**Важно:** в конфигурации виртуальный почтовый адрес уже создан. Увидеть его можно здесь: **Управление аккаунтом** — **Работа с e-mail** — **Виртуальные почтовые адреса** : 


![Thzaof.png](images/Thzaof.png)


  

Виртуальный почтовый адрес будет такого вида: 


![MlOlgJ.png](images/MlOlgJ.png)


  

Скопируйте его в вашем аккаунте и укажите в качестве адреса для переадресации в своей электронной почте. 


**Полезно:** Примеры настройки переадресации в популярных почтовых сервисах: 


  * [ Gmail](Поля и формы/Получение_писем_Gmail_в_ПланФикс.md)

  * [ Mail.ru](Поля и формы/Получение_писем_Mail.ru_в_ПланФикс.md)

  * [ Яндекс Почта](Коммуникации/Получение_писем_Яндекс_Почты_в_ПланФикс.md)


После этого все письма, приходящие на ваш обычный адрес электронной почты, будут переадресовываться в ПланФикс. Таким образом, систему можно использовать как для обычной почтовой переписки, так и для приема заявок от клиентов, получения заказов с сайта и т.п.. 


  


## Исходящие письма


  * Так как ПланФикс не является почтовым клиентом в привычном понимании, он использует раздельное получение и отправку писем. Вы можете управлять тем, с какого ящика будут уходить ответы на письма, отправляемые из ПланФикса. По умолчанию, письма отправляются с [ внутреннего адреса сотрудника в ПланФиксе](Задачи/Как_поставить_задачу_по_e-mail.md). Вы можете настроить [ отправку писем с внешнего ящика сотрудника](Коммуникации/Отправка_писем_с_внешнего_почтового_ящика.md).


  


## Полезно


  * Дополнительные параметры создаваемых из писем задач могут быть настроены с помощью [ правил создания задач по почте](Задачи/Правила_создания_задач_по_e-mail.md).

  * В переписке с клиентами для ответа на типовые вопросы вы можете использовать механизм [быстрых ответов](Прочее/Быстрые_ответы.md).
