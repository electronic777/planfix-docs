В ПланФиксе осуществить экспорт [ контактов](Контакты.md "Контакты") можно несколькими способами: 


  * из фильтра контактов;

  * при помощи отчетов;

  * в рамках общего экспорта данных.


## Навигация


  * Экспорт контактов из фильтра контактов

  * Экспорт контактов при помощи отчетов

  * Экспорт контактов в рамках общего экспорта данных

  * Лог экспорта контактов


## Экспорт контактов из фильтра контактов


Запустить экспорт контактов можно из любого [Фильтра контактов](Фильтры_контактов.md "Фильтры контактов") при помощи кнопки "Экспорт". Она позволяет получить файл со списком контактов, которые попадают под условия этого фильтра. По умолчанию указан файл Excel, также можно выбрать файл CSV или Google Таблицу. 


Чтобы экспортировать только контакты, подходящие под определенные критерии, отфильтруйте их при помощи быстрых фильтров, доступных по клику на соответствующие столбцы списка:

## Экспорт контактов при помощи отчетов


Для экспорта контактов необходимо: 


  * [ Создать новый отчет](Как_создать_новый_отчет_.md "Как создать новый отчет?").

  * Выбрать нужные параметры отображения [общей информации](Отчеты__Общая_информация.md "Отчеты: Общая информация") и задать нужный [вид отчета](Отчеты__Вид_отчета.md "Отчеты: Вид отчета").


    **Важно:** добавьте в отчет столбцы для **каждого** реквизита контактов/компаний, который хотите увидеть в итоговом файле.


  * [Настроить фильтр](Отчеты__Параметры_отбора.md "Отчеты: Параметры отбора") для отбора данных в отчет — это позволит отобрать контакты по определенному сочетанию условий. Если не задавать никаких условий, в отчет попадут **все** контакты и компании, добавленные в систему.

  * [Запустить отчет](Как_запустить_отчет_на_выполнение_.md "Как запустить отчет на выполнение?").

  * Готовый отчет можно [экспортировать](Экспорт_отчетов.md "Экспорт отчетов") в Excel. В результате вы получите файл с необходимыми данными по [контактам и компаниям](Контакты.md "Контакты").


## Экспорт контактов в рамках общего экспорта данных


Вы также можете произвести экспорт контактов в рамках [ общего экспорта](Экспорт_данных.md "Экспорт данных") всех данных и файлов в разделе **Управление аккаунтом**. Данные будут выгружены в виде zip-архива, разбитого по разделам, в формате JSON. 


Общий экспорт доступен только владельцу аккаунта и может быть выполнен не чаще, чем один раз в 30 дней, в силу большой нагрузки на систему при проведении этой операции.
