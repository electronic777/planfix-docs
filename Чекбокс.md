Поля данного типа служат для хранения логических значений. Поле имеет два состояния: чекбокс активирован / чекбокс не активирован (True / False). Создать поле типа **Чекбокс** можно в качестве: 


  * [ пользовательского](Пользовательские_поля.md "Пользовательские поля") поля,

  * поля [ справочника](Справочники.md "Справочники"),

  * поля [ аналитики](Аналитика.md "Аналитика").


## Параметры для поля этого типа в аналитиках:


  * **Итоги аналитик** — позволяет суммировать варианты True по строкам аналитики;

  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.


## Логика работы, если поле обязательно для заполнения


Поле типа Чекбокс считается заполненным только при выборе варианта «Да». Чаще всего эту логику используют для кейсов, в которых нужно обязательное ознакомление с задачей.
