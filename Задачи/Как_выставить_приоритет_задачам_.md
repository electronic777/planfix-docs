В ПланФиксе есть возможность реализовать разные способы задания приоритета задачам. Наиболее правильным мы считаем [ ручную сортировку](Задачи/Ручная_сортировка_задач.md) задач по приоритетам. 


Кроме этого, у задач в есть два стандартных уровня приоритета: обычная и срочная, которыми можно пользоваться: 


![piULtY.png](images/piULtY.png)


  

Еще один удобный вариант — сделать для этой цели [ пользовательское поле](Задачи/Пользовательские_поля_задачи.md) (или даже несколько), проставлять в нем приоритет по своей любимой методике и потом сортировать/группировать в [ фильтрах](Задачи/Фильтры_задач.md) и [ отчетах](Аналитика/Отчеты.md) задачи по этому полю.
