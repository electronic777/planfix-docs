Строка поиска всегда находится под узнаваемой пиктограммой в правой верхней части рабочего окна системы. Она позволяет провести глобальные поиск по всем данным вашего аккаунта: задачам, действиям, контактам, названиям и описаниям файлов: 


## Принципы работы поиска


Поиск возможен по **словоформам** и **по части слова** : 


  * При поиске **по части слова** поиск работает при введении трех и более символов, то есть по двум буквам найти объект не получится. В результатах поиска выводятся те объекты, которые содержат запрос в начале слова. Например, по запросу “коло” будет найден контакт с фамилией "Колосов", но не будет найдет контакт с фамилией “Околотков”.

  * Поиск **по словоформам** работает при введении в строку поиска слова целиком. Такой поиск понимает склонения, падежи и спряжения. Это удобно, когда нужно найти объект по примерному сочетанию слов. Например, запрос "заявка атс" поможет найти задачу "Оформленные заявки на установку АТС".


Выбранный режим поиска "По словоформам" или "По части слова" запоминается автоматически и применяется до момента, пока вы сами его не измените. 


Поиск по части слова работает по всем сущностям ПланФикса – [задачам](Задачи/Задачи.md), [проектам](Проекты/Проекты.md), [контактам](Контакты/Контакты.md), [справочникам](Аналитика/Справочники.md) и так далее. 


## Поиск по пользовательским полям


Задачу, контакт или другую сущность ПланФикса можно найти по значению в [пользовательском поле](Задачи/Пользовательские_поля_задачи.md). 


На текущий момент поиск производится только по полям числового и текстового типов. 


## Сортировка результатов поиска


Вы можете управлять сортировкой результатов поисковой выдачи по каждому из объектов (Задачи, Проекты и т.п.): 


  * по дате

  * по релевантности


Если выбрана сортировка по дате, то первыми будут выдаваться наиболее свежие данные. 


## Дополнительные опции поиска


  * Заключите свой запрос в кавычки (пример: **"заявка на оплату"**) и получите в поиске только те задачи, которые содержат именно такое словосочетание.


## Поиск в Избранном


Клик по звездочке в окошке глобального поиска вызывает панель [Избранного](Прочее/Избранное.md), на которой отображаются важные для вас объекты, которые вы отметили значком Избранного. 


Когда открыта панель Избранного, строка глобального поиска работает только по этой панели. 


## Автофокусировка поиска


При начале набора любого текста на клавиатуре, если курсор не стоит в конкретном поле, текст будет вводиться сразу в строке глобального поиска. Такую возможность каждый сотрудник настраивает самостоятельно в своей карточке в разделе «Интерфейс»:
