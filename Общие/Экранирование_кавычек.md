Если одним из аргументов любой [функции](Прочее/Функции.md) является строка с кавычками (или другими спец. символами) внутри, тогда эти внутренние кавычки обязательно надо экранировать. 

    

    

    ФУНКЦИЯ("строка \"кавычка")


  

Например, есть строка и вам надо получить ссылку: 

    

    

    <a href="https://site.com/oferta/krem-1229">Крем для кожи</a>


  

Используйте функцию [REGEXPFIND](Прочее/Функция_REGEXPFIND.md), обязательно экранируйте внутренние кавычки вокруг ссылки (выделено красным): 


REGEXPFIND({{Задача.Строка для ссылки}};"(?<=href=**\** ").*?(?=**\** ")")


  

В итоге в [вычисляемом поле](Задачи/Вычисляемые_поля_задач.md) получите нужную ссылку: 

    

    

    https://site.com/oferta/krem-1229


## Все экранируемые спецсимволы


Написано внутри строки |  | Останется при её обработке   

---|---|---  

\" | -> | "   

\\\ | -> | \   

\n | -> | символ переноса строки   

\t | -> | символ табуляции
