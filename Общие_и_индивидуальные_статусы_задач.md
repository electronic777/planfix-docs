Статусы задач могут быть общими и индивидуальными: 


  * Если у статуса установлен признак **Статус устанавливается для всех участников** , то при переводе задачи в этот статус, он автоматически меняется для всех исполнителей и постановщика задачи.


  * Если у статуса установлен признак **Статус устанавливается для каждого в отдельности** , то исполнитель, изменяя статус, изменяет его только для себя. Для остальных исполнителей и постановщика задачи статус остается в том состоянии, что был до изменения задачи. Когда все исполнители изменят статус на этот, задача автоматически приобретает этот статус для постановщика и участников задачи.


Этот механизм нужен для того, чтобы отслеживать, кто из исполнителей уже выполнил свою часть работы по задаче, а кто еще нет. Т.к. статусы имеют свои цветовые настройки, на панели **Детали задачи** можно сразу увидеть, в каком статусе находится задача для каждого из исполнителей.
