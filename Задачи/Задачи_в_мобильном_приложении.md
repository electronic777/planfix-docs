**Задачи** — раздел мобильного приложения, в котором можно создавать новые [ задачи](Задачи.md) или работать с уже существующими. В нем доступны фильтры и сортировка по полям задач. 


Для того, чтобы открыть раздел, нажмите на вкладку **Задачи** в главном меню приложения. Загрузится фильтр задач, установленный по умолчанию в веб-версии ПланФикса: 


![mh6QjL.png](images/mh6QjL.png)


  

Для того, чтобы открыть другой фильтр, нажмите на название текущего и выберите нужный в выпадающем списке. 


## Сортировка по полям задачи


Нажмите на название поля, чтобы активировать фильтр по нему: 


![VHFCDp.png](images/VHFCDp.png)


  

Используйте **Поиск** по списку значений фильтра, чтобы быстро найти нужное значение. 


## Отображение задач


Еще один способ упростить работу с задачами — настроить список их отображения. Откройте **Меню** раздела: 


![HBrA6z.png](images/HBrA6z.png)


  

Выберите: 


  * Вид списка, чтобы отобразить задачи Таблицей или Древовидно.

  * Отфильтровать, чтобы оставить в списке только Активные, Просроченные или собственные (Мои) задачи.

  * Настройку **Выберите** , чтобы выбрать сразу несколько задач и провести над ними массовые действия.


## Создание новых задач


Нажмите на **Плюс** , чтобы создать новую задачу: 


![hWRqOe.png](images/hWRqOe.png)


  

Новую задачу в разделе можно создать тремя способами: 


  * Голосовой ввод

  * Краткая форма создания

  * Полная форма создания


## Важно


  * Задачи, созданные в пользовательских фильтрах, будут соответствовать настройкам отбора этих фильтров.

  * Процесс [создания новых задач](../Интерфейс/Мобильное_приложение_—_Хроника.md) одинаковый во всех разделах приложения.
