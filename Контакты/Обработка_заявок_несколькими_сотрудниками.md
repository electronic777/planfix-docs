В рамках конфигурации [CRM: Прием и обработка заявок](../CRM/CRM__Прием_и_обработка_заявок.md) подключиться к переписке с новым клиентом может любой менеджер, приняв задачу-переписку из списка **Новый клиент** : 


![wSCTMh.png](images/wSCTMh.png)


  

На всех остальных этапах общения, после принятия задачи, каждый сотрудник будет видеть только свои переписки и оформленные заказы/покупки: 


![Nr8sG3.png](images/Nr8sG3.png)


  

В планировщике **Выполнение заказа** также могут работать одновременно несколько сотрудников. В списке **Заказ отправлен на выполнение** отображаются все покупки, которые необходимо обработать. Каждый сотрудник может взять в работу любую задачу-покупку: 


![Ur0IVz.png](images/Ur0IVz.png)


  

На остальных этапах выполнения заказа, каждый сотрудник видит только свои задачи: 


![nzn3hG.png](images/nzn3hG.png)


  

Для того, чтобы все сотрудники могли видеть обращения новых клиентов, необходимо: 


  * создать для них [ рабочую группу](../Общие/Рабочие_группы.md);

  * созданную группу указать исполнителем в шаблоне **Диалог с клиентами** :


![giryUW.png](images/giryUW.png)


  

Для раздела **Выполнение заказа** также можно создать соответствующую рабочую группу и указать ее в исполнителях шаблона **Покупка**.
