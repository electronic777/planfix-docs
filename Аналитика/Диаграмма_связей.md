Диаграмма связей — это [вид списка планировщика](../Общие/Виды_списков_планировщика.md), с помощью которого можно добавить блоки или задачи на доску и установить между ними необходимые связи для визуального планирования и реализации бизнес-идей. Это не просто «рисовалка» диаграмм, а часть ПланФикса с прямой связью с задачами: 


![YKr6Jv.png](images/YKr6Jv.png)


![rfqEUd.gif](images/rfqEUd.gif)


  

Установленные связи между задачами «живут» только в рамках одной диаграммы. Это значит, вы легко сможете перетаскивать из списка планировщика одни и те же задачи на разные диаграммы, при этом на каждой из них связывать эти задачи между собой по-разному. 


## Добавление блока и создание задачи


  * Нажмите на новый блок, чтобы добавить его на диаграмму:


![dfRUNx.gif](images/dfRUNx.gif)


  


  * Или нажмите на иконку существующих задач, чтобы выбрать нужную из списка:


![O3BsZ1.png](images/O3BsZ1.png)


## Управление


  * Двойной клик по блоку = изменить название.

  * Колесо мыши вверх-вниз = зум доски. Вниз — отдалить, вверх — приблизить.

  * Зажатая ПКМ (правая кнопка мыши) на пустом месте или зажатый пробел + ЛКМ (левая кнопка мыши) = перемещение доски в любую сторону.

  * Клик ЛКМ по блоку = выделение карточки и отображение дополнительных возможностей.

  * Зажатый CTRL: 

    * Клик ЛКМ по очереди на блоки = выделение сразу нескольких карточек.

    * Повторное нажатие ЛКМ на выделенный блок = снятие выделения.

  * Зажатой ЛКМ нарисовать прямоугольник вокруг блоков = выделение сразу нескольких карточек. Выделенные карточки можно перемещать, зажав любую из них ЛКМ.

  * Клик по стрелке, соединяющей блоки = выделение связи. Нажмите кнопку «Delete», чтобы связь удалить.

  * CTRL + C = скопировать один или несколько блоков / задач.

  * CTRL + V = вставить из буфера обмена скопированные блоки / задачи.

  * CTRL + Z = отменить последнее изменение на диаграмме.

  * F2 — запустить редактирование текста в блоке.

  * Клик по лупе = развернуть диаграмму на весь экран:


![riBuRE.png](images/riBuRE.png)


## Важно


  * Связи в диаграмме не стоит путать со [связями между задачами](../Задачи/Связи_между_задачами.md) на панели «Зависимости», которые влияют на даты планируемого начала или завершения задач.
