В этом подразделе раздела "Документы" содержатся все доступные пользователю файлы [ шаблонов экспорта данных из отчетов](Экспорт_данных_из_отчетов.md): 


  * В подразделе отображаются только те файлы, которые пользователь может редактировать. Обычно это шаблоны, которые создал сам пользователь.

  * Наличие всех доступных шаблонов в одном месте упрощает процедуру их ревизии и замены более актуальными версиями.


## Пример шаблона экспорта данных


Шаблон можно скачать, кликнув по ссылке: 


  * [Отчёт: Тайм-трекинг](https://p.pfx.so/pf/su/TojpOf.xlsx)


## Требования к шаблону:


  * Файл шаблона должен быть в формате эксель *.xlsx, допускается также использование формата *.xlsm (эксель-файл с поддержкой макросов)

  * Лист, куда Планфикс экпортирует отчет, должен быть первым

  * Если лист скрыт и он первый, то данные будут экспортироваться в него в любом случае.
