[Импорт контактов](Импорт_контактов.md "Импорт контактов") позволяет загружать в ПланФикс данные компаний и контактов из других источников, а также обновлять эти данные. Импорт производится из заранее подготовленного файла в одном из поддерживаемых форматов: 


  * Файлы формата CSV распространены как способ выгрузки данных из различных программ и сервисов. Они представляют собой строки со значениями, разделенными запятой или точкой с запятой.

  * Также основой импортируемого файла может являться таблица Excel, в которую внесены произвольные данные контактов, например: Имя, Фамилия, Номер телефона, и любые другие, которые вы используете при создании контактов в вашем аккаунте.


Файл может иметь любую структуру. Название колонок не должно обязательно совпадать с названиями полей в карточке контакта. Вы можете соотнести их с полями, которые используете, при задании условий импорта. 


## Важно


  * Обязательным является хотя бы одно из стандартных полей для имени контакта или названия компании. Без этого импорт произвести не получится.

  * При импорте в поля, содержащие дату и время: 

    * для типа [Дата](Дата.md "Дата") данные должны быть представлены в формате: **дд-мм-гггг**

    * для типа [Дата и время](Дата_и_время.md "Дата и время") данные должны быть представлены в формате **дд-мм-гггг мм:чч**

    * номер дня в виде одной цифры и год в виде двух цифр не поддерживаются

    * вместо символа "-" могут использоваться символы "." (точка) или "/" (слэш).

  * для корректного импорта файлов формата CSV кодировка файла должна быть **UTF-8**


Получившийся файл используется для [импорта контактов в ПланФикс](Импорт_контактов.md "Импорт контактов").
