Правила перехода между статусами настраиваются для каждого [ набора статусов](Задачи/Создание_набора_статусов.md) во вкладке **Управление аккаунтом** — **Объекты** — **Нужный Объект** — **Набор статусов**. 


## Добавление статуса в набор


  * Для добавления перехода в новый статус нужно навести курсор мыши на статус, из которого нужен переход, и кликнуть на появившийся плюс.

  * Появится окно, в котором можно выбрать статус для перехода в него.

  * По клику на добавленный статус откроется меню дополнительных настроек, в нем можно: 

    * определить, кто имеет право на перевод в данный статус

    * добавить отдельную кнопку для статуса (кнопка настраивается **для каждого** перехода статуса)

    * добавить [ автоматический сценарий](Автоматизация/Автоматические_сценарии.md)

    * удалить переход в статус


## Удаление статуса из набора


Для удаления статуса из набора статусов: 


  * Зайдите в набор статусов нужного Объекта.

  * Наведите курсор мыши на нужный статус и нажмите на иконку корзины.

  * Таким образом статус будет удален из набора, однако останется в системе, его можно найти по пути: **Управление аккаунтом** — **Статусы**.


## Советы по настройке


  * В процессе настройки правил перехода не забывайте об «экстренной остановке» задачи — в большинстве случаев, задача может быть в любой момент отменена или завершена, а значит надо обязательно добавлять соответствующие переходы.

  * Также необходимо помнить о том, что во многих случаях задача может быть возвращена на предыдущий шаг. Необходимо хорошо представлять себе все варианты путешествия задачи по ее жизненному циклу и устанавливать все возможные варианты перехода.

  * [Дополнительные параметры набора статусов](Задачи/Дополнительные_параметры_набора_статусов.md) помогут вам организовать нужное поведение в процессе работы над задачей.

  * Набор статусов можно отобразить в разных видах, кнопка переключения расположена сверху справа сразу над набором. Существуют следующие виды отображения набора статусов: 

    * Список — отображаются сразу все переходы из статуса в статус.

    * Таблица — отображаются права на перевод в статус и дополнительные настройки.

    * Сценарии — отображаются права на перевод в статус и автоматические сценарии, связанные со статусом.
