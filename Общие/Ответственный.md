Для каждого [контакта](../Контакты/Контакты.md) можно определить ответственного за него сотрудника. Эта роль задается в [карточке контакта](../Контакты/Карточка_контакта.md) — Права доступа к контакту — в поле **Ответственный** , не обязательно для заполнения. 


## Ответственный в фильтрах контактов


Для удобства отслеживания заполненности этого поля можно использовать [фильтр](../Контакты/Фильтры_контактов.md) контактов с параметром отбора **Без ответственных**. 


Например, вот такая настройка фильтра отобразит в списке только контакты, у которых не выбран ответственный сотрудник: 


![p77l2z.png](images/p77l2z.png)


## Роль Ответственного в интеграции с ВАТС


При интеграции с [виртуальной АТС](Виртуальные_АТС.md) ответственный, указанный в карточке контакта видит окно входящего вызова по своим контактам и уведомляется о [ пропущенных звонках](../Время и учет/Учет_пропущенных_звонков.md). Если в ответственных указана группа сотрудников, то окно входящего вызова увидит один из сотрудников группы. 


При необходимости эту опцию можно отключить в настройках интеграции. 


## Важно


  * При назначении ответственным, сотрудник автоматически добавляется в список **С правом редактирования** :


![Swl8jH.png](images/Swl8jH.png)


При этом: 


  * Вы можете удалить сотрудника из списка "С правом редактирования", если ему не нужен доступ на редактирование этого контакта. Доступ к просмотру данных контакта у него останется, пока он находится в списке ответственных.

  * Если убрать сотрудника из ответственных, он автоматически будет удален из списка "С правом редактирования".

  * При добавлении сотрудника в список ответственных для компании, этот сотрудник автоматически добавляется в список ответственных для всех контактов этой компании. После этого работа с ответственными за контакты компании происходит так же, как для обычного контакта (см. предыдущий пункт).

  * Если сотрудник принимает звонок от нового контакта, он автоматически становится ответственным за этот контакт.

  * Роль ответственного не дает сотруднику автоматического доступа к [ задачам контакта](../Задачи/Связь_Контактов_и_Задач.md).

  * Если в виртуальной АТС включен перевод звонка на ответственного и у него задан ВРИО, звонок будет переведён на ВРИО.
