Поля данного типа служат для хранения информации о контактах. Создать поле типа **Контакт** можно в качестве [ пользовательского поля](../Поля и формы/Пользовательские_поля.md), поля [ справочника](../Аналитика/Справочники.md) и поля [ аналитики](../Аналитика/Аналитика.md). 


## Параметры для поля этого типа в справочниках:


  * **Обязательное для заполнения** — не даст сохранить запись справочника, пока это поле не содержит значения;

  * **Отображать в выборе записи** — позволяет выбрать дополнительные поля справочника для отображения.


## Дополнительные параметры для поля этого типа в аналитиках:


  * **Переносить заголовок** — позволяет переносить длинные заголовки на новую строку.


За отображение контакта в этом поле отвечает настройка [ Отображать в списке выбора участников задачи](Как_работать_с_клиентом_по_e-mail_.md) в карточке контакта. 


## Полезно


  * При помощи [ фильтров в пользовательских полях](../Интерфейс/Фильтры_в_пользовательских_полях.md) вы можете выводить в списке для выбора в поле только контакты, соответствующие заданным условиям.
