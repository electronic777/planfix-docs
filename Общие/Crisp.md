Добавьте на сайт компании чат Crisp и [интегрируйте](../Интеграции/Интеграции.md) его с ПланФиксом, чтобы стать ближе к клиентам и оперативно отвечать на их вопросы. 


## Возможности


  * **Автоматизация процессов**. Чат с клиентом будет автоматически становится задачей в ПланФиксе.

  * **Командная работа**. К задаче в ПланФиксе можно подключить коллег и работать сообща.

  * **Накапливание и анализ данных**. Задачи позволяют отслеживать время, затраченное на работу по ней, анализировать качество работы сотрудников, расходы и другие показатели.

  * **Обмен файлами**. Менеджер может автоматически или вручную формировать файлы и отправлять их клиенту прямо в чат.


## Подключение интеграции


  * Зарегистрируйтесь в [Crisp](https://crisp.chat/).

  * Получите код чата, скопируйте его и добавьте на свой сайт:


![bhS6cg.png](images/bhS6cg.png)


  


  * Нажмите на эту [ссылку](https://app.crisp.chat/initiate/plugin/b0ee38ff-995a-4675-bddc-874daccfbacf/), чтобы подключить интеграцию с ПланФиксом.


## Настройка в ПланФиксе


  * Перейдите в раздел **Управление аккаунтом** — **Интеграции** — **Чаты** — **Crisp** — **Добавить**. Заполните параметры интеграции:


![8eDhH3.png](images/8eDhH3.png)


  


  * Добавьте название компании.

  * В поле Website ID добавьте ID веб-сайта. Его можно скопировать в вашем аккаунте Crisp в разделе **Настройки сайта** в блоке **Инструкции по установке** :


![Zrh1We.png](images/Zrh1We.png)


  


  * В поле **Введите зарегистрированный email оператора** добавьте тот email, который использовали для регистрации в Crisp.

  * Укажите шаблон, по которому будут автоматически создаваться задачи по чатам Crisp.

  * Добавьте шаблон контакта, по которому будут создаваться карточки новых клиентов из чатов Crisp.

  * Сохраните настройки.
