## Идеология ПланФикса


Основные сущности, с которыми работает ПланФикс это **Проект** , **Задача** и **Действие** ([|подробнее об их предназначении](https://planfix.ru/ideologiya/)). Эти названия не обязывают Вас использовать ПланФикс только для проектной работы. Вот примеры названий проектов и задач, которые хорошо иллюстрируют разные варианты использования ПланФикса: 


  


* * *


**Проект:** Открытие магазина на Чкалова 18 


**Задачи:**


  * Заказ мебели

  * Санитарное разрешение

  * Пожарники: закупка оборудования

  * Пожарники: получить разрешение

  * ...


* * *


  

**Проект:** Работа с НЭУ ГПЦНиТТМ 


**Задачи:**


  * 2 принтера и факс

  * Ноутбук главбуху

  * Встреча с шефом

  * Поставка учебного класса

  * ...


* * *


  

**Проект:** Заказы с eshopexample.com 


**Задачи:**


  * Order #125948

  * Order #125949

  * Order #125950

  * Order #125951

  * Order #125952

  * ...


(Примечание к последнему примеру: Вы можете автоматически загружать задачи из разных источников с помощью функционала [Задачи-Почтой](Общие/Работа_в_ПланФиксе_с_помощью_e-mail.md)) 


  

Продумайте, каким будет Ваш подход к названию проектов и задач и расскажите о нем своим коллегам.
