Гистограмма — это тип диаграммы в [отчетах](Аналитика/Отчеты__Графики_в_отчетах.md) ПланФикса. Данные отображаются в виде столбцов. Каждый столбец показывает количество или значение, соответствующее определённой категории. Он полезен для сравнения данных. 


Например, так могут отображаться активные и завершенные задачи сотрудников за месяц: 


![ijCpLF.png](images/ijCpLF.png)


  

Гистограмма строится на основе данных из отчета и состоит из двух осей: 


  * Горизонтальная ось (X) — это категории (например, названия товаров, группы сотрудников, проекты).

  * Вертикальная ось (Y) — это числовые значения (например, количество задач, выручка, трудозатраты).


Для того, чтобы отображались нужные данные — используется группировка и сортировка. 


## Как настроить


  * В «Вид отчета» добавьте столбцы с данными, по которым будет собрана гистограмма.

  * Далее в разделе «Графики и диаграммы» добавьте новый график типа «Гистограмма»:


    ![M534sv.png](images/M534sv.png)


  


  * В «Данные для отображения» укажите значения: 

    * Столбцы, по которым будет строиться график: «Активные задачи», «Завершенные задачи».

    * Группировка: По столбцу «Исполнитель» и по столбцу «Завершенные задачи».

    * Сортировка: По столбцу «Активные задачи» и по возрастанию.


    ![4K7GDV.png](images/4K7GDV.png)


  


  * Во «Внешнем виде» настройте отображение графика: 

    * Высота графика — определяет высоту отображаемого графика в пикселях.

    * Расположение подписей — определяет, как будут отображаться подписи осей. Совет: "Вертикально" позволяет компактно разместить длинные подписи, а "Горизонтально" подходит для более коротких значений.

    * Цвета элементов — позволяет различать группы данных на графике с помощью разных цветов.


    ![ksF4wh.png](images/ksF4wh.png)


  


  * Сохраните график и запустите отчет для просмотра.
