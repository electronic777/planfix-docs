Во вкладке **Настройки** [ страницы пользователя](Страница_пользователя.md "Страница пользователя") собрано все, что касается настроек его [ интерфейса](Интерфейс_ПланФикса.md "Интерфейс ПланФикса"). 


## Язык и время


  * [ Язык интерфейса](Язык_интерфейса_для_сотрудника.md "Язык интерфейса для сотрудника") — язык, на котором сотруднику будут отображаться объекты в интерфейсе системы. Не распространяется на контент, загруженный в систему: названия и содержимое задач, комментариев, пользовательских полей и т.п.

  * **Язык общения** — язык, на котором общается сотрудник.

  * **Часовой пояс** — влияет на рабочее время, а также время задач, напоминаний и других объектов ПланФикса для этого сотрудника.

  * **Формат даты** — в каком виде будут отображаться даты для этого сотрудника.

  * **Формат времени** — в каком виде будет отображаться время для этого сотрудника.


## Личные настройки


  * Вид по умолчанию 

    * **Рабочее пространство по умолчанию** — [Рабочее пространство](Рабочее_пространство.md "Рабочее пространство"), в котором окажется сотрудник при входе в ПланФикс.

    * **Раздел по умолчанию** — раздел ПланФикса, в котором окажется сотрудник при входе в систему, если он работает в базовом рабочем пространстве.

    * **Фильтр задач по умолчанию** — [ фильтр задач](Фильтры.md "Фильтры"), который будет открываться при входе в раздел [Задачи](Задачи.md "Задачи") или в [ проект](Проекты.md "Проекты").

    * **Запрет доступа к базовому пространству** — активная настройка запрещает сотруднику доступ к базовому рабочему пространству.


  * Шаблоны 

    * [Шаблон задачи по умолчанию](Шаблон_задачи_по_умолчанию.md "Шаблон задачи по умолчанию") — шаблон, по которому сотрудник будет создавать задачи.

    * **Шаблон контакта по умолчанию** — шаблон, по которому сотрудник будет добавлять контакты.

    * **Шаблон компании по умолчанию** — шаблон, по которому сотрудник будет добавлять компании.


  * Дополнительные функции 

    * **Автофокусировка поиска** — при включенной автофокусировке любой ввод текста на странице ПланФикса, когда пользователь не находится в поле ввода, будет воспринят как [ поисковый запрос](Строка_поиска.md "Строка поиска") по системе.

    * [Дополнительное подтверждение прочтения события в Хронике](Дополнительное_подтверждение_прочтения_события_в_Хронике.md "Дополнительное подтверждение прочтения события в Хронике") — дополнительная защита от случайного клика на отметку "Прочитано" в Хронике.

    * [Таймер](Таймер.md "Таймер") — включает / выключает встроенный таймер.

    * **Предупреждение о том, что никто не уведомлен в комментарии** — всплывающее окно, которое оповещает о том, что в комментарии никто не уведомлен, т.е. сообщение без адресата.

    * **Быстрое цитирование** — включает / выключает появление контекстного меню цитирования при выделении блока текста.


## Оформление


  * [ Шрифт](Выбор_шрифта.md "Выбор шрифта") — позволяет выбрать один из доступных вариантов шрифта для системных элементов интерфейса ПланФикса.

  * **Форма аватара** — в каком виде для пользователя будут отображаться аватары сотрудников и контактов.

  * **Системное уведомление** — управляет внешним видом системных уведомлений в интерфейсе ПланФикса.

  * **Вид ленты комментариев по умолчанию** — регулирует вид отображения ленты действий и комментариев в карточке задачи.

  * **Ограничение видимой длины ссылок** — включение опции приводит к автоматическому ограничению видимой длины ссылок, вставленных в описание задачи, текст комментария и текстовые поля. Сама ссылка при этом остается работоспособной.

  * [Уведомления на вкладке браузера](Счетчик_непрочитанных_событий.md "Счетчик непрочитанных событий") — включает либо отключает отображение количества непрочитанных событий в заголовке страниц ПланФикса.

  * [Адаптация для людей с нарушением восприятия цвета](Адаптация_для_людей_с_нарушением_восприятия_цвета.md "Адаптация для людей с нарушением восприятия цвета") — позволяет установить особые настройки интерфейса для людей с нарушением восприятия цвета.
