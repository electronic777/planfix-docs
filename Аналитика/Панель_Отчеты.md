На панели **Отчеты** [ карточки задачи](../Задачи/Карточка_задачи.md) отображается набор [ отчетов](Отчеты.md), которые можно запустить для отображения данных по этой задаче. Этот механизм позволяет быстро выводить сводные данные по конкретной задаче в настраиваемом пользователем виде. Например, свод по затраченному и оплаченному времени на выполнение задачи, диаграмма распределения времени, которое задача находилась на стороне клиента и сотрудников службы поддержки, и так далее. 


  * Список отображаемой на этой панели отчетов изменяется в [ шаблоне задачи](../Задачи/Шаблоны_задач.md), на аналогичной панели.

  * Вы можете добавлять в список обычные отчеты. При запуске система автоматически произведет фильтрацию данных и отобразит в отчете только данные, относящиеся к конкретной задаче, из которой вы запускаете этот отчет.


## Отображение отчетов на панели


Включить отображения любого из существующих отчетов необходимо: 


  * Открыть [ шаблон](../Задачи/Шаблоны_задач.md), по которому создается задача

  * На панели справа выбрать Отчеты

  * Напротив нужного отчета нажать на пиктограмму лампочки


После этого данный отчет будет отображаться на панели Отчеты во всех задачах, созданных по этому шаблону задачи. 


## Полезно


  * В панели **Отчеты** карточки контакта работает аналогичный механизм, позволяющий в один клик запускать любой [ отчет](Отчеты.md) с дополнительным фильтром по этому контакту или компании. При этом он позволяет [выбирать роль](https://p.pfx.so/pf/Zz/VPwKUH.jpg), в которой данный контакт или компания находятся в задачах или действиях, которыми оперирует отчет.

  * [ В панели **Отчеты** карточки проекта](../Проекты/Отчеты_по_проекту.md) также работает аналогичный механизм, позволяющий быстро запустить отчет по всем задачам проекта.
