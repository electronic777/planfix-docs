В [ виде отображения](Прочее/Виды_списков_планировщика.md) "Календарь по дням" отображаются даты завершения задач. Каждая задача представляет собой прямоугольник, расположенный в столбце с датой ее завершения. Если у задачи помимо даты завершения указана и дата начала, то она имеет вид полоски, протяженностью от столбца с датой начала задачи до столбца с датой ее завершения. 


![9dKWAB.png](images/9dKWAB.png)


## Параметры настройки списка типа "Календарь по дням"


В параметрах списка вы можете самостоятельно определить, какие задачи будут в нем отображаться, и настроить различные варианты их отображения. 


Доступны следующие параметры: 


  * [ Тип объектов, отображающихся в списке ](Прочее/Отображение_объектов_в_списке_планировщика.md)

  * [Параметры отбора объектов](Настройка/Параметры_отбора_объектов.md)

  * Настройки календарной сетки

  * Ссылка для [ экспорта календаря в формате iCal](Планирование/Экспорт_календаря_в_формате_iCal.md)

  * Блок синхронизации с [Google Contacts](Интеграции/Google_Contacts.md)

  * [Цвет заголовка списка](Прочее/Цвет_заголовка_списка.md)

  * Признак [Использовать полную форму](Поля и формы/Использовать_полную_форму.md)

  * Признак [Автоматически сворачивать пустой список](Автоматизация/Автоматически_сворачивать_пустой_список.md)

  * [ Отображение карточки задачи / контакта](Настройка/Настройка_отображения_карточки_в_планировщике.md)
