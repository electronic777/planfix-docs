Функция JSONARRAY() — возвращает строку ввиде валидного JSONArray для переменной. 


## Формат


**JSONARRAY(** Переменная**)**


## Пример


Используется при отправке JSON в POST-запросах. 


Например, при отправке телефонов POST-запросом, можно указать переменную {{Задача.Контрагент.Телефоны списком}}. ПланФикс автоматически превратит данные в нужный формат. Если в задаче нет или не указан контрагент, то вместо переменной будет вставлен пустой JSONArray. 

    

    

    {

        phones: %%% JSONARRAY({{Задача.Контрагент.Телефоны списком}}) %%%

    }

    


  

**Результат** : 

    

    

    {

        phones: []

    }
