В каждом отчете есть свой уникальный список переменных, который зависит от используемых в нем объектов (Задача, Действие, Аналитика) и выведенных в результатах столбцов. Список переменных доступен в режиме редактирования отчета, на вкладке **Экспорт** :

Клик по ссылке с **использованием различных полей** открывает список доступных для данного отчета переменных: 


![ox66RA.png](images/ox66RA.png)


  


## Важно


  * [Переменные](../Документы/Переменные_в_шаблонах_документов.md) работают и в названиях шаблонов для экспорта. При генерации документа на основе шаблона в имя файла на место переменных подставятся нужные данные из отчета. Это позволит автоматически задавать документам уникальные имена, быстро их находить и идентифицировать.
