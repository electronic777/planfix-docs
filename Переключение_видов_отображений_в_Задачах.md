В разделе [Задачи](Задачи.md "Задачи") есть возможность просмотра списка задач в разных видах отображения: 


  * **Таблица** — позволяет увидеть все нужные данные задачи в виде столбцов таблицы. Состав этих столбцов можно [настроить](Фильтры_задач.md "Фильтры задач").

  * **Дерево** — отличается от табличного тем, что позволяет отображать иерархию вложенных задач ([подзадач](Подзадачи.md "Подзадачи")) в виде дерева.

  * **Хроника** — позволяет видеть задачи в разрезе событий по ним. Подробнее о Хронике можно прочитать в [этой справочной статье](Хроника.md "Хроника").

  * **Гант** — позволяет отобразить отобранные в список задачи в виде [диаграммы Ганта](Диаграмма_Ганта.md "Диаграмма Ганта").

  * **Карта** — позволяет отобразить отобранные в список задачи на карте. Для этого в задачах должно быть поле типа [Местоположение](Местоположение.md "Местоположение"), которое содержит адресную информацию с привязкой к карте.


## Важно


  * Переключить вид просмотра можно над списком задач в дополнительном меню вверху справа.

  * В других разделах ПланФикса переключение вида отображения происходит аналогичным образом. Доступные виды отображения зависят от сути конкретного раздела.
