Функция HMACMD5() — принимает сообщения произвольной длины и специальный секретный ключ, затем вычисляется уникальный цифровой отпечаток сообщения с помощью алгоритма MD5. Используется в работе с API различных сервисов для их интеграции в ПланФикс. 


## Формат


**HMACMD5(** value; key**)**


## Пример


HMACMD5("Привет, мир!";"секрет123") 


Результат: fa5f5d4bc7d8f9e8248b909af324d7e3
