При выборе такой настройки для каждого сообщения из [задачи](../Задачи/Задачи.md) в **Reply-to** будет добавляться уникальный сгенерированный адрес. 


Это настраивается в разделе: 


**Управление аккаунтом** — **Работа с e-mail** — **Уведомления внешних контактов** — **Использовать уникальный адрес ответа для каждого сообщения**. 


  

Этот параметр обеспечивает попадание в задачу ответа на такое сообщение даже в случаях, если почтовые агенты или сервера, используемые внешними контактами, некорректно обрабатывают данные в скрытых заголовках писем. Обычно его активация не требуется.
