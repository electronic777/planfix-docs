При создании задачи выберите нужный [шаблон](Задачи/Шаблоны_задач.md) из списка. После этого в форму создания задачи подгрузятся все заранее установленные в шаблоне параметры. 


Для того, чтобы шаблон задачи появился в списке выбора при создании задачи, надо [настроить его видимость](Задачи/Как_настроить_видимость_шаблона_задачи_.md). 


## Важно


Данные из шаблона применяются только в момент создания задачи по нему. Если в шаблон внести изменения (например, добавить исполнителя), то это никак не отразится на уже созданных по данному шаблону задачах. Но новые задачи, создаваемые по этому шаблону начиная с этого момента, уже будут содержать этого исполнителя.
