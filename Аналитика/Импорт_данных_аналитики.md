При необходимости внести в [ аналитики](Аналитика/Аналитика.md) большое количество данных, накопленных за пределами ПланФикса, используется импорт данных аналитики. Интерфейс импорта данных аналитики вызывается кнопкой в разделе [ аналитики](Аналитика/Аналитика.md), внутри конкретной аналитики. Для примера импортируем данные аналитики Фактическое время работы: 


![IvoVOV.png](images/IvoVOV.png)


  


  * Поддерживаются исходные файлы форматов: csv, xls, xlsx, csv, xls, xlsx, mpp, mpx, mpd, mspdi, pod.

  * (1) После выбора файла система просит указать, в какие [задачи](Задачи/Задачи.md) или [контакты](Контакты/Контакты.md) импортировать данные.

  * (2) В импортируемом файле можно указать принадлежность каждой записи к конкретной задаче / контакту; совпадения определяются по номеру или названию задачи / контакта.

  * (3) После выбора файла система анализирует его и просит сопоставить имеющиеся в файле столбцы с данными аналитик в ПланФиксе:


![6KmEZN.png](images/6KmEZN.png)


  

**Важно:** если значение в первой строке столбца совпадает с названием реквизита в Планфиксе, сопоставление происходит автоматически — это позволяет подготовить файл определенного формата, чтобы при частом импорте не приходилось каждый раз делать сопоставление вручную 


## Создание файла с данными аналитик для импорта


Основой импортируемого файла является таблица Excel, в которую внесены данные для импорта. Файл может иметь любую структуру, и название колонок не должно обязательно совпадать с названиями полей в структуре справочника. Вы можете соотнести их с полями, которые используете, при задании условий импорта. 


Готовый файл с данными сохраняется в формате **CSV (разделители - запятые)** (Файл- Сохранить как...): 


![tTKj3G.png](images/tTKj3G.png)


## Важно


  * При импорте в поля, содержащие дату и время: 

    * для типа [Дата](Прочее/Дата.md) данные должны быть представлены в формате: **дд-мм-гггг**

    * для типа [Дата и время](Время и учет/Дата_и_время.md) данные должны быть представлены в формате: **дд-мм-гггг чч:мм**

    * для типа [Период времени](Время и учет/Период_времени.md) данные должны быть представлены в формате: **чч:мм** \- **чч:мм**

    * номер дня в виде одной цифры и год в виде двух цифр не поддерживаются

    * вместо символа "-" могут использоваться символы "." (точка) или "/" (слэш).

  * Для корректного импорта файлов формата CSV кодировка файла должна быть **UTF-8 ​​​​​​​**


## Импорт данных аналитики сотрудниками


  * Сотрудники также могут импортировать данные аналитики. Импорт для них доступен в разделе "Задачи" по нажатию на соответствующую кнопку.
