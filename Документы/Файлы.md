Работа с документами и файлами в ПланФиксе может вестись: 


  * На уровне [задачи](../Задачи/Задачи.md) или [ комментария](../Автоматизация/Действия.md).

  * На уровне [проекта](../Проекты/Проекты.md).

  * На уровне всего аккаунта в [Документах](Документы.md).


На каждом уровне можно добавлять, просматривать и редактировать файлы (при наличии соответствующего доступа). 


## Добавление файлов


Файл можно добавить: 


  * [В интерфейсе ПланФикса](../Задачи/Как_прикрепить_к_задаче_файл_.md).

  * Приложив файл к письму, отправленному на [определенный адрес](../Прочее/Работа_в_ПланФиксе_с_помощью_e-mail.md).

  * При помощи [API](../Интеграции/API_ПланФикса.md).

## Просмотр файлов


Для просмотра файла достаточно кликнуть по нему. В открывшемся интерфейсе просмотра можно: 


  * Просмотреть уменьшенную/полную версию файла (переключается кликом по изображению).

  * Скачать файл.

  * Отредактировать файл онлайн (только для некоторых типов файлов, см. **Редактирование файлов**).

  * Управлять доступом к файлу для сотрудников аккаунта и внешних пользователей.

  * Просмотреть [историю версий файла](Ведение_версий_документа.md).

  * Просмотреть все файлы, прикрепленные к данной задаче.


## Редактирование файлов


Файлы могут быть отредактированы одним из трех способов: 


    1\. Скачать файл, отредактировать его и [ загрузить новую версию](Ведение_версий_документа.md) в ПланФикс. При загрузке файла с уже имеющимся в данном проекте именем, ПланФикс спросит, является ли загружаемый файл новой версией уже имеющегося в системе, и при утвердительном ответе загрузит его и будет отображать в истории версий.


    2\. [ Отредактировать файл при помощи онлайн-редактора](Редактирование_документов_в_онлайне.md). На данный момент эта опция доступна только для файлов Microsoft Word и Microsoft Excel. Для редактирования требуется доступа к файлу на редактирование.


    3\. [ Отредактировать файл на локальном устройстве](Редактирование_документов_на_локальном_устройстве.md) при помощи специализированного программного обеспечения.


## Дополнительная информация


  * В ПланФиксе существует ограничение на максимальный размер добавляемого файла: **100Мб** — для платных и премиум-аккаунтов и **15Мб** — для бесплатных аккаунтов.

  * Общий размер файлов в аккаунте ограничен [пакетом использования](https://planfix.ru/#pagepricing) и может быть расширен владельцем аккаунта в разделе **Подписка на сервис**.

  * Количество файлов в аккаунте или задаче не влияет на скорость работы ПланФикса.

  * Руководитель по [ структуре компании](../Контакты/Структура_компании.md) может редактировать файлы сотрудников-подчинённых.
