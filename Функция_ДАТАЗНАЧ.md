Функция ДАТАЗНАЧ преобразует дату, которая хранится в виде текста, в число, которое соответствует этой дате для целей сравнения и вычитания. 


Обычно используется в связке с [ функцией ЕСЛИ](Функция_ЕСЛИ.md "Функция ЕСЛИ") или вычитанием. При вычитании результатом является разница в минутах. 


  

Формат: 


**ДАТАЗНАЧ(Строка)**


**DATEVALUE(Строка)**


  

где: 


**Строка** соответствует одному из следующих форматов: 


  * ДД-ММ-ГГГГ

  * ДД-ММ-ГГГГ ЧЧ:ММ

  * ДД.ММ.ГГГГ

  * ДД.ММ.ГГГГ ЧЧ:ММ

  * ГГГГ-ММ-ДД

  * ГГГГ-ММ-ДД ЧЧ:ММ


  

**Пример:**


ЕСЛИ(D>ДАТАЗНАЧ("01-08-2015");B*2;B) 


  

В этом примере для каждой строки отчета в столбце, содержащем указанную формулу, проверяется, больше ли дата в столбце D этой же строки, чем первое августа 2015 года и в случае положительного ответа выводится значение из столбца B этой строки, увеличенное вдвое, иначе выводится просто значение из столбца B.
