Список допустимых значений для сортировки файлов: 


Значение | Описание | Примечание   

---|---|---  

NUMBER_ASC | сортировка по номеру (возрастание) |   

NUMBER_DESC | сортировка по номеру (убывание) |   

NAME_ASC | сортировка по имени (возрастание) |   

NAME_DESC | сортировка по имени (убывание) |   

DATE_ASC | сортировка по времени загрузки (возрастание) |   

DATE_DESC | сортировка по времени загрузки (убывание) |   

EXT_ASC | сортировка по типу (возрастание) |   

EXT_DESC | сортировка по типу (убывание) |
